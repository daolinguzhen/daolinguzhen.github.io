道林镇“破冰”行动昼夜无休，吹响除雪抗冰“集结号”

发表于 2021-12-28 分类于 乡镇动态 本文字数： 467 阅读时长 ≈ 1 分钟

“凌晨五点左右路面相关情况，雪已停，气温零下二度，道善公路，G354国道等主要公路全线无块状结冰现象。龙泉湖门楼处，龙泉湖与善山岭交界界碑处，善山岭新建治超摄像头处等零星几处地点路面积雪稍多，高速公路口畅通。”12月27日凌晨五点，道林镇机关群里值班人员正在通报最新的路面巡查情况。
    早上六点，天还没亮，道林镇相关负责人再次到道林大桥等路段查看桥面结冰情况。与此同时，该镇六个主要路段的值守干部已经扛着铁锹铁铲等工具，开始了新一轮的“破冰行动”。
    12月26日以来，道林镇大雪纷飞，路面积雪、冰冻，给群众的出行带来了极大的不便。道林镇党委、政府迅速投入人力物力，集结工作人员到各主要路段积极除雪抗冰，全力维护正常交通秩序。道林镇党政主要负责同志均奔赴现场调度，镇域内六个主要路段实行三班轮岗制，每班次安排干部职工21人，铲车5台，昼夜不停铲除积雪、疏导车辆。同时，在重点路段、桥面洒工业盐融雪剂、设立提醒警示牌，全镇主次干道均做到了保安保畅。

道林镇：深化“五进”宣传，筑牢交通安全防线

发表于 2021-12-24 分类于 乡镇动态 本文字数： 1.2k 阅读时长 ≈ 1 分钟

12月，是全国交通安全月，道林镇紧紧围绕“守法规知礼让，安全文明出行”的主题，依托网格化管理，深入开展交通安全“五进”宣传教育活动成效明显。
    宣传教育进企业
    为做好系统防范化解道路交通安全风险工作，把好源头关，由该镇道安办牵头，联合交管站、交警一中队，先后深入全镇4个加油站、鑫美矿业、长鸣路面建材公司、先明砖厂等企业进行宣教。活动中，宣讲人员用通俗易懂的语言讲述了交通安全法律知识和安全常识，与大家“面对面”零距离交流，让大家从思想上认识、从行动上重视交通安全的重要性。
    宣传教育进学校
    12月以来，镇道安办联合交警一中队、派出所到全镇17所中小学、幼儿园开展新一轮的交通安全主题宣传教育活动。利用校园广播、电子显示屏、张贴标语、发放宣传册画等多种途径进行广泛宣传，发动各班开展以交通安全为主题的班会、知识讲座、竞赛、演讲等活动，在班级微信群推广《致家长的一封信》，号召家长和学生共同参与，通过“小手拉大手”的形式，让学生们向家中的爸爸妈妈爷爷奶奶宣传交通安全常识，让每个学生都成为交通安全的“传播者”，达到“教育一个学生，带动一个家庭，影响整个社会”的效果。
    宣传教育进社区
    近日，镇道安办还联合交警一中队、派出所在道林社区开展“一盔一带”、“戴帽工程”群众性宣教活动，参与人数达1000余人，现场免费发放头盔500余顶，签订“守法规知礼让，安全文明出行”承诺书800余份，群众参与度高，反响强烈。
    宣传教育进农村
    由镇道安办牵头，以网格为单位，交警、交管站、各村（社区）干部、片区网格员共同参与，利用“四联五到位”，组织召开屋场户主会，先后开展300多场“屋场夜话”，播放交通安全警示片，宣传交通安全知识，发放交通安全宣传单，增强群众的交通安全出行意识。
    宣传教育进家庭
    该镇先后组织镇村干部和党员、志愿者140余人，开展“四联五到位”和“屋场夜话”等活动，先后深入15000余户人家，张贴交通安全宣传漫画，并通过身边的事例、简洁的语言，开展交通安全知识宣讲，增强群众的交通安全出行意识。截至目前，该镇交通安全宣传入户率达到了100%。
    通过开展“五进”宣传活动，道林镇居民群众的交通安全意识和守法观念得到了明显提升，安全隐患得到了有效整治，群众自觉向亲友传播交通安全理念，影响并带动更多家庭参与到交通安全宣传工作中来。

道林镇：小小号牌为群众解难，种粮大户送锦旗致谢

发表于 2021-12-20 分类于 乡镇动态 本文字数： 905 阅读时长 ≈ 1 分钟

12月15日，道林镇道林社区种粮大户鲁国义夫妇来到道林镇党委副书记、镇长姜晓办公室，送上一面锦旗，不停道谢：“感谢党委政府想出来的好办法，我家的30万斤粮食全部入库，卖了38万多元。”

鲁国义说的“好办法”是今年道林收粮点的创新举措。道林收粮点于11月29日开放，由于宁乡其他两个粮库已满仓，每天都有好几十台从其他乡镇和本乡镇的售粮车赶来，镇区主要道路新枫路和收粮点入口道路被堵得水泄不通。
    12月1日，这一幕被下村路过的姜晓看到，上前一问，有的司机已经等了一天了，预计还要等两天，再与其他几位司机深入交谈，得知往年也是一样，大家担心送不上粮，还怕被别人插队，只好都守在车上排队，排着排着就排出了长龙，排个三四天是常事。由于车子挤挤攘攘，工作效率极低，粮库一天只能入库百来吨，而且粮库的入口就是个大陡坡，大车还存在溜坡的危险，有极大的安全隐患。
    于是姜晓想了一个“号牌叫号”的办法，用一式两联的“号牌”登记车主电话、车牌号码，按先后顺序排号，“号牌”车主和收粮点各执一份，快轮到号的时候收粮点人员提前电话通知，这样车主就可以把车先开去其他地方等候“叫号”，排在后面的反正当天轮不到还可以利用这个时间去做些其他事情，不用把时间白白耗在排队上。这个办法一出来，粮库第二天就入库了172吨，第三天就是230吨，最高的一天入库了280吨，效率提高了近一倍。
    “往年这么短的时间入库想都没想过。按照以往的模式，我这些粮食一车至少得等三四天才能入库，今年实施送粮取号排队后，一车只需要最多两天就能入库，效率上去了，送粮的车费也降下来了，那个号码牌不光是给我省了时间，仅车费都省了两万多元，这真的是党委政府为我们老百姓办的一件大好事啊。”鲁国义发自肺腑的感慨道。
    道林镇共有种粮大户130多户，其中像鲁国义这样种植面积上300亩的大户有16户。这张小小的“号牌”不但为这些大户们节省了时间，提高了效率，还为他们累计节省了约四十万元的送粮车费。

道林镇：“叫号”售粮省心又省力

发表于 2021-12-13 分类于 乡镇动态 本文字数： 943 阅读时长 ≈ 1 分钟

“来送粮的先取号，快轮到了就会有人通知，不用挤在这里等，非常人性化，大家也安安心心一边休息一边等。”12月6日，艳阳高照，记者在宁乡金霞粮食购销有限公司道林收购点看到，一辆辆满载秋粮的售粮车正在接受扦样员进行扦样，一旁粮库里的清筛机也在开足马力将粮食传送入库，现场一派井然有序的收购景象。
    “我是昨天下午来的，号码是6，预计今天就轮到我了。”从金洲镇过来的售粮户江国庆靠着粮车晒太阳，他拿着一张粉色的“号牌”告诉记者，今年他已经是第三次来送粮食了，昨天一来他就轻车熟路先取了号，然后安安心心地休息了一夜。记者发现，这个“号牌”一式两联，上面有号码、车牌号、车主姓名和电话号码。售粮户手持粉联，粮站手持白联。“快轮到我了，粮站工作人员就会给我打电话通知我，非常省心。不像往年，车子塞路，人挤人，效率低，人也疲劳。”

跟江国庆一样的，预计当天可以过磅的还有十多位送粮户，他们在道林镇农业综合服务中心工作人员的安排下，将车有序停在了粮库入口。在现场指挥调度的镇干部何伟告诉记者，道林收购点自11月29日开放，当天来自全市各个乡镇的售粮车近百辆，大家“一窝蜂”地涌入，导致镇区主要道路新枫路和入口道路水泄不通，不但影响了工作效率，更产生了安全隐患。“入口是个大陡坡，货车挤在这里非常危险，所以我们一方面另寻安全停车区疏散交通，另一方面想出了这个取号排队、依次入库的办法，现在看来效果非常好。

“多亏镇政府想出了‘叫号’这个好办法，缓解了我们的工作压力。”该收购站负责人罗高告诉记者，今年粮食大丰收，到11月底，宁乡其余两个粮库已经满仓，现在仅有道林镇这一个收购点还有“余力”纳粮，但还有许多农户的粮食没有入仓。而该站点只有两套设备进行清筛和传送入库，工作能力有限。“第一天，大家挤挤攘攘，工作效率低，才入库108吨，但是这个办法一出来，第二天就入库了172吨，第三天就是230吨，昨天最高入库了280吨……”

“我是打心眼里高兴，5天时间我已经送了十几万斤稻谷来，都已经经过检测、过磅入库了！”道林镇的种粮大户曾碧辉手拿一沓收据，非常激动。她告诉记者，她家今年种植了350多亩水稻，晚稻粮食超过150吨，看当前的进度，预计再过10天，她家的粮食都能入库。

12月3日下午5点，道林镇召开镇村干部大会，传达学习宁乡市安全生产工作紧急会议精神，部署道林镇安全生产百日大会战工作。

道林镇深入推进“戴帽工程”专项行动

发表于 2021-11-30 分类于 乡镇动态 本文字数： 1.4k 阅读时长 ≈ 1 分钟

为深化“一盔一带”安全守护行动工作成效，按照宁乡市《关于全市深入推进“戴帽工程”专项行动的工作方案》的具体要求，2021年9月30日下午3点，道林镇组织召开《宁乡市道林镇道路交通安全现场办公会会议精神工作部署会》，全体党政领导、办公室主任及各村（社区）支部书记参会，会议传达学习了花明楼9.29现场会精神，党委书记彭舜就《宁乡市2021年国庆节期间道路交通安全管理工作方案》及《宁乡市深入推进“戴帽工程”专项行动的工作方案》进行深入学习和研讨，并对“戴帽工程”专项行动作动部署。
    一是多措并举，加强宣教。镇道安办和交警一中队在各村（社区）集镇、加油站、源头企业、学校、幼儿园开展群众喜闻乐见的宣传教育活动，通过开展“小手拉大手，少年儿童平安回家”交通安全主题班会,张贴宣传警示标语,开办交通安全宣传专栏,交通事故图片展览等方式开展形式多样的交通安全宣传活动；通过全镇55个片区网格干部下村召开第二次交通安全屋场会，组织1788名群众观看《车祸猛如虎》、《让生命无憾》等交通宣传视频,通过发放《知危险，会避险安全文明出行》、《平安宁乡，路见文明》等宣传资料,签订《车辆驾驶人安全承诺书》1788份；各村级交通安全劝导站也通过微信群发、播放村村响、交通安全宣传车巡逻喊话等方式积极引导广大群众学法、懂法、守法,劝导广大驾乘人员自觉遵守交通法规，使交通安全宣传工作入村、入校、入户、入耳、入心。
    二是细致摸排，不留死角。镇道安办与辖区内客运车辆和各源头企业货运车辆驾驶人签订“安全驾驶承诺书”，大力开展优秀交通安全单位,优秀机动年辆驾驶人评选活动,树立典型,推行交通安全情况公告和交通安全责任追究制度，每月组织全体司机进行“不超限超载、不超速、不违法载人、不饮酒后驾车、不乱停乱放”等“五不”教育，对记录在册的有多次违规驾驶的车辆驾驶人实行重点教育，并在各村（社区）开展了第三季度安全隐患大排查，对镇域内32条农村公路，16.68公里隐患里程进行摸排整改，同时将18处平交、交叉路口进行绿化遮挡清理，大大提升了路域综合环境。
    三是严格执法，防治结合。自10月8日起，道林镇道安办每日组织交警一中队、交管站、派出所、城管中队、综合执法大队在辖区内开展道路交通安全早晚高峰执法专项行动。联合执法行动队深入各村（社区）重点路口，现场开展执法工作，对驾乘摩托车不戴安全头盔、违法搭载多人、非法安装遮阳伞、超限超载、酒驾醉驾等农村道路交通违法行为进行严厉查处。截至目前，联合执法行动共检查车辆580余台，组织驾驶员127人进行现场学习，暂扣无牌无证摩托车18辆，老年代步车2辆，对6辆超载大货车进行处罚及批评教育，另对104起非法改装、闯红灯、逆行、不系安全带等违法行为进行查处，行政拘留1起小货车无证驾驶行为，有力地震慑了各种道路交通违法违规行为。
    通过开展一系列的交通安全宣传活动，道林镇交通安全宣传工作深入人心，越来越多的志愿者参与道日常宣传劝导工作中来，道林镇10-11月份平均戴帽率提升到90%以上，各类车辆驾驶人道路交通安全法律法规和安全知识得到了全面加强和提高，同时有效的控压辖区交通事故的发生，为促进道路交通安全发展夯实了基础。

道林镇多措并举开展“金融创安”工作

今年以来，道林镇多措并举扎实开展“金融创安”工作，促进社会信用意识的大提高、金融生态环境的大改善、经济金融的大发展。

道林镇“青年微讲堂”开讲，助力年轻干部成长

发表于 2021-11-16 分类于 乡镇动态 本文字数： 437 阅读时长 ≈ 1 分钟

“打电话应根据对方的工作、学习、休息等习惯选号通话时间，通话应简单明了，重要内容可以扼要地向对方再叙述一遍，以求确认。”近日，在宁乡市道林镇每月一次的“青年微讲堂”活动中，一场精彩的商务礼仪知识授课正在进行，台下30多名年轻干部听得津津有味，边听边做笔记。
    大家纷纷表示，课程内容接地气、有内涵，令人收获颇丰。

道林镇从今年8月份开始，每月利用一个周四的下午，组织镇机关35岁以下年轻干部利用“微讲堂”形式进行交流学习。课堂上都是年轻干部自己上台讲课，没有空话套话，全是工作中派得上用场的“纯干货”。道林镇机关共有干部职工115人，其中35岁以下年轻干部42名。
    举办“青年微讲堂”活动，旨在通过让年轻干部上台授课，讲学习感悟、专长技能和工作实操，进一步提升基层干部的语言表达能力，促进互动交流，相互学习共同进步，不断激发青年干部干事创业的热情，努力在全镇上下营造浓厚的学习工作氛围。

党建引领传承红色文化、文旅融合助推乡村振兴

发表于 2021-11-09 分类于 乡镇动态 本文字数： 1.4k 阅读时长 ≈ 1 分钟

道林镇“千年铁魂湖南人”景区全面贯彻落实长沙市第十四次党代会精神和宁乡市第二次党代会、市两会精神，以党建引领传承红色文化，以文旅融合助推乡村振兴，取得了可喜的成效。
    璀璨绚烂的红色火光中，湖湘历史上一位位仁人志士缓缓走来，以“革命曙光、军民情谊、湖湘英烈、走向胜利”四个篇章，与观众进行穿越时空的对话……这场融中国三千年冶炼文化与湖南地方传统文化于一体的“千年铁魂湖南人”大型山水实景剧以独特的方式呈现在观众面前。
    利用红色资源，打造文旅融合新样板
    “太震撼了，作为湖南人我感到非常骄傲。”刚在景区看完演出的胡先生一脸惊喜和自豪地说。“千年铁魂湖南人”景区位于我市道林镇，地理位置优越，西6公里邻刘少奇主席故里花明楼，西南距毛泽东主席故乡韶山20公里，是湖南省红色旅游线路上的重要节点。道林镇还是有着1100多年历史的古镇，湘军名将周达武、著名革命教育家朱剑凡、民国三省主席鲁涤平、美学大师李泽厚等都是从道林走出的近现代湖湘人物的杰出代表。该镇依托交通优势和红色资源，打造文旅融合新样板，开创了经济社会高质量发展的新局面。
    弘扬红色精神，谱写“湖南精神”新篇章
    “听说景区非常火爆，带着一家老小本来还担心找不到停车位，没想到在志愿者的贴心引导下非常顺利地停好了车。”从湘潭来的游客陈女士不停地为景区的党员志愿服务点赞。景区及周边随处可见的“红马甲”，是道林镇为配合做好“千年铁魂湖南人”景区开园运营、服务游客的党员志愿者。
    “开园以来，每天安排58名党员干部志愿者在景区周边10个重要路口进行交通秩序维护，另外还设立了党员服务岗，在景区内及周边开展导游、协调、应急处理等服务。”据道林镇党委委员、组织委员邹培介绍，今年，该镇充分发挥党建引领作用，积极倡导“有事您说话，有事找党员”，成立党员志愿服务队伍、设立党员服务岗、评选党员示范门店，通过一系列党员志愿服务活动，引导广大党员干部增强党性意识、责任意识、大局意识，打造党员志愿服务品牌，发挥党员先锋模范作用，传承发展优秀湖湘文化，谱写弘扬“湖南精神”的新篇章。
    赓续红色基因，构建乡村振兴新格局
    道林镇始终坚持以人民为中心的发展理念，让发展成果普惠道林群众，为实现“千年古镇、幸福道林”不懈奋斗。一方面强化产业带动，提供镇域经济发展新活力。将“千年铁魂湖南人”景区开园和运营好作为抓发展的头等大事，全面描绘好“旅游导览图”，不断丰富旅游业态，优化项目旅游功能设施配套，高标准指导建设一批现代农庄、家庭农场、诗意民宿，注重带动景区周边村和群众增收致富。二是建好美丽宜居村庄，激发乡村振兴新动能。采取“群众主人、村级主体、政府主导”模式建设5个美丽宜居村庄，发动群众筹资203.58万元、筹工900余个，对“田园、果园、菜园、家园”统一规划、统一标准，共同打造党员干部带头奉献、群众共同参与、构建共建共治共享的乡村振兴新格局。
