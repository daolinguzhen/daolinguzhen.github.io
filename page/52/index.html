纵身一跳，道林镇驻村第一书记勇救落水儿童

发表于 2020-04-27 分类于 乡镇动态 本文字数： 937 阅读时长 ≈ 1 分钟

“救命、救命啊！”4月24日上午11时许，道林镇善山岭村驻村第一书记黄俊杰，正在道林镇河东新村黄新组贫困户黄明亮家进行脱贫攻坚“回头看”交叉互查走访，突然一阵揪心的“救命”声传了过来。
    “不好，出事了！”黄俊杰立马冲出门，朝发出呼救声的池塘方向跑去。跑到池塘边，看到一位老人在离岸边大约3米的地方，手上托着一个小孩呼喊救命。危急之中，黄俊杰纵身一跃跳入水塘中，从老人手中接过小孩向岸边使劲游去。爬上岸后，黄俊杰把小孩抱在怀里，只见小孩脸色惨白，不省人事。
    “必须马上把小孩腹中的水排出来。”当过兵的黄俊杰在部队学过溺水急救。只见他蹲在地上，把小孩头朝下，腹部放在膝盖上，用手不停地拍打小孩的背，过了一会，小孩慢慢地吐了一口水。黄俊杰松了口气，继续拍打小孩背部，小孩接着又吐出了几口水。3分钟后，小孩的嘴终于动了一下，随后苏醒过来。　
    这时小孩的外婆拿来了一床被子赶来，黄俊杰把小孩的湿衣服脱掉，用被子裹了起来，抱到了小孩的家中。交谈中，黄俊杰得知小孩不到两岁，随家中亲人（老人）出门散步时不慎摔倒滚入水塘中。随后，黄俊杰打通了村卫生室医生的电话，医生赶了过来看到小孩的嘴角有血，建议送去医院检查。
    浑身湿透的黄俊杰来不及更换衣服便抱着小孩朝自己的车子跑去，与小孩外婆一起往道林镇卫生院赶。因卫生院条件有限，黄俊杰随后用手机拨通了小孩妈妈的电话，告知事情大概经过，并征求她送医的意见。之后，黄俊杰拨打了120急救电话，告诉了医生小孩的情况。
    救护车赶到后，黄俊杰看到载着小孩的救护车远去，才长长舒了一口气。下午，黄俊杰仍不放心，便拨打小孩母亲的电话询问小孩的情况。小孩的母亲告诉他，孩子肺部的水已全部排出，医生说没有危险了，但需要在医院观察两天。此时，黄俊杰悬着的心才落了下来。
    “回想起来，当时救人也是一种本能使然！” 黄俊杰说，他做了一个共产党员应该做的事，做了一个扶贫干部应该做的事，心中无比欣慰。

道林镇开展“4•15”全民国家安全教育日暨平安建设宣传活动

发表于 2020-04-16 分类于 乡镇动态 本文字数： 589 阅读时长 ≈ 1 分钟

4月15日，道林镇以第五个全民国家安全教育日为契机，组织镇社会治安和应急管理办、综合行政执法队和各村（社区）辅警、志愿者，在镇域范围内开展了一场内容丰富、形式多样、效果明显的平安建设集中宣传活动。
    活动旨在通过群众喜闻乐见的宣传活动，让责任和安全意识在基层生根发芽，共同建设平安道林，不断提升全镇人民的获得感、幸福感、安全感。
    活动现场，布置的宣传横幅、展板吸引了不少群众围观，工作人员通过发放宣传资料、引导观看展板等形式，向在场群众普及国家安全、平安创建等知识。
    启动仪式结束后，宣传车辆整齐划一、列队出发，在集镇范围开展流动宣传。通过在人流聚集的汽车站和农贸市场专门设点，播放国家安全教育、平安创建、预防蘑菇中毒等宣传音频，发放宣传资料，切实提高群众知晓率和参与率。当天，共计发放国家安全教育、禁毒等资料3000余份。
    “前期，我镇在停车场、景区、车站等公共场所以及进出道林镇的主要交通路口增设了平安创建固定宣传标语和海报等。”道林镇政法委员张文宏表示，下阶段，该镇将利用户主会、屋场会将平安建设宣传在9个村（社区）全面铺开，切实把“责任在心，安全同行”的理念深入人心，让平安建设家喻户晓。

道林镇：强化党管武装、锤炼民兵队伍

发表于 2020-04-14 分类于 乡镇动态 本文字数： 634 阅读时长 ≈ 1 分钟

4月13日，道林镇武装部组织开展了民兵队伍集训演练。
    会议对参训民兵进行了基层民兵政治教育、基层民兵防汛抗旱应急演练和应急处突动员培训，并提出了三点要求：一是要始终坚定举旗铸魂，提高政治站位；二是要持续提升民兵队伍素质，确保充分发挥道林镇基层民兵在国防宣传、防汛抗旱、服务基层等工作中的砥柱作用；三是要结合镇党委、政府中心工作，通过开展武装工作来助力脱贫攻坚、春耕生产、古镇建设、文明创建等重点工作。
    随后，民兵集训队伍参观了镇武装部，熟悉和掌握了应急器材工具，进行了绕镇10公里拉练和清水塘水库防汛抗旱应急演练。最后，民兵志愿者为种烟贫困户送去了农药化肥，对5亩烟田进行了除草服务。
    据悉，一直以来，道林镇党委、政府把武装工作纳入重要议事日程，保证国防动员、民兵整组、民兵训练等各项工作落到实处，加强基层武装工作的经费保障，完善阵地建设，确保星级达标，加强政治建军和岗位练兵，全面提升基层专武干部和民兵队伍综合素质。
    道林镇武装部长刘军介绍，本次民兵集训只是一个开始，2020年，道林镇武装部将围绕“千年古镇 幸福道林”建设目标，创建“三星武装部”，力争“五星武装部”，持续开展思想政治建设、阵地建设、队伍建设等系列工作，造就一支具有铁一般信仰、铁一般纪律、铁一般担当的基层民兵队伍。

道林镇不负春光抓春耕

发表于 2020-04-07 分类于 乡镇动态 本文字数： 495 阅读时长 ≈ 1 分钟

4月的道林镇处处生机勃勃，一片片油菜籽等待收割，一丘丘烟田长势良好，一亩亩水田正在翻耕……
    4月1日，道林镇金华村种粮大户杨海清一大早就在田里忙活起来，把家门口的几亩田进行了翻耕，随后与妻子谢翠娥一道把刚到的肥料一袋袋搬到仓库码放好。夫妻二人一边搬运，一边说着今年农资备耕的情况：家里流转了130亩水田准备种植双季稻，现在已经翻耕了70多亩，还储备了不少农资。
    谢翠娥说，今年当地政府和支村“两委”早早就发动农户种植双季稻，每家每户发放了宣传单页，也把今年上级相关补助政策及镇里的配套政策进行了宣传，让他们对于今年种植双季稻充满了信心。金华村村委会主任罗建军表示，在镇村两级的推动下，村里今年双季稻落实面积及农资储备情况都比较好，全村4075亩水田，已经落实双季稻种植面积3166亩。
    据悉，至目前，包括金华村在内，道林镇9个村（社区）共落实双季稻种植面积27500亩，已翻耕22000亩，全镇有150多台大型农机在进行全面作业，水田翻耕亩数还在增加。

道林镇四举措抓实春耕生产工作

发表于 2020-03-27 分类于 乡镇动态 本文字数： 387 阅读时长 ≈ 1 分钟

一是浓厚种植氛围，加强宣传引导。通过召开粮食种植大户集中会议、分片户主会来宣传粮食生产补贴政策和耕地抛荒治理办法；印发粮食种植和早稻育秧技术知识资料10000余份，做到家喻户晓。

宁乡道林：油菜花里开火车

发表于 2020-03-16 分类于 乡镇动态 本文字数： 288 阅读时长 ≈ 1 分钟

草长莺飞的三月，正是油菜花盛开的季节。今日，记者在宁乡市道林古镇看到，5000亩连片油菜花开放，沿河而建的762毫米窄轨观光小火车静静停驻在如诗如画的田野上，构成了一幅浓墨重彩的美丽乡村画卷，吸引了不少游人前来赏花“打卡”。

道林古镇景区建设以绿色、自然、生态、野趣、人与自然和谐共生为主旨，定位为生态文明建设的重要载体、农村综合改革的重要探索、乡村振兴战略的重要抓手和全域旅游景区的重要板块，构建了一带、两翼、三园功能结构。

“看着这满目金黄，人也非常愉悦，非常放松。”特意从长沙赶来道林踏春的市民夏冰说，久违的阳光、绽放的鲜花、可爱的小火车，像画卷一般徐徐展开，让人心情大好。
