道林镇开展“春暖花开、悦读相伴”儿童文化活动

发表于 2023-03-27 分类于 乡镇动态 本文字数： 528 阅读时长 ≈ 1 分钟

为丰富道林镇儿童的文化生活，营造“书香道林”的环境氛围，3月25日上午，道林镇文化站以党建为引领，践行党员“三亮三比”行动，主动作为，组织50余名困境儿童及家属，开展“春暖花开，悦读相伴”为主题的亲子阅读文化活动。
    活动中，由老师带领孩子们进行《中华德育故事》赏析，并开展问答互动游戏。通过“动画+传统文化故事”的方式，寓教于乐，让孩子们在学习脍炙人口的德育故事中，汲取中华优秀传统文化的养分，鼓励他们自强不息、发奋读书，拥有克服困难、知难而进的勇气，用知识改变命运，力争将来做一名对社会、对国家有用的栋梁之才。
    在道林镇党委、政府的支持下，文化站为参加活动的所有困境儿童发放了读书大礼包，助力他们未来的学习生活，希望能够用悦读点亮孩子的心灯，共同成长，悦读相伴。
    这场充满趣味与意义的亲子阅读文化活动，在孩子们的欢声笑语中画上了圆满的句号。活动通过将“春暖花开，悦读相伴”为主题的亲子阅读文化活动与关爱困境儿童，为困境儿童办实事相结合，把爱心与温暖送到孩子们的心头，让广大困境儿童切实感受到党、政府和社会的关怀温暖。

宁乡：兴修水利正当时，“通经活络”助春耕

发表于 2023-03-14 分类于 乡镇动态 本文字数： 920 阅读时长 ≈ 1 分钟

惊蛰始，春耕忙。万顷农田从沉睡中苏醒，人们开始了春耕备耕的火热浪潮。
    水利是农业的命脉，渠道是百姓的生命线。春耕时节，道林镇提前部署，兴水利，全力保障农田生产灌溉用水，一口口山塘、一条条水渠织密水网，助力春耕生产，为今年农业增产、农民增收夯实基础。
    在河东新村山塘冲机埠至谢家塘组渠道修缮硬化施工现场，灌溉沟渠已完成整修，清淤扩容，等待硬化。据河东新村村干部徐建平介绍，该区域属于道林镇灌溉盲区，此次投资100多万元，新建3.1公里水渠，补齐农田水利短板，完善民生水利，建成以后将解决约900人的生活生产用水，200亩农田的旱涝保收。
    头年修水利，来年底气足。据道林镇农业综合服务中心主任陈银知介绍，自去年冬修水利以来，道林镇投入资金510万元，推进山塘清淤、山塘基加固以及渠道修缮三大农田基本水利设施建设。“截至目前，已完成山塘清淤93口，塘基加固18口，新建渠道3.1公里，为今年早稻粮食生产打好了一个基础。”
    “有水了有水了，可以育秧了。”今年春耕，农田用水有了保障，增强了蒋炳超对今年双季稻收成的信心。
    在道林镇广阔的田间地头，随处可见农民春耕备耕的身影。道林镇为强力推进早稻生产工作，坚决扛牢扛实粮食安全重任，坚持早部署、广宣传，优服务，严督查，抓住“有人种”“有田种”“有水种”“有钱种”四个关键问题，整装待发打好今年农业生产“第一仗”。
    道林镇全镇领导干部下沉到98个网格，以网格化推进早稻水田流转和任务面积落实，利用微信群、进村入户等多种形式宣传粮食生产政策，拿出真金白银，激发种粮积极性。
    农业综合服务中心负责人介绍：“今年，全镇128名种粮大户共流转土地2.2万亩，共设置29处集中育秧点，1个集中育秧工厂。截至3月8日，已落实早稻育秧面积4500亩。预计今年早稻面积在去年的基础上可能要翻两番。”

道林镇召开2022年粮食生产表彰暨2023年早稻春耕生产动员大会

3月3日，道林镇召开2022年粮食生产表彰暨2023年早稻春耕生产动员大会，全面贯彻落实中央、省、市农村工作会议精神，回顾总结2022年粮食生产工作，动员全镇上下迅速行动，全力以赴抓好早稻春耕生产，全面落实早稻和粮食生产目标任务。
    2022年，道林镇积极响应市委、市政府实施“百井万米”找水打井工程，成功钻井7口，积极协调韶山灌区管理所有效保障农业灌溉用水。全力争取上级政策性补助资金180余万元，镇财政投入60余万元，共计修缮水利设施9处，修缮重点骨干山塘9口，清淤山塘56口。

    当天，宁乡市农业农村局农业技术指导员对优质品种选择及种植、种植结构调整、病虫防治等知识进行培训和指导，积极引导农户抓好粮食生产，提高粮食生产综合效益。

道林镇2023年度安全生产监管执法工作计划

发表于 2023-03-06 本文字数： 2.8k 阅读时长 ≈ 3 分钟

为进一步规范行政执法行为，强化行政执法责任及安全生产监察职责的落实，保证安全生产执法工作经常化、规范化、制度化，推进安全生产主体责任的落实，切实提高企业安全生产水平，根据上级应急管理部门有关要求，结合我镇实际，特制定2023年度安全生产执法计划。

一、指导思想

以党的二十大精神为指导，坚持安全发展战略，坚持“安全第一、预防为主、综合治理”方针和“属地管理与分级管理相结合，以属地管理为主”以及“谁主管、谁负责”、“谁审批、谁负责”的原则，认真贯彻落实《安全生产法》、《安全生产许可证条例》、《湖南省安全生产条例》、《安全生产监管执法职责和行政执法责任追究的暂行规定》等法律、法规和规章的规定，不断加强安全生产监督管理工作，依法履行安全生产监管执法职责，保障人民群众生命财产安全，努力推动全镇安全生产形势持续稳定。

二、监管检查范围及对象

（一）监管职责范围。镇社会治安和应急管理办主要依法对本辖区内30家合法的生产经营单位开展计划检查，纳入直接检查的有加油站8家、工贸行业企业9家、烟花爆竹经营12家、非煤矿山企业1家。
    （二）监管检查对象。对列为执法检查对象的30家生产经营单位，按本身生产经营危险程度，划分为A、B二类。

三、检查时间及频率

A类企业至少一月一次检查；B类企业至少一季度一次检查；对重大危险源和重大事故隐患每月进行不少于1次的监管执法检查。

四、检查方式及内容

（一）检查方式
    监管执法以“四不两直”（即：不发通知、不打招呼、不听汇报、不用陪同和接待、直插基层、直奔现场）的方式为主，采取明查与暗访、定期与不定期、集中行动与专项整治相结合。包括日常检查、定期检查、专项检查、重大节日、重大活动检查等。检查方法主要采用一听、二查、三看、四问、五评议、六反馈的方法进行。
    运用湖南省应急管理局行政执法软件进行执法。
    （二）检查内容
    1、依法取得有关安全生产行政许可的情况；
    2、作业场所职业危害防治的情况；
    3、建立和落实安全生产责任制、安全生产规章制度和操作规程、作业规程的情况；
    4、按照国家规定提取和使用安全生产费用、安全生产风险抵押金，以及其他安全生产投入的情况；
    5、依法设置安全生产管理机构和配备安全生产管理人员的情况；
    6、从业人员受到安全生产教育、培训，取得有关安全资格证书的情况；
    7、新建、改建、扩建工程项目的安全设施与主体工程同时设计、同时施工、同时投入生产和使用，以及按规定办理设计审查和竣工验收的情况；
    8、在有较大危险因素的生产经营场所和有关设施、设备上，设置安全警示标志的情况；
    9、对安全设备设施的维护、保养、定期检测的情况；
    10、重大危险源登记建档、定期检测、评估、监控和制定应急预案的情况；
    11、教育和督促从业人员严格执行本单位的安全生产规章制度和安全操作规程，并向从业人员如实告知作业场所和工作岗位存在的危险因素、防范措施以及事故应急措施的情况；
    12、为从业人员提供符合国家标准或者行业标准的劳动防护用品，并监督、教育从业人员按照使用规则正确佩戴和使用的情况；
    13、在同一作业区域内进行生产经营活动，可能危及对方生产安全的，与对方签订安全生产管理协议，明确各自的安全生产管理职责和应当采取的安全措施，并指定专职安全生产管理人员进行安全检查与协调的情况；
    14、对承包单位、承租单位的安全生产工作实行统一协调、管理的情况；
    15、组织安全生产检查，及时排查治理生产安全事故隐患的情况；
    16、制定、实施生产安全事故应急预案，以及有关应急预案备案的情况；
    17、危险物品的生产、经营、储存单位以及矿山企业建立应急救援组织或者兼职救援队伍、签订应急救援协议，以及应急救援器材、设备的配备、维护、保养的情况；
    18、按照规定报告生产安全事故的情况；
    19、依法应当监督检查的其他情况。

五、职责分工及工作要求

（一）职责分工
    道林镇应急管理办现有执法人员3名，每次参加执法检查人员不少于2人，其中1人侧重负责档案资料的执法检查和执法文书的填写，1人侧重负责生产现场的执法检查。
    （二）工作要求
    1、坚持严格执法，热情服务，积极督导企业落实主体责任；
    2、对监督检查中，发现生产经营单位存在的安全生产违法行为或者事故隐患的，依法采取当场予以纠正；责令限期改正、责令限期达到要求；责令立即停止作业（施工）、责令立即停止使用、责令立即排除事故隐患；责令从危险区域撤出作业人员；责令暂时停产停业、停止建设、停止施工或者停止使用现场处理措施；
    3、对被责令限期改正、限期达到要求、暂时停产停业、停止建设、停止施工或者停止使用的生产经营单位提出复查申请或者整改、治理限期届满的，自收到申请或者限期届满之日起10日内要进行复查，并填写复查意见书，由被复查单位和执法复查人员签名后存档。对逾期未整改、治理或者整改、治理不合格的生产经营单位，依法分别提报镇政府和市应急管理局。
    4、对生产经营单位及其从业人员作出现场处理措施、行政强制措施和行政处罚决定等行政执法行为前，应当充分听取当事人的陈述、申辩，对其提出的事实、理由和证据，应当进行复核。当事人提出的事实、理由和证据成立的，应当予以采纳，并应当依法制作有关法律文书，送达当事人。

旗帜鲜明讲政治，持之以恒抓落实，为建设“千年古镇、幸福道林”而努力奋斗

2月28日，宁乡市道林镇召开第二届代表大会第三次会议暨经济工作大会，盘点总结过去一年的工作，表彰先进集体和先进个人，动员和号召全镇上下旗帜鲜明讲政治、持之以恒抓落实，为建设“千年古镇、幸福道林”而努力奋斗。宁乡市人大常委会副主任刘寄华出席会议。
    会上宣读了道林镇《关于2022年度目标管理考核先进单位和先进个人评选结果的通报》和《关于兑现2022年度烟叶生产工作目标管理考核奖励和表彰2022年度烟叶生产先进单位和先进个人的决定》，并对先进单位及先进个人代表进行现场颁奖。

    河东新村用发展改变过去落后面貌，丰富村民物质精神生活，村民的道德观念、思想水平、集体意识、环保理念等也在发展中得到了提升，乡村矛盾显著减少，乡村局面欣欣向荣。该村正确地把矛盾纠纷问题放在发展中去解决，用翻天覆地、实实在在的变化提升群众生活幸福感。
    自河东新村结合实际开展法治乡村工作建设以来，网格管理工作模式不断创新，各项工作依法有序开展，乡村振兴工作稳步推进。
