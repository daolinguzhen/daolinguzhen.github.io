开设红黄榜搭建比武台，道林镇城乡管理深化改革

发表于 2018-09-18 分类于 乡镇动态 本文字数： 861 阅读时长 ≈ 1 分钟

今年以来，道林镇立足乡村振兴抓管理，创新体制机制抓长效，让城乡环境治理从原来单一的环境卫生评比向全域城乡管理拓展，从政府干部单打独斗向镇村组干部齐抓共管转变，从疲于打突击向常抓常态长效转变。

创新机制 城乡管理有了“硬指标”

为切实提升城乡管理水平和管理效率，道林镇出台了《2018年城乡管理考核工作方案》，把道路整治、控建拆违、打击非法开采纳入治理范畴，通过分项检查、综合计分、逐月考核、动态排名等方式，把工作责任落细到村、到片区、到个人。创新管理机制，考核计分实行“3+1”模式，“3”即设立环境卫生、城建城管和河长制三大块日常百分制计分，“1”即设置“环境卫生黄牌警告、违章建设、非法洗沙、非法开采、养殖污染、非法搭建、失职问责”等七项一票否决指标，直接进入后两名。通过日常工作积分，坚持按月排名，扎牢城乡管理“紧箍咒”，用好考核评比“硬指标”。

比学赶超 综合治理成了“一盘棋”

道林镇自开展城乡管理红黄榜活动以来，通过月月评比，月月出榜，让先进村有甜头，后进村有压力，形成了比学赶超、创先争优的浓厚氛围。干好有奖，干差有罚，每月前三名上红榜给予奖励，后两名上黄榜予以处罚，道林镇对照任务清单、责任清单抓落实，开展“比学赶超”擂台赛，树立一批可看、可比、可学的典型样板，统筹联动打好“组合拳”，综合治理形成“一盘棋”。

注重实效 城乡面貌换了“新容颜”

今年以来，道林镇坚持铁腕整治城乡环境顽疾，对违章建筑露头即打，共拆除违章建筑23处；成功取缔非法经营场地10处；继去年靳江两岸1000米范围规模养殖场户清零后，今年又完成禁养区20户规模退养任务；镇域县乡道全部纳入日常管理，上半年路长制工作进入全市一类乡镇行列；由点及面推进垃圾分类减量工作，首批试点村靳水、华鑫市、龙泉湖村垃圾分类分拣中心已投入使用，垃圾量减少约三分之二，主干道两厢垃圾池基本拆除；加大“空心房”整治力度，一个月拆除空心房210余栋；有效遏制非法开采，群众举报、村级监管、快速打击的效果日益彰显；切实推动全域城乡环境治理，让千年古镇“颜值”更高、“气质”更佳。

宁乡市委副书记刘永红调研道林镇龙泉湖村

发表于 2018-09-11 分类于 乡镇动态 本文字数： 895 阅读时长 ≈ 1 分钟

9月10日晚至9月11日上午，市委副书记刘永红一行前来我镇龙泉湖村进行调研，道林镇党委书记潘志国，党委副书记、镇长陈浩全程陪同。
    9月10日下午，刘书记第一站来到原烧汤河村老支部书记童石良家，与老支书进行亲切交谈，详细询问老支书的生活状况，并且在老支书的口中对村级状况及发展进行了解，对老支书退休不退岗，一如既往的关心支持村上工作的精神表示肯定。
    接着，刘书记来到村级致富带头人杨明家进行走访。杨明是村上烟叶种植大户，前几年在种田效益不高的情况下，积极支持村级产业结构调整，带动部分贫困户进行烟叶种植，增加了收入。刘书记在了解过程中重点询问了其在发展过程中遇到的困难，并鼓励他继续改变思路，争当党员先锋，带头把党的好政策落实到农民致富当中，带领群众实现更好发展。
    随后，刘书记一行来到群众刘明山家中，深入了解村上环境卫生、空心房拆除情况。在调研走访过程中刘书记充分肯定了道林在蓝天保卫战中所取得的成绩，龙泉湖村更是全市的榜样，整体环境越来越好，保持着山清水秀、家家户户干净整洁的成果；空心房拆除也在有序推进中，为当地居民带来了一个安全舒适的生活环境。
    9月11日上午，刘永红副书记继续着这一次的道林调研。首先查看了善山岭村美丽乡村示范点，同时了解移风易俗工作开展情况。现场对美丽乡村示范点建设工作作出进一步指示。
    调研工作结束后，刘书记参加了龙泉湖村召开的支村两委会议。龙泉湖村支部书记朱战武汇报了村上党建工作、精准扶贫、村级债务、基础设施建设等方面的情况。会上刘书记指出。一是坚定实施乡村振兴战略，进一步推进产业结构调整，打造绿水青山，生态宜居的美丽乡村示范点；二是推动农村专业合作社的发展、壮大，提高农产品产量；三是多举措多形式帮助贫困户脱贫，坚决打赢精准扶贫攻坚战；四是村领导班子要加强团结，相互支持，与时俱进，进一步提升战斗力，全力推动村集体经济发展。

道林镇“分片监管、奖罚明晰”推动农村生活垃圾分类工作“渐入佳境”

发表于 2018-08-28 分类于 乡镇动态 本文字数： 1.6k 阅读时长 ≈ 1 分钟

改善农村人居环境、建设美丽宜居乡村是实施乡村振兴战略的第一场硬仗，“五治”（治厕、治垃圾、治房、治水、治风）工作便是主攻方向，而治垃圾又是最能影响广大群众生活质量的一环。对此，道林镇积极实行“分片监管、奖罚明晰”垃圾回收分类处理模式，选定靳水村、龙泉湖村、华鑫市村作为首批垃圾分类减量示范村，大力推进农村生活垃圾分类减量，朝建设美丽宜居乡村的目标又迈进了一步。

目前，三个垃圾分类减量示范村的减量分拣中心已建设完毕，由镇政府给予补贴、村委会统一采购的5800套农户垃圾分类桶已分发到示范村的村民家中。 
    广泛宣传得民心
    自今年4月开始垃圾分类减量工作以来，三个示范村在每户张贴垃圾分类减量宣传单，利用村村响每日广播垃圾分类减量小知识，在村干部微信朋友圈和村党员微信群广泛宣传垃圾分类减量政策，确保户户知晓垃圾分类政策知识。“我们的村干部分片召开小组户主会，在会上传达垃圾分类减量政策、学习垃圾分类减量知识、分析垃圾分类好处。每家每户都配发了两个环保垃圾桶，绿色的是用于存放日常生活垃圾，黑色的是存放可回收垃圾。由党小组长就近联系5户群众、每名党员联系3户群众、每名组长代表联系1户群众，挨家挨户上门教群众垃圾分类，现在群众普遍都还是比较支持垃圾分类减量工作的。”靳水村总支部书记成友良向我们介绍。
    三个示范村对垃圾分类减量推进工作实施网格化管理制度，实行村干部包片负责制，每个片区安排一名村干部负责监督，一名保洁员负责清洁并分类回收垃圾，可回收垃圾交至村分拣中心由村统一出售，不可回收垃圾安排专人清运。靳水村还制订了垃圾分类实施方案，对保洁员、党小组明确了严格的奖惩机制。
    靳水村环卫专干谭放武告诉我们：“今年我们村上加大了工作经费的投入力度，在镇1200元/人/月的基础工资基础上，村上再补贴900元/人/月用于保洁员工资。同时，我们要求保洁员沿户回收垃圾并进行分类，对保洁员回收的可回收垃圾由村统一出售，村里按照出售价格补贴30%作为奖励性工资支付保洁员。这样一来，保洁员的工作劲头就更足了。”在一名农户家里，我们看到了正在拿着垃圾桶往电动三轮车里倾倒垃圾的许国仕，“我是地地道道的农民，现在能为村上的工作出一份力，并且事情做得好还有奖励，以前想都不敢想哟”，他笑呵呵的说道。这样一幅幅笑脸其实无不凸显了村上推进垃圾分类减量工作是一件有利于民的好事。
    垃圾分类成效佳
    在华鑫市村，我们见到了正在参加户主会的杨明，一个年轻的小伙子：“我家住在公路沿线，出门不远便有一个垃圾池，没搞垃圾分类之前，几乎每天垃圾池都是满的，虽然保洁员每天及时清理，但是垃圾散发的恶臭还是影响了我们的生活。现在实行垃圾分类减量以后，垃圾池都没了，道路两旁清清爽爽，我家也能天天开着窗户，真的要点个赞”。

推行垃圾分类减量工作以来，三个示范村拆除公路两旁垃圾池82个，其中龙泉湖村一鼓作气将全村约56个垃圾池全部拆除，切实促进了垃圾的回收减量。靳水村总支书记成友良向我们算起了垃圾分类处理的“省钱”账： “以前每个月全村的垃圾需要经过3-4次拖运才能到镇里的垃圾中转站，共计120吨左右，目前每月只需拖运一次，共计40吨，清运费也由原来38000元一年包干，到如今拖一次300元一个月，一年约5000块钱。一年的垃圾清运费可以省下三万多块钱。”而华鑫市村总支书记张延富则算的是“环保”账：“自从4月实行垃圾分类以来，村民的环保意识大大提高，随手丢现象杜绝，房前屋后整整洁洁、村居环境干干净净，农村人居环境大为改善。”

这个初秋，36场激情篮球赛燃爆道林古镇

发表于 2018-08-20 分类于 乡镇动态 本文字数： 1.1k 阅读时长 ≈ 1 分钟

立秋过后，傍晚气温有了些许凉意，可是道林古镇依然热度不减，原来是道林镇2018年全民运动会之“古镇杯”篮球赛正在如火如荼地举行。

全民运动  干群广泛参与

8月16日18:00，道林镇“古镇杯”篮球赛正式开幕，镇党委书记潘志国在开幕式上宣布开幕，来自政府机关、镇卫生院、镇派出所和各村（社区）的12支球队142名队员参加比赛，随后AB两组小组循环赛分别在镇政府篮球场和镇卫生院篮球场拉开战幕。比赛将持续到8月24日，小组赛期间每天晚上在两个场地进行六场比赛，交叉赛、半决赛和决赛期间每晚进行两场比赛，总共36场比赛。镇党委书记潘志国还以普通球员的身份参加了政府机关队的首场比赛并贡献了多粒进球。“如此大规模的全镇篮球赛这些年来应该是第一次。”靳水村支书成友良感慨，“我们村上也是高度重视这次篮球比赛，广泛发动村上的篮球爱好者参加比赛，有从长沙、宁乡专门赶回来报名参加比赛的，还有一个迟了一步错过了报名时间遗憾不已。”

雨中作战 球迷热情不减

每天晚上，道林镇政府和镇卫生院车水马龙，观看比赛的群众将球场四周挤得水泄不通。8月17日晚上镇政府球场第一场善山岭村对河东新村的比赛，善山岭村的球迷更是统一服装，手持加油棒，统一节奏击打并呼喊口号，十足的铁粉啦啦队模样，河东新村的啦啦队也不甘示弱，尽情为本村加油，场面更是热闹。第二场比赛打到一半开始下雨，可是球迷的热情依旧不减，球员们雨中比赛依然激情四射，直到晚上八点半，雨下得太大，比赛不得不中途休战，政府机关队15:7暂时领先鑫星村队。散场时还有球迷不断询问什么时候继续比赛，“鑫星村的实力还是不错的，只是运气差了那么一点，政府队的进球命中率比鑫星村要高，看得正过瘾，突然中断，有些舍不得。”

力挽狂澜  比赛惊心动魄

8月18日，镇政府球场第一场比赛河东新村对战华鑫市村，已经连胜两场的河东新村出师不利，一直落后华鑫市村，甚至一度相差11分，就在所有人都以为河东新村这场输定了的时候，河东新村在最后两节集中发力，以两粒精准的三分球和多粒罚球命中频繁得分，最终以42:36反败为胜，取得胜利。第二场比赛派出所队迎战政府机关队，比赛一开始，政府机关队就力压派出所队，派出所一直到0:12才以一粒三分球打开僵局，但随后的两节里，政府机关队一直疯狂进球，继续保持大比分领先，直到第四节开场，派出所队仍然是19:30落后，所有人都觉得这场比赛毫无悬念的时候，派出所队开始发力，尽力牵制对手，最后以28:31惜败。“这场比赛着实精彩，要是比赛时间还多那么一点，兴许派出所队就能反败为胜了。”现场有球迷替派出所队惋惜。

道林镇2018年全民运动会之“古镇杯”篮球赛拉开战幕

发表于 2018-08-17 分类于 乡镇动态 本文字数： 360 阅读时长 ≈ 1 分钟

8月16日，道林镇2018年全民运动会之“古镇杯”篮球赛在道林镇政府机关篮球场拉开战幕。来自道林镇政府机关、派出所、卫生院及各村（社区）的12支篮球队将在8月16日至24日展开36场比赛。

18:00，在道林镇政府机关篮球场举行了开幕式，各参赛球队依次整队入场，运动员和裁判员代表分别上台宣誓，镇党委书记潘志国宣布：“道林镇‘古镇杯’篮球赛正式开幕。”     运动员进行热身后，18:30，政府机关代表队与靳水村队的比赛在机关篮球场正式开始。比赛吸引了众多干部和群众观看，大家热情地为球员加油打气。镇党委书记潘志国也换上了球员衣服坐在替补席，并在第二小节登台上场。最终政府机关代表队以48:40战胜靳水村队，取得了首战胜利。

在卫生院篮球场同时进行的鑫星村与卫生院比赛中，鑫星村获胜。

道林镇安监站开展安全生产检查，全方位的排查安全隐患

发表于 2018-08-09 分类于 乡镇动态 本文字数： 172 阅读时长 ≈ 1 分钟

入夏以来，天气持续高温，为切实做好高温季节安全生产经营工作，防范事故发生，确保辖区安全稳定，连日来，道林镇安监站组织安监员深入到生产单位，开展安全生产检查，全方位排查安全隐患。

道林镇：“四个精准”演绎拆除“空心房”加速度

发表于 2018-08-01 分类于 乡镇动态 本文字数： 627 阅读时长 ≈ 1 分钟

工欲善其事，必先利其器。道林镇在农村“空心房”整治工作中，坚持精准摸排、精准宣传、精准突破、精准考核，全面提升群众主体作用，扮靓群众房前屋后，着力改善农村人居环境，下好了乡村振兴的“先手棋”。 
    精准摸排。迅速召开农村“空心房”整治工作动员会，对各村整治工作细化任务，明确时间节点和责任人。摸清农户基本信息、房屋现有状况等信息，分类建立镇、村（社区）、组三级台账，确保数据实事求是、准确无误。
    精准宣传。全镇连片干部和9个村（社区）干部下到每个小组、屋场宣讲相关政策，确保人人知晓。通过微信公众号、村村响、短信等多种形式，加大宣传引导力度，进行政策解读。
    精准突破。镇村干部会同国土等部门工作人员，下到农户家中，挨家挨户做工作，陈述“空心房”对环境的破坏、对资源的浪费，耐心细致与有疑虑的房主沟通，纠正错误思想，同时协调解决房主实际困难，消除农户的顾虑，主动引导群众理解、支持“空心房”整治工作。
    精准考核。将“空心房”整治工作列入镇对村（社区）年底绩效考核，对空心房整治工作进行专项督查，实行每周一通报，对拆除工作滞后的村（社区）主要负责人进行约谈和通报，对驻村帮扶干部进行考核评比，压实工作责任，确保“空心房”整治工作落实到位。截至7月31日晚，共拆除182栋空心房。

道林镇集中开展“五治”工作，强力推进环境治理

发表于 2018-07-25 分类于 乡镇动态 本文字数： 1.1k 阅读时长 ≈ 1 分钟

今年以来，道林镇深入贯彻落实宁乡市委、市政府关于对“五治”工作的指示，对全镇范围内垃圾分类处理、消除农村地区黑臭水体和劣V类水体、农村旱厕改造、拆除农村“空心房”和倡导农村婚丧喜庆新风等“五治”工作进行全面部署。工作按照“谁主管谁负责”、“谁审批谁负责”和“密切配合，系统作战”原则，围绕整治重点对群众反映涉及“五治”的问题，进行全面清理。

治垃圾
    我镇选定靳水村、华鑫市村和龙泉湖村三村推进垃圾分类减量工作示范村。制定了垃圾分类减量工作方案，设置了分拣中心，配备垃圾分类回收桶，加强保洁队伍力量。接下来，我镇将继续推广村（社区）分拣中心的配置和到户垃圾桶的配备，推动垃圾分类工作。
    治水
    镇河长办牵头组织镇环保站、综治办、派出所、畜牧站、城建城管办，对全镇范围内的河流、水渠、水库进行巡查和整治，在推进退养工作方面，整体效果较好。2018年我镇计划完成禁养区规模退养20户，其中18户已签订退养协议并完成了场地拆除。
    7月份我镇将继续推行领导干部包户责任制，要求包干领导、干部每日上门做退养户的思想工作，并联合畜牧站和环保站进行联合督查，对退养情况进行周通报，争取在7月底前完成全部退养任务。
    治厕
    联村党政领导、联村干部到村（社区），村（社区）干部到片、组，对全镇所有农户旱厕和公共旱厕进行摸底，首先对贫困户、困难家庭、五保户的旱厕进行改造。今年全镇厕改计划总数为130座，预计在7月底将完成任务总数的50%，9月底前将全部完成。
    治房
    镇重点办将“一户多宅”、“建新未拆旧户”等“空心房”任务分解到各村，并通过土地增减挂钩，将腾出的土地进行复原。今年我镇空心房整治任务数为141户，目前全镇已拆除空心房122处。
    治风
    我镇把龙泉湖村作为治风示范村，成立红白理事会、制定村规民约，同时，计划帮助将该村民联和群联两个小组打造成“新风庭院”市级示范阵地，扎实开展宣传入户工作。接下来，我镇将继续加大宣传引导力度，倡导“婚事新办、丧事简办、其他不办”，重拳出击农村三大歪风邪气（赌博风、彩礼风、攀比风），强化对各种顽劣恶习和不正之风的专项整治，提速乡风文明建设，以点带面推进全镇移风易俗工作再上新台阶。
