道林镇举办2018村级人民调解员调解知识暨法制宣传业务培训班

发表于 2018-07-19 分类于 乡镇动态 本文字数： 806 阅读时长 ≈ 1 分钟

为进一步促进人民调解工作的规范化管理，提高我镇人民调解员的业务素质，充分发挥人民调解员的作用，更好地维护我镇社会稳定，于7月18日，在镇政府大会议室组织各村治调主任、村级人民调解员共计40多人，召开2018村级人民调解员调解知识暨法制宣传业务培训会议。镇党委委员、政法书记张文宏出席并讲话，会议时间为一天，共分为四个议程。

首先，张文宏同志做了培训动员讲话，对各人民调解员为道林所做出的贡献表示了充分的肯定，并强调，在新时代中国特色社会主义的建设过程中，为了给改革和发展创造良好的社会环境，合法及时处理社会矛盾纠纷、大力开展人民调解工作的必要性。
    其次，市司法局委派律师杨跃飞主要从婚姻家庭、民间借贷、人身损害赔偿、邻里关系等方面讲解了与农村群众息息相关的法律知识，并就协议合同签订的七要素进行了分析。靳水村治调主任谭放武、石金村治调主任蒋胜仁就当前人民调解工作中遇到的难题，与杨律师进行现场互动。
    然后，镇司法所所长刘利民就人民调解文书规范化制作进行专门的讲授，他指出在撰写人民调解文书中应该特别注意的文本格式、构成等；一份完整的调解文书应具备人民调解申请书、调解调查记录、调解证据材料、调解记录、调解协议书、调解回访记录、司法确认有关材料、卷宗情况说明、封底等九部分。
    最后，镇司法所助理员许浩就宁乡市司法局微信公众号如何运行与推广，以及社区矫正相关知识进行了讲解。
    此次培训，镇政府投入大量的人力物力财力，从邀请专家授课到规范人民调解公文袋（人民调解案卷模板、台账制作、调解记录本）等方面进行了统一规范，培训内容指导性、实用性、针对性都很强，为不断提升道林人民调解工作水平，继续保持全镇的社会稳定、经济繁荣起到了很好的作用。

道林镇石金村驻村工作队当起了“鱼贩子”

发表于 2018-07-06 分类于 乡镇动态 本文字数： 877 阅读时长 ≈ 1 分钟

7月6日早上4点多，天还刚蒙蒙亮，道林镇石金村驻村第一书记熊友泉和驻村队员刘泽坤就摸黑来到了该村贫困户杨林家的鱼塘边。等他们到的时候，鱼塘边已经来了不少热心肠的村民。原来，因为近期雨水稀少，鱼塘中的水位越来越低，杨林决定开网打鱼卖掉一批。
    打鱼的头一天，杨林就找到了驻村第一书记熊友泉求助，看他有没有渠道可以帮忙销掉一些鱼。“道林石金村牌阳组贫困户杨林家自家鱼塘里的鱼因为近期雨水稀少，决定明天7月6日凌晨四点开网打鱼上来，5点-6点有需要的帮忙买走一条，草鱼8元一斤，鲢鱼3.5元一斤，雄鱼7元一斤。大家有需要的可以微信联系我或者打杨林电话……”熊友泉立马替她在自己的微信群和朋友圈里发布广告，不一会的功夫，就接到了大大小小的二十多个订单。其中道林社区的扶贫专干姜银连更是个“热心肠”，她看到熊友泉的广告后立马转发到了社区的群里，陆陆续续收到了八九个订单，其中有几户没有车不方便去取鱼的，姜银连更是揽下了送货的任务。
    “还有鱼卖没？”早晨六点多还有人在群里问，看到销量这么翘，杨林家九点又撒网捕了一批，同样销售一空。当天杨林共卖掉了991.8斤鱼，收入5810元，此外，驻村工作队顺带还替另外一位贫困户廖巧华也卖掉了100多斤鱼，收入680元。
    十点多钟，送完鱼的熊友泉和刘泽坤分别将卖鱼所得的收入送到了杨林和廖巧华手中。“太感谢你们了，你们真是我的贴心人啊！”杨林万万没想到那么多鱼这么快就全部卖掉，激动不已。
    据悉，自从3月26日派驻到石金村担任驻村第一书记以来，熊友泉带领驻村工作队不断在激发群众内生动力上下功夫，熊友泉表示：“只要贫困户家有农产品，我甘愿当推销员，通过帮忙销售贫困户劳动成果的意义远比直接给钱给物要深远得多。”在他的积极引导下，石金村贫困户的帮扶责任人也积极帮助自己结对的贫困户销售农产品，“廖大妈家的猪肉、张大爷家的包菜、辣椒都是不愁销的。”

发表于 2018-07-02 分类于 乡镇动态 本文字数： 554 阅读时长 ≈ 1 分钟

7月2日上午，在庄严的国歌声中，道林镇庆祝中国共产党成立97周年“七一”表彰大会拉开帷幕。

大会对过去一年中表现突出的先进基层党组织、优秀党组织书记、优秀党务工作者、优秀共产党员进行表彰，同时通过“知敬畏、存戒惧、守底线”纪律教育活动集中学习，进一步提高全体党员的纪律意识，更好发挥党组织的领导核心和政治核心作用，最广泛的团结党内外党员和群众。
    道林镇党委书记潘志国、党委副书记镇长陈浩等全体党政领导，受表彰对象，办公室主任，离退休党（总）支部书记，部门单位负责人及党建专干，村（社区）第一书记，村（社区）两委成员及党建专干，共约130人参加了会议。
    镇党委书记潘志国同志代表镇党委向全镇广大共产党员致以节日的问候，向所有受表彰的先进集体和优秀个人表示热烈的祝贺。潘书记在讲话中回顾了党的光辉历程，充分肯定了一年来我镇党建工作及全体党员干部取得的成绩，并就进一步加强党建工作进行了安排和部署。同时希望受表彰的优秀共产党员要珍惜荣誉，发扬成绩、戒骄戒躁、奋发有为，以更高的标准严格要求自己，更好地发挥模范表率作用；希望与会全体人员以优秀党员为榜样，在各自的工作岗位上开拓进取，克难奋进，为我镇各项事业再立新功，再创佳绩 。

“陈年谷”卖了个“好价钱”

发表于 2018-06-13 分类于 乡镇动态 本文字数： 678 阅读时长 ≈ 1 分钟

“董大妈，这里是245元的卖谷钱，您收好。剩下的谷子我再找人来收走。”6月13日，道林镇靳水村第一书记刘献和驻村队员徐坚带了两个“陌生人”来到该村贫困户董淑元家，挑走了两担陈年老谷，把卖谷钱给了董淑元。

今年79岁的董淑元是道林镇靳水村2014年建档立卡的贫困户，家中只有她和一个精神二级残疾的儿子相依为命。由于儿子没有劳动能力，年近八旬的董淑元还得靠自己种田为生。可是种田收获的谷子怎么都囤积在家中不卖出去呢？今年三月底，该村第一书记刘献在入户走访的时候就发现了董淑元家的“异常”之处——家中堆满了陈年老谷，有些已经三四年了，都开始起虫了。一打听才知道，原来是早几年董淑元的儿子精神病发作时不准她把谷子卖掉，等到去年儿子住进了精神病医院，这些陈年谷也已经没人要了。
    从此董淑元家的“陈年谷”就如同一块石头压在刘献心上，他四处替董淑元寻求买家，可是问了好几个米厂，人家都嫌这个谷子太陈旧，打出的米质量不过关。直到前不久，刘献在大屯营老家无意中遇到一位熟人，交谈中得知他现在自家养殖了不少鸡鸭，刘献立马就想到了董淑元家的“陈年谷”，立马向熟人推销，想让他买点去喂鸡鸭。哪知道这位熟人也是一位颇具爱心的人士，听说了董淑元的生活现状后，爽快地答应按高于市场价的价格收购二百斤。于是就出现了文章开头那一幕。
    董淑元接到卖谷的钱后感激不已，“这多亏了刘书记帮忙，我从没想过放了几年的谷子还可以卖得这么贵！”目前，还有几百斤的“陈年谷”没卖完，刘献表示还将继续为董淑元寻求销路。

道林镇善山岭村：第一书记带来700只“扶贫鸡”

发表于 2018-06-08 分类于 乡镇动态 本文字数： 608 阅读时长 ≈ 1 分钟

6月6日下午，道林镇善山岭村贫困户刘龙飞接到村干部通知，到村委办公楼旁领回了50只鸡苗，还有15户贫困户和刘龙飞一样也接到了领鸡苗的通知，欢天喜地地捧着一窝鸡仔回家。当天共发放鸡苗700只，共有16户贫困户收益，多的领回了100只，少的也领了20只。这些鸡苗是哪里来的？

原来，根据今年宁乡市委组织部和市扶贫办的统一安排，明确城郊街道为道林镇善山岭村的帮扶单位，并派出驻村工作队，由城郊街道农办主任胡泉江任驻村工作队队长（驻村第一书记）。第一书记胡泉江自三月底到善山岭村驻村以来，扎实走访，潜心钻研，一心想着如何谋划致富门路，让村上的贫困户尽快脱贫。经过走访调研，胡泉江发现村上有一小部分贫困户家庭劳动力不足，但并非完全丧失劳动力，他们无法外出务工，仅靠种植家中几亩水稻维持生计。“能不能让这些户子再发展点什么产业，既不用太强的劳动力，又能产生一定经济效益？”胡泉江经过思考，发现养鸡对于这些贫困户可能是一条比较合适的途径。“土鸡蛋、土鸡肉都是现在城里比较畅销的农产品，养起来也不是很费力。”

胡泉江在村上进行了前期摸底，再牵线由城郊街道筹集了14000元经费购买了700只鸡苗，免费赠送给有养鸡意愿的贫困户。为了确保养殖成效，村委还与领养鸡苗的贫困户签订了养殖协议，村委将提供养殖过程中的技术指导，贫困户保证小鸡养殖三个月以上或者每只重量达到三斤以上才能出售，如果养殖过程中有小鸡意外死亡，养殖户还得进行拍照留存报备。
