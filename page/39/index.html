道林镇组织召开反诈工作调度会议

发表于 2022-02-25 分类于 乡镇动态 本文字数： 911 阅读时长 ≈ 1 分钟

为增强辖区群众防范电信网络诈骗意识，遏制电信网络诈骗案件蔓延趋势，最大限度减少群众财产损失，2月25日上午，道林镇组织召开2022年反诈工作调度会，全体机关干部职工、部门单位负责人、基地校校长及各中小学校长、村（社区）两委干部、派出所全体职工等191人参加了会议。
    会上，宁乡市公安局刑侦大队队长郭凯同志作反诈宣传，分析当前反电诈的严峻形势，通报全国及宁乡电信诈骗发案情况，详细解析了受害群体的年龄、学历层次，强调思想上重视、行动上才有指南。在思想形成合力的基础上，推动道林镇广泛、精准、长期的反诈宣传。
    道林镇派出所戴雄同志对近年来辖区内电信诈骗典型案例作了详细、全面的分析报告，进一步加强了反电诈宣传教育。镇分管负责领导陈义军同志通报了反诈工作进展情况，并对当前反诈工作再动员、再部署。
    镇党委书记彭舜同志对推进当前反诈工作落实作出强调，道林镇连续两年获评“长沙市平安建设先进乡镇”，道林社会治理牢不可破的网决不允许被电信诈骗这个顽瘴痼疾撕开口子。3月份为我镇反诈工作“攻坚月”，要以加强社会宣传教育防范坚决遏制电诈犯罪的多发态势，守住人民群众的钱袋子。会议强调“安居乐业才是真平安”，要深入践行习近平总书记以人民为中心的发展思想及关于打击治理电信网络犯罪工作的重要指示精神，镇财政加大投入，网格干部当好“宣传员”，通过村村响、宣传车队及网格干部上户发放宣传资料等全方位、多维度的宣传，营造浓厚的宣传氛围；“逐门上户才是真功夫”，结合“四联五到位”民情大走访，开展网格干部上门走访行动，推进反诈工作在群众中走深、走实，脚步为亲，用干部的辛苦指数换取人民群众的安全幸福指数；“杜绝发案才是真效果”，牢固树立“一盘棋”思想，凝聚多方力量，形成工作合力，构建“镇党委、政策牵头，镇综治办、司法所、派出所协调联动，网格干部广泛参与”的治安防控基础信息网络，通过日常多种形式不间断的宣传教育，让群众对反电诈入脑入心，不断巩固、提高人民群众反诈的观念和能力，全力维护道林镇人民群众的财产安全。

道林镇：擦亮绿色名片，以文旅融合促乡村振兴

发表于 2022-02-03 分类于 乡镇动态 本文字数： 989 阅读时长 ≈ 1 分钟

道林镇，是宁乡市东南部一座有着1000多年历史的古镇，湘江下游一级支流靳江河穿镇而过，沿岸绿植丰茂，山青水碧。道林渡口连通长株潭，古时商贸繁荣，人文底蕴深厚，素有“小南京”之称。
    道林镇立足“千年古镇、幸福道林”目标定位，紧扣道林古镇开园，加大人居环境整治，将道林区位优势、生态优势、人文优势，集中转化为发展优势，精心将道林打造成长株潭周边好看、好吃、好玩、好住的乡村旅游目的地，大力促进乡村振兴。
    擦亮了绿色名片。今年以来，道林镇党委政府聚焦蓝天保卫战、农户改厕、河长、路长和村容村貌整治，通过村村响、宣传单、微信圈群、宣传车全方位开展日常宣传，利用“四联五到位”走访入户，开展垃圾分类宣讲屋场会400余次，发放环卫环保、秸秆禁烧等宣传资料10000余份，渣土扬尘执法43起，处理环保投诉8件，完成450座农户改厕、2座公厕建设。2020年道林镇成功创建了“省级卫生乡镇”、省级样板河（湖）。现在镇里的水更清了，山更绿了，道路更干净了，群众的人居环境得到了显著改善。
    融合了文旅产业。道林镇以古镇开园为契机，在擦亮绿色名片的同时，严格按照景观标准发动群众建设一批美丽屋场，培育一批现代农庄、家庭农场、诗意民宿，引导发展乡村旅游、民宿经济、农耕体验；引进社会资本，谋划跑马场、冰雪小镇、神仙岭星空露营基地等配套项目，按市场需求充实内容，增强旅游产品供给能力；完善镇域内旅游景点基础设施，建设一批生态停车场、公共厕所、生活超市等配套，吸引了一批又一批的旅客前来游玩“打卡”。
    促进了乡村振兴。道林镇党委政府搭建“有事找党员”、“贤达工作站”等服务平台，建设一批现代农业采摘体验基地，规模化发展特色农产品种植和精深加工，辅以许广高速旁的冷链仓储基地建设，以人流的汇集带动全镇群众就业和增收致富，为新型村级集体经济栽下常青树，走绿色、可持续的文旅融合特色小镇发展之路，促进了乡村振兴。
    “玉泉之南麓山殊，道林林壑争盘纡”——这是千年前诗圣杜甫流寓长沙时的盛赞。千年后的今天，道林镇山水人居，绿带环抱，正以矫健的步伐，响应新时代乡村振兴的号召，大阔步向前！

道林镇：深入实施“党建聚合力”工程，推动高质量发展

发表于 2022-01-25 分类于 乡镇动态 本文字数： 1.7k 阅读时长 ≈ 2 分钟

长沙市第十四次党代会报告指出，要深入实施“党建聚合力”工程，全面提高党的建设质量和水平，为现代化新长沙建设提供坚强政治保证。宁乡市道林镇全面落实“党建聚合力”任务，以高质量党建引领高质量发展，始终把抓好党建作为最大的政绩，推进和保障“千年古镇、幸福道林”建设行稳致远，为现代化新长沙建设贡献力量。

担当初心使命 凝聚思想合力

道林镇党委坚持把增强“四个意识”、坚定“四个自信”、做到“两个维护”落到具体行动上。在党史学习教育中，全镇基层党组织党员去到花明楼、沙田、文家市等红色教育基地，把理论学习融入具体党建活动，瞻仰革命先烈，学习伟人精神；开展“两优一先”评选表彰，让党员们从内心涌现出强烈的荣誉感和自豪感。在党员教育培训中，镇党委着眼于“十四五”规划和乡村振兴发展，组织开展基层党员夜校、乡村振兴专题学习班，通过高质量党员教育培训，强化党员意识，打造高知识高素质的复合型党员人才队伍，实现组织振兴支撑乡村振兴、人才振兴带动产业振兴。
    “作为青年干部，最根本的就是要时刻记得自己的身份，关键时刻站得出来，危急关头豁得出去，永葆奋斗精神，永怀赤子之心。”在道林镇机关党支部组织召开的党史学习教育专题组织生活会上，机关第一党支部书记肖丹和第二党支部书记陈银知分别上台述职，并代表机关党支部党员交流学习感悟。

党建赋能发展 凝聚组织合力

道路镇始终坚持在推动党建高质量发展上强主责，坚持以高质量党建引领高质量发展，通过强化基层组织建设，充分发挥基层党组织的战斗堡垒作用。
    在美丽宜居村庄建设中，为解决部分群众建设积极性不高、态度消极的问题，支村“两委”多次组织召开党员会，选出党员理事会代表，并组织村民代表外出参观学习。通过党组织和党员的不断宣传，加之党员同志的带头示范，普通群众的参与性随着及时通报的工作动态和工程进度而水涨船高，经常可以看到村民自发为工地人员送凉茶、西瓜的感人场景频，争先恐后集体出工的热闹场面以及烈日炎炎下乡亲们汗流浃背的劳动场景……
    在“四联五到位”民情大走访工作中，道林镇按照“前提是联系，关键是问题解决到位”的总体思路，将村民建房、禁止非法开采、道路交通安全、医疗保险、春耕生产等工作融入“四联五到位”民情大走访中，全覆盖走访、广范围宣传、多方式收集问题，并积极争取支持、切实担当作为，有效解决了一批群众关注、关切的民生问题，得到了人民群众的高度肯定。

团结创新奋进 凝聚治理合力

建设现代化新长沙，更要广大党员干部团结奋进聚合力，让党的旗帜在一线高高飘扬。道林镇始终号召广大党员要干在实处、走在前列，更要带领群众团结拼搏、砥砺奋进、再攀高峰。
    今年，道林镇以党员带动群众的工作思路，举办首届党员服务示范门店评选活动。本次评选为宁乡市首次在党员经营户中评选服务模范、示范门店，此创新举措既增强了党员经营户的党员意识、责任意识、服务意识，又营造了经营户创先争优的良好氛围，充分发挥了党员的先锋模范带头作用。道林社区党员、菜根香饭馆负责人、获评“党员服务示范门店”代表蒋江林表示:“我的党员服务示范门店是一块金字招牌。我作为一名党员，将诚信经营、热情服务，提供免费停车、茶水与洗手间等服务，为‘千年铁魂湖南人’景区运营作出积极贡献。”
    道林镇发起“志愿服务我先行”“传递爱心，关爱儿童成长”“走访帮扶聚民心，助力打赢脱贫攻坚战”“共话脱贫攻坚，共谋乡村振兴”屋场夜话、“助推全国文明城市创建”“问初心，砺作风”等一系列党员主题活动，通过充分发动党员力量，团结群众，创新实干，为道林治理开创良好新局面。
    未来，道林镇党委将深入贯彻落实长沙市第十四次党代会精神，突出党的建设重点，团结带领广大党员干部和人民群众，凝聚思想合力、组织合力、治理合力，为建设以“三高四新”为引领的现代化新长沙作出应有贡献。

道林镇：凝聚乡贤力量、助力乡村振兴

发表于 2022-01-18 分类于 乡镇动态 本文字数： 733 阅读时长 ≈ 1 分钟

近年来，宁乡市道林镇按照“有德行、有才能、有声望、有影响、有热心”的“五有”标准，评选出23位社会贤达，并通过激励社会贤达履职积极性，不断促进和保障其在扶贫济困、乡村振兴、矛盾调解、产业发展等领域发挥的积极作用，取得了一定的成效。
    2021年，为进一步凝聚一股多层次、广角度，有财有智、有德有心的社会贤达力量，道林镇鑫星村社会贤达杨志明主动提供房屋并出资40余万元，耗时4个月建设装修了宁乡市社会贤达道林镇工作站。目前，全镇23位社会贤达全部入站。
    除此之外，道林镇还积极引导社会贤达积极参与家乡公益事业、乡村治理、产业发展等方面的工作，助力乡村振兴。去年，该镇积极组织发动社会贤达开展慰问活动，累计筹集善款90余万元；引导社会贤达带头捐资近100万元建设美丽宜居村庄，累计带动群众筹资200余万元、筹劳900余个。同时，还组织社会贤达参与“四联五到位”民情大走访，23位社会贤达联户230户，收集意见建议50余条，有效化解矛盾纠纷20余起。社会贤达曾胜洪投资1500万元探索民营经济与村集体经济入股分红新模式，向9个村（社区）合作社筹集200万元入股建设冷链仓储基地，预计正式运营后将促进9个村（社区）集体经济增收超50万元。
    道林镇负责人表示，社会贤达是聚民心、凝共识的乡村振兴因子。下一步，将继续以社会贤达工作站为载体，激发乡贤履职活力，乡贤不闲，当好“八员”，展示贤达风貌，讲好贤达故事，进一步发挥社会贤达群英优势，凝聚共识，激发活力，推动道林镇社会贤达工作朝着更高工作水平的目标迈进，为道林镇发展凝聚人心、智慧、力量。

道林镇：党建引领聚合力、驻村驻心助振兴

发表于 2022-01-11 分类于 乡镇动态 本文字数： 1.3k 阅读时长 ≈ 1 分钟

宽敞平坦的道路、干净整洁的广场、风景如画的美丽屋场，走进宁乡市道林镇善山岭村，一幅环境优美、风景秀丽的美丽乡村画卷正在徐徐展开。
    善山岭村是典型的合并村，一些涉及干部管理、民生事务等方面的遗留因素，且存在基础设施公共服务欠账多、脱贫攻坚期内投入少的现实情况，是道林镇唯一的湖南省乡村振兴重点帮扶村。
    今年5月，宁乡市委组织部为善山岭村选派了乡村振兴驻村工作队，几个月时间，工作队结合村里的实际精准发力，抓党建、聚民心、促发展，推动乡村振兴战略实施高点起步、良好开局。

抓实抓好基层党建 凝聚更强发展合力

善山岭村的驻村第一书记喻夕东作为一名党龄34年的党员，曾在脱贫攻坚期间担任两年驻村第一书记，深知抓实基层党建、发挥党员示范带头的重要作用。
    驻村之后，他指导配合支村两委班子，按照“一支部一特色一品牌”工作思路，强化组织保障，凝聚组织力量，着力打造善山岭村樟林坳屋场党员活动中心，成立村级党员志愿服务队伍。他们既是交通安全劝导员、各类政策法规宣传员，又是美丽屋场建设调解员、基础设施建设参与者，还是关心困难党员群众的贴心人、关爱困境儿童的守护者。
    “作为村里面的无职党员，找到了对自身身份的认同感和荣誉感。”善山岭村老党员蒋森林说。依托“四联五到位”民情大走访、“五个到户”工作、“我为群众办实事”实践活动等，驻村工作队与支村两委班子对全村296户村民进行了遍访，召开了47场屋场夜话，收集解决各类民生实事问题36项，并逐户进行疫情防控、交通安全、反电信诈骗等政策宣传。通过闲话家常、谈心交流，掌握了群众生产生活情况，拉近了党群干群关系；通过帮办实事、帮解难题，解决了群众“急难愁盼”问题，提升了群众的获得感、幸福感；通过问计于民、问需于民，明确了今后工作努力方向，也提升了群众参与村级治理的积极性，善山岭村党总支2021年获评“宁乡市先进基层党组织”。

发挥自身驻村优势 精准助力乡村振兴

“善山岭村产业发展潜力大，又是省级乡村振兴重点帮扶村，我想到自己的派出单位宁乡市农业农村局是善山岭村乡村振兴的联点单位，自己原来又是下属畜牧水产事务中心从事养殖相关工作的职工，可以结合善山岭村的实际情况，发挥联点单位的优势，在助力乡村振兴方面精准施策。”善山岭村驻村第一书记喻夕东欣喜地介绍道。通过联系宁乡市农业农村局下属畜牧水产事务中心，实施基层农技推广补助项目，在全村遴选能力较强、乐于助人的新型农业经营主体带头人、养殖大户、乡土专家等25户作为农业科技示范主体，以资金奖励、技术支持等扶持其不断发展壮大，提高自身对周边农户的辐射带动能力，目前已带动周边小农户70户，实现了产业增值约200万元。
    截至11月底，善山岭村集体经济收入实现了40.76万元，预计全年能实现村集体收入42万元以上，全面实现了乡村振兴战略实施高点起步、良好开局。

道林镇：“探头”将精准监督延伸至村级“末梢”

发表于 2022-01-04 分类于 乡镇动态 本文字数： 1.3k 阅读时长 ≈ 1 分钟

“残疾人子女补助公示（大学生、高中生）；低保名单公示；三四级困难残疾人生活补贴公示……”近日，记者在宁乡市纪委工作人员的带领下来到道林镇金华村，看到该村纪检组长许巧林正在宣传栏旁边记录2021年12月第4周三务公开情况。
    许巧林手里拿着的正是道林镇纪委深入基层监督的“探头”。2021年道林镇村（社区）“两委”换届工作圆满完成，一批政治过硬、敢于监督、作风正派的高素质人才加入村级纪检组织。但因人员变动较大、监督业务水平良莠不齐、村级事务监督发展不平衡，为破解村级纪检组织“不会监督”难题，道林镇纪委创新机制，梳理村级纪检委员监督清单和运行机制，编制了一本监督纪实手册《卫士手记》，在该镇9个村（社区）推行，将精准监督延伸至村级“末梢”。
    烟叶产业是金华村的重要产业之一，2021年5月，因烤烟需求，急需对进出元沙组烤烟房的道路进行拓宽硬化，由于时间紧、任务重，金华村支村“两委”班子成员将该项目列为“一事一议”项目，通过召开多次会议，依程序立项审批后，将元沙组750米道路拓宽硬化工程承包给了村民蒋春高。
    那时，许巧林也成了最忙碌的人，她往返于村部和道路硬化施工现场，监管工作质量和进度，发现问题立马叫停、返工，并将监督情况一一记录在《卫士手记》中，及时公示。

蒋春高回忆道：“那时候因为工期短、时间紧，许组长以及村上其他干部每天来监督工程质量和进度，幸好有他们的监管，我们才按质按量竣工交付。”
    “忠诚履职 规范程序 强化监督 防患未然”，《卫士手记》封面上16个大字格外醒目，让人时刻保持警醒，牢记工作职责。翻开《卫士手记》，记者看到内容包括村级纪检组织成员名单、村级纪检工作管理制度、纪检委员工作职责等内容，要求纪检委员每周按实逐项记录监督工作，由联片的镇纪检干部不定期抽查。召开会议情况及典型案例通报情况、村（社区）重大事务监督情况、“三务”公开、廉情信息收集情况、处理本级信访情况等一一装入其中，一目了然。
    “《卫士手记》的推行，让道林镇的纪检监督工作更好地规范了日常履职记载，强化了内容的真实准确，保持了工作开展的持续完整。”道林镇党委委员、纪委书记刘军告诉记者，通过对照《卫士手记》填报情况，既能排查村（社区）当前可能存在的矛盾隐患和廉情风险，还将《卫士手记》纪实情况列入镇对村党风廉政建设考核内容，结合村情，合理确定各村纪检工作经费，实行一定的激励约束制度。通过“每月点评立标杆”“年度打分评优劣”的差异化考核方式，激励纪检委员忠诚履职，促进道林镇村级纪检工作平衡发展。
    “只有擦亮监督‘探头’、创新监督方式，才能推动纪检监督工作由‘虚化’转向具体化、常态化。”刘军说：“2022年，道林镇将进一步规范小微权力的运行，进一步完善村民自治机制，提升基层治理水平。”
