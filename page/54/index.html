市人社局验收道林镇就业扶贫车间

发表于 2020-01-15 分类于 乡镇动态 本文字数： 177 阅读时长 ≈ 1 分钟

1月14日，宁乡市人社局、扶贫办到道林镇验收扶贫车间项目。

验收组来到靳水村察看居家式扶贫车间项目——LED灯线板制作。他们来到从事该生产项目的贫困户家中，看到贫困户正在熟练地穿插灯线板，验收组详细询问了解了贫困户的就业工作情况，并核实了相关情况资料。之后验收组对该项目负责人就验收工作进行了详细的业务指导。

据悉，道林镇扶贫车间共吸纳6户11名贫困户就业。

道林镇组织政协委员、社会贤达开展“献爱心，送温暖”活动

发表于 2020-01-12 分类于 乡镇动态 本文字数： 215 阅读时长 ≈ 1 分钟

1月12日，道林镇组织辖区内的政协委员和社会贤达开展“献爱心，送温暖”活动。

道林社区柿下塘组贫困户王治勇由于两次车祸导致肢体一级残疾，靠年近72岁的父亲王汉昆照顾，政协委员们详细了解了该户的生活困难情况，并送上了米、油等生活物资和慰问金。

据了解，9位爱心人士共自发捐款35000余元，将在近期将这笔捐款送到道林镇41户贫困群众和寒门学子手中，为他们解决生活和学习上的困难。当天道林镇还组织政协委员和社会贤达开展座谈，共话道林发展。

道林镇卫生院开展消费扶贫活动

发表于 2020-01-04 分类于 乡镇动态 本文字数： 399 阅读时长 ≈ 1 分钟

道林镇卫生院是道林镇华鑫市村后盾帮扶单位，春节临近，道林镇卫生院组织开展消费扶贫活动，集中收购贫困户家中的农副产品，为贫困户增加收入。
    近日，道林镇卫生院驻华鑫市村第一书记张进和镇派常驻队员张志强早早的来到了村部，经过之前摸底，村里共有6户贫困户有销售自家农副产品的意愿。为免去贫困户农副产品运输的麻烦，驻村工作队和村干部找来了一辆运输车逐一到6户贫困户家中进行收购，收购结束后资金统一由道林卫生院转入贫困户明白卡中。
    此次道林卫生院共集中收购华鑫市村谭志刚等6户贫困户大米5000斤、土鸡406斤，按照收购价大米3元/斤、土鸡35元/斤计算，6户贫困户户均收购4868元农副产品。看到自己辛苦一年的成果有了收获，华鑫市村的贫困户纷纷表示来年要加大种养殖力度，争取靠自己的双手脱贫致富。

动真格！道林镇多管齐下治理“人情风”

发表于 2019-12-02 分类于 乡镇动态 本文字数： 857 阅读时长 ≈ 1 分钟

“整治人情风，人人有责任……”近日，一场别开生面的演出在道林镇道林社区网络课堂大厅展开。节目借鉴花鼓戏《张先生讨学钱》的曲调，以快板加花鼓小调的形式，唱词则完全是以当前居高不下的婚丧喜庆随礼“人情风”为内容，劝导大家共同抵制“人情风”。

之前一段时间，道林镇也跟各地一样，十乡八里每逢谁家有婚丧喜庆之类的事情，必定大操大办一番，随之而来的是送礼风气越来越严重，加上村民攀比心理作祟，人情礼金越来越高，一个家庭一年下来最少也要几千元的人情支出，这对于一些困难家庭来说，是一笔相当不小的负担，也让操办酒席的村民不堪重负，是一件双方都费力不讨好的事情。
    针对这种情况，道林镇党委政府一直在思考，如何用群众听得懂、喜欢听的方式来推进移风易俗工作，让更多的群众理解并主动配合。于是，该镇试着以屋场夜话、网络课堂、文艺演出等群众喜闻乐见的方式，在潜移默化中，让文明新风尚在靳江大地生根开花结果。
    当天的花鼓小调由道林镇退休教师许万先创作，自今年创作以来在道林镇各个村（社区）或是红白喜事现场进行表演，目前出演20多场次，通过宣传整治“人情风”，当地的风气有了明显好转。
    “以前随便就是三百元起步，如今好多了，一百元二百元顶多三百元，至少这个风气已经刹住了，没有再高了，这就是一种进步。”道林社区居民张科明对此深有感触。
    道林镇金华村村民李菊生，今年70岁的他，原本打算办一场酒席好好庆祝一番生日的，结果后来主动放弃了大操大办。“没受地方人情（礼金），也没受亲戚人情（礼金），就请亲戚吃了一顿饭。”李菊生觉得这样的方式挺好。
    今年以来道林镇共召开村组移风易俗宣讲活动60余场次、设立移风易俗公益广告142处、发放宣传册5000份营造了全镇坚持节俭操办婚丧喜庆事宜的良好氛围让移风易俗成为当地的新风尚！

道林镇迎来发展新机遇！拭目以待！

发表于 2019-11-26 分类于 乡镇动态 本文字数： 705 阅读时长 ≈ 1 分钟

好消息！道林镇迎来发展新机遇！11月24日下午，大汉控股集团与潇湘电影集团战略合作签约仪式在道林镇人民政府举行。
大汉集团董事长、总裁傅胜龙，湖南广播电视台党委委员、湖南广播影视集团副总经理、潇湘电影集团总经理谷良及双方企业相关负责人和代表出席了签约仪式。

签约仪式上，道林镇党委书记李纯、镇长姜晓对双方的到来表示欢迎，道林古镇、靳江源流域、刘少奇故居等旅游景区的人文气息浓厚、历史底蕴深厚、市场前景广阔，热忱期待双方强强联手，共同为道林的建设发展作出新贡献，造福道林人民。
双方负责人深入交流
傅胜龙表示，大汉控股集团作为一家跨地区、跨行业的大型综合性非公有制企业，是中国500强企业，正在倾力打造“道林公社”项目，“道林公社”集旅游观光、休闲娱乐、民俗体验、购物美食、文化演艺、度假康养等服务于一体，是大汉集团近年来重大投资项目，希望能与潇影携手，共同把道林公社打造成为湖湘文化的一个新地标、一张新名片。
谷良表示，潇影集团作为中国七大电影产业集团之一，拥有61年深厚的历史和文化底蕴，拥有强大的市场影响力、号召力；大汉集团在贸易、物流、开发、教育领域及市场运营等方面有着雄厚的实力，希望双方紧密合作，精诚团结，发挥好各自优势，力促各项项目尽快落实落地，为湖南文化事业作出应有的贡献。

双方商定将在产业发展、专业运营、广告宣传、文化创意、空间美学设计和影视艺术教育、国学教育、影院建设等领域互为重要合作伙伴，各取所长，实现合作共赢，共同打造中华优秀传统文化教育基地、道林书院、红色影视拍摄基地、微电影拍摄基地，助力双方取得良好的社会效益和市场效益。
签约仪式后，双方考察了道林古镇民国街、明清巷子、红色影视拍摄基地等地。

道林镇：一面锦旗和一份检讨

发表于 2019-11-19 分类于 乡镇动态 本文字数： 944 阅读时长 ≈ 1 分钟

近日，道林镇主题教育办的工作人员下村督查指导工作时，在金华村便民服务大厅发现了一面写着“为民解难”的锦旗，仔细一看还是“熟人”送来的一面锦旗。

锦旗是村民刘某送来，一名精准扶贫对象。刘某今年女儿大学毕业开始工作，于是顺利脱贫。高兴之余却又多了一件“烦心事”：家里申请厕改，却没有落实到位，是不是因为脱贫了，村上就不关心了？

原来施工队统一进行修建“三级化粪池”时，刘某本人不在家，家人又不能确定修建的位置，于是没能修建。现在施工队单独修建一个化粪池成本过高，于是迟迟未能进场。虽然联村干部和村干部表示一定会协调处理好，但刘某还是执意通过拨打12345市民热线、信访局、联村领导和干部等电话重复反映情况。
    道林镇主题教育办工作人员得知这一情况后，立即将该问题纳入到了金华村主题教育“为民办实事好事”工作，并和刘某的女儿交流，争取到了刘某的初步认可。
    道林镇金华村党总支随即通过支委会议决定特事特办，适当补偿增加的施工成本，限期要求施工队进场并完成修建，确保刘某顺利完成厕改。
    不到一个星期，刘某卫生厕所化粪池就修建好了，刘某对修建速度和质量都很满意。同时，刘某在村部“蹲守”村干部反映问题时更深入了解到村干部的不容易，也为自己当时的不信任感到了不好意思，于是定制了锦旗送来。
    “以前认为群众信访增加了村上不必要的工作量、分散了时间和精力，于是先入为主地将信访对象当作麻烦，在处理信访问题时不自觉地带有抵触情绪，对解决群众反映的具体问题不够积极，这说明我内心还是有一点官僚主义的存在，这不符合党的要求，我以后一定注意，尽心尽力为群众办实事好事。”在与镇主题教育办干部交流时，金华村党总支书记崔泽洪主动做了检讨。
    据悉，自道林镇“不忘初心、牢记使命”主题教育开展以来，该镇积极开展“三单两榜”为民服务，将群众反映的问题和诉求纳入到各党组织主题教育民生实事清单，全镇共形成77项问题，目前已解决68个，进一步浓厚了为民解难题的氛围，人民群众的幸福感、获得感得到不断提升。
