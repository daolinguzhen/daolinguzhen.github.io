道林镇职业技能培训火热进行，今年取得职业资格证书人数已超500人

发表于 2020-10-29 分类于 乡镇动态 本文字数： 883 阅读时长 ≈ 1 分钟

近日，记者了解到，为提高贫困户脱贫致富能力，助力脱贫攻坚，年初以来，在市人社局等部门的组织下，道林镇联合赛兰职校等7家职业技能培训机构为有需要的村民免费提供叉车、电工、面点、焊工等方面的技能培训，贫困劳动力优先报名。截至目前，该镇已开设各类培训班近20个，500多人通过培训获得相关职业资格证书，其中贫困劳动力达200余人。
    10月20日，记者在道林社区一广场内见到，社区居民黄志勇和另一名驾驶员各操作一台叉车，在教练员的耐心指导下，小心翼翼地练习举物、堆放、过限宽门等操作技巧，其他20多名学员则站在场外等待。
    “因为受疫情影响没有外出打工，正好镇里组织开展免费的职业技能培训，我就报名来学习开叉车。等拿到操作证，我准备就在宁乡找一份合适的工作。”黄志勇告诉记者。
    来自大屯营镇大屯营村的村民王灿辉在学员中格外抢眼，作为为数不多的女学员，她的学习劲头丝毫不亚于男学员。“我是从长沙赶过来的，老师手把手教我们，基本上15天就能完全学会。技多不压身，学会了开叉车，我以后找工作更方便了。”王灿辉说。
    “我们学校根据道林镇的实际需求，在这里开办了第二个叉车培训班，培训为期15天，包括2天的理论学习和13天的实操培训。今年，我们已经在夏铎铺、煤炭坝、流沙河、花明楼等乡镇举办了多个培训班，由当地提供场地，我们提供免费培训，今年报名总人数已超过2000人。”赛兰职校教师江利群告诉记者。
    “保就业就能稳脱贫。今年，我们根据市人社局的统筹安排和宁乡的实际用工需求，有针对性的开设了叉车、电工、面点、焊工、育婴师、养老服务等方面的职业技能培训班近20个，贫困劳动力优先报名，现在有超过500人取得了相关职业技能资格证书，其中贫困劳动力超过200人，大部分已经找到了合适的工作。今年预计培训人数将达到600人以上。”道林镇社会事务办主任宁罗军介绍。

道林镇金华村村民龙应禄舍小家为大家

发表于 2020-10-21 分类于 乡镇动态 本文字数： 929 阅读时长 ≈ 1 分钟

记者了解到，10月6日到13日期间，道林镇金华村8组年过50岁的龙应禄不顾疲劳、不分昼夜为当地8个组的村民抢收稻谷近700亩，却让自家60多亩稻谷继续在田中淋雨，其中20多亩已经倒伏发芽，收成锐减。
    10月19日，久雨转晴，记者来到道林镇金华村8组建档立卡贫困村民刘赞平家中，看到房间里堆满了稻谷。
    “前段日子正是收割稻谷的黄金时节，却一直刮风下雨，成熟的稻穗倒了，稻谷浸泡发芽了。我家只有我一个劳动力，天气好的那几天，一个人根本收不了多少。”刘赞平告诉记者，10月6日，雨终于停了，家家户户开始抢收稻谷。10月13日，还剩2亩多稻谷没有收割完，刘赞平从天气预报中得知14日又将下雨，无奈的他，硬着头皮去请同组村民龙应禄帮忙。
    “村上只有龙应禄家有收割机，他自己种了100多亩晚稻，也才开始收割，每天请他帮忙的人太多了。6日开始，他从早忙到晚，没歇过一天。”让刘赞平没有想到的是，龙应禄一口答应了他。
    “13日那天，他帮忙收完别人家的稻谷，到晚上12点了，但他没有休息，开着收割机笔直到了我家田里，一直忙到凌晨两点，摸黑帮我把2亩多稻谷收割了。”刘赞平说，龙应禄不仅及时出手相助，自己还倒贴油钱，没有收取任何费用。虽然2亩多稻田仅收回300多公斤稻谷，但刘赞平没有抱怨，“如果没有龙应禄帮忙，稻谷全会发芽，颗粒无收。”
    据刘赞平介绍，当地还有5户贫困户近20亩稻谷是龙应禄免费收割进仓的。
    当天，记者找到龙应禄时，他正开着收割机在自家田里收割稻谷。记者见到，一丘半亩左右的稻田中，大部分稻穗倒伏，有的稻谷已经发芽，产量不及正常年份的三成。
    “大家有困难，我能帮助，受点损失算不了什么。那几天实在是忙不过来，现在好了，村民们的都收完了，我也有时间了，这几天天气好，我要抓紧时间收割自家稻谷了。”接受完记者采访，龙应禄驾驶收割机渐行渐远……

道林镇建起流动党员“充电站”，既“充电”又“放电”

发表于 2020-10-16 分类于 乡镇动态 本文字数： 664 阅读时长 ≈ 1 分钟

国庆节过后，返乡过节的道林镇龙泉湖村周莎又要返城务工了。村干部及时给她送来了一套《习近平谈治国理政》第三卷、《宁乡市流动党员积分管理办法》等教材，上门为她“补课”。“虽然入党很长时间了，但一直难有时间系统学习。现在，我可以与家里的党员‘同步充电’了。”周莎说。
    道林镇共有流动党员352名，占全镇党员总数的五分之一。今年以来，针对流动党员较为分散、组织学习困难、服务群众时间少的现状，该镇建起了流动党员“充电站”，借助上门授课、远教辅导、实践服务三大课堂，全面提升流动党员为群众服务的能力和素质。“充电站”不仅是流动党员返乡学习“充电”的地方，还是流动党员回乡服务“放电”的平台。
    趁着流动党员回乡过节之机，道林镇各村（社区）的流动党员“充电站”都忙着给流动党员“充电”，“主要是补习党的相关理论知识、《准则》与《条例》等相关内容，同时穿插一些农业方面的知识。”道林镇党委委员、组织委员邹培表示，在开展“充电”学习的同时，该镇还通过召开流动党员创业经验交流会，请流动党员介绍在外工作学习情况和所见所闻，并向他们征求意见建议，鼓励返乡流动党员献计献策。
    据悉，这种集中“充电”，更是一种集中“放电”，通过“理论学习+志愿活动”相结合，全镇流动党员管理有力度也更有温度。目前，道林镇共开展各类流动党员宣讲、教学、培训活动29次，收集流动党员提出的建议102条、金点子38个，解决各类难题40个。

中共宁乡市委第二十七巡察组向道林镇村级党组织反馈巡察情况

发表于 2020-10-15 分类于 乡镇动态 本文字数： 1.4k 阅读时长 ≈ 1 分钟

根据市委统一安排，2020年10月14日，市委第二十七巡察组向道林镇村级党组织反馈巡察情况。市委第二十七巡察组副组长邹彦代表巡察组反馈巡察情况，道林镇党委副书记、镇长姜晓主持反馈会议并作表态讲话。
    根据市委统一部署，2020年4月26日至2020年6月30日，市委第二十七巡察组对道林镇村级党组织开展了巡察。巡察组坚持以习近平新时代中国特色社会主义思想为指导，认真贯彻全国巡视工作会议、全省巡视巡察工作会议精神，积极践行初心使命，把“两个维护”作为根本任务，紧扣“三个聚焦”，围绕中央、省市县委重大决策部署，紧盯群众切身利益问题，通过开展个别谈话，受理群众来信来访，调阅有关文件资料，深入了解情况，顺利完成了巡察任务。
    邹彦指出，近年来，道林镇所辖村（社区）党组织不断强化基层党组织建设、脱贫攻坚、生态文明建设、村级项目建设等工作，乡村振兴取得了阶段性成就。巡察也发现了一些问题，主要是：脱贫攻坚、乡村振兴战略政策落实不到位；基层治理落实不到位，效果不明显；意识形态工作不够重视，村级精神文明建设领导不力；“三资”管理廉政风险较为突出；违反中央八项规定精神仍然存在；党组织不健全、凝聚力不强；党建制度落实不到位；党建工作基础薄弱，党员教育管理缺失。
    邹彦提出了两个方面的整改意见。一是向道林镇村级党组织提出的意见：要关心关注群众集中反映的利益诉求，抓好信访及矛盾纠纷调解，增强基层吸附力，做到“小事不出村、大事不出镇、矛盾不上交”；要提高对党建工作的认识，高标准落实好“三会一课”等党建工作制度，推进支部“五化”建设，切实发挥支部战斗堡垒作用和党员先锋模范作用；要加大“三大攻坚战”落实力度，特别是在脱贫攻坚、污染防治工作中进一步明确思路，务实作为，提高群众满意度。二是向道林镇党委提出的意见：要进一步加强村级党组织班子建设，注重后备力量培养，提前做好换届选举准备工作；要进一步引导村级组织利用区位优势，优化村级经济结构，发展壮大村级集体 经济；要进一步加强廉政风险防控，定期组织村级村务项目清查，做到项目招投标规范管理。
    姜晓表示，镇党委对巡察组反馈的意见和建议全盘照收，将对照“问题清单”逐条逐项抓好整改落实。一是要统一思想、深化认识。深刻认识巡察反馈意见的重要作用和重大意义，切实把思想和行动统一到巡察反馈意见的精神和要求上来，把巡察组的巡察反馈意见当作一次洗洗澡、治治病、优化政治生态的机会，不折不扣地落实好各项整改任务。二是要明确责任、强化整改。坚持“整改不力是失职，不抓整改是渎职”的原则，明确整改内容、整改目标、责任主体和整改时限，进一步落实责任，逐条从严、从快、从实整改到位。三是要立足长远、整章建制。牢固树立党风廉政建设主体责任意识，做到真管真严、敢管敢严、长管长严；坚持把落实整改与作风建设结合起来，严格贯彻执行中央八项规定，严防“四风”问题反弹回潮。
    市委第二十七巡察组成员、道林镇党政班子成员、联村干部、机关全体中层骨干、部门单位负责同志、全体村（社区）干部、县级以上党代表、人大代表、政协委员和退休老同志代表列席会议。

家门口有条幸福路

发表于 2020-10-02 分类于 乡镇动态 本文字数： 1.1k 阅读时长 ≈ 1 分钟

9月中旬的三湘大地，秋高气爽。记者来到湖南省宁乡市道林镇龙泉湖村，这里一片湖光山色。
    对于龙泉湖村村民朱战武来说，家门前曾经的那条土路让他记忆犹新。“那条路真的是‘晴天一身土，雨天一身泥’。2019年4月村里修好路，我的幸福感一下子就提高了。”如今，朱战武家种的50多亩黄桃不用运出去，每天来采摘的就有100人至200人，加上原来种植的50亩水稻，一年能收入30万元左右。
    像朱战武这样致富的贫困户，在宁乡市还有很多。宁乡市道林镇石金村建档立卡贫困户张建秋身有残疾，家庭主要收入依靠生猪养殖，在村道路未拓宽硬化前，生猪出售需用拖拉机转运到主公路边才能装车运输，不但费时费力，也卖不了好价格。自从村公路拓宽硬化后，生猪收购车能直接开到养殖场，降低了运输成本，提高了销售价格，他家年收入直接增加5万元以上。张建秋常说：“如果没有修好这条路，我的养猪事业就快半途而废了，公路修好以后，我养猪的劲头越来越足，幸福感和获得感也不断提升。”
    公路拓宽硬化后，石金村村民刘湘购买了一台农用车跑运输，年收入7万至8万元，儿子还购买了一台挖掘机，年收入15万元，整个家庭一年总收入22万元。“感恩党和政府的好政策，让我们的生活越来越好。”刘湘说。
    吉首市矮寨镇幸福村紧邻湘川公路边的矮寨德夯景区服务中心，高速公路通车7年来，幸福村发生了翻天覆地的变化。
    村民刘山在景区服务中心对面开了一家特色小吃店。“高速路通到家门口，乡亲们都开始做些小生意，农家乐和旅馆随处可见。越来越多的村民返乡挣钱，也方便照顾家人。”刘山说。
    据了解，湖南省实施25户/100人以上自然村通水泥（沥青）路建设工程，共完成道路建设4.37万公里，直接惠及3.6万个自然村。
    湖南省交通运输厅农村公路建设处处长喻波介绍，近3年的建设充分表明，通组道路建设切实提高了脱贫攻坚的成效，极大提升了群众获得感和满意度，为打赢脱贫攻坚战、全面建成小康社会提供了有力支撑。目前，湖南省参与农村公路养护工作的农民群众超过2.3万人，其中建档立卡贫困群众超过1万人，每月增收800元至1500元，既帮助贫困户稳定脱贫，又建立起稳定的养护队伍。“预计到2022年，湖南省将基本形成权责清晰、齐抓共管的农村公路管理养护体制机制，形成财政投入职责明确、社会力量积极参与的管养新格局。”喻波说。

道林镇三举措抓实道路交通顽瘴痼疾整治工作

组组通、百业兴，宁乡通组公路建设获全国主流媒体集中采访报道

发表于 2020-09-16 分类于 乡镇动态 本文字数： 1.2k 阅读时长 ≈ 1 分钟

9月11日，中国交通报社党委委员、副社长庄长波及新华社、中央人民广播电视台、经济日报、澎湃新闻等记者一行集中采访考察了宁乡市道林镇龙泉湖村、石金村通组公路建设。
    “修成了这条路，真的大大方便了我们当地老百姓，出行就不用说了，生活更有盼头了，现在我弄了个果园，有50亩的样子，估计年收入30万元，附近还有办起了农家乐的，开起了打米厂的等，真是觉得日子过得越来越有盼头，很幸福，”宁乡市道林镇龙泉湖村（原烧汤河村）群联组朱战武说道。
    烧汤河村组路1.36公里，由当天群众投工投资整理出合格的路基，交由长沙市路桥公司建设，投入资金约80万元，于2019年4月完工通车。龙泉湖村（原烧汤河村）位于三县交界区，全村由56个村民小组组成，6813人，是农村的桥头堡。但是，由于基础设施特别是交通设施滞后，致使农民收入不高，年轻人大都纷纷外出打工。2019年实施自然村通水泥（沥青）路以来，特别是烧汤河村村组路项目，全长1.36公里，贯穿4个村民小组，受益群众达到500余人，是当地人民群众盼望多年的惠民路、幸福路。项目建成通车后群众满意度非常高，拉动了当地经济的发展，外出务工的年轻人也纷纷回乡创业，抓住道林古镇开发的契机，开起了农家乐、民宿，在家门口就实现了就业、致富。
    通树湾至清泉寺通组公路，全长3.372公里，涉及9个组，受益群众千余人。原来是一条不足2米宽坑坑洼洼的砂石路，严重影响当地居民生产生活，人居环境脏、乱、差现象非常严重，学生上学叫苦不堪，群众生产生活十分不便，怨声载道，信访不断。
    2019年4月实现全线通车，项目的实施给沿线群众带来了翻天覆地的变化。如建档立卡户张建秋系该村养殖户，公路未拓宽硬化之前，生猪出售需用拖拉机转运到主公路边才能装大车，不但费时而且价格低，买不了好价钱。如今收购车能直接开到养殖场，大大减少了运输成本，价钱也提高不少，据他介绍修路给他带来的经济效益达到5万元以上。受益的不只有张建秋一个人，沿线农户的农产品由于公路直达家门口，前来收购的客商络绎不绝，一点也不愁销路，在家门口就能将农产品销售出去卖个好价钱了。
    宁乡市自然村项目实施统建统管模式，由长沙市级交通运输局统一招标，长沙市公路桥梁建设有限责任公司统一建设、县（市）级交通运输局统一管理，工程实施零利润，除省市补助外剩余缺口资金由县级财政兜底，做到了模式有创新、利润有控制、资金有保障。从2018年至今，省规自然村通水泥（沥青）路共下达计划691.731公里，目前已完成678.635公里，剩余13.096公里预计在今年10月底前完工。

精准帮扶到村头、电力扶贫暖人心

发表于 2020-09-10 分类于 乡镇动态 本文字数： 740 阅读时长 ≈ 1 分钟

“真是太感谢你们了，多亏你们的帮扶，我们村变得越来越美好!”9月8日上午，国网宁乡市供电公司迎来了一批“特殊”的客人——道林镇政府相关负责人、鑫星村支村两委及村民代表一行专程带着锦旗，前来感谢该公司一直以来在扶贫村电网建设、脱贫攻坚中的积极贡献。
    鑫星村位于宁乡市道林镇北部，共有1786户5890人。为解决该村群众用电困难，帮助困难群众脱贫致富，用上放心电、致富电，国网宁乡市供电公司加大电力扶贫支持力度。近3年来，针对鑫星村电力设施老化，电网结构较为薄弱的情况，新增变压器13台、改造7台，新建改造10千伏线路4.17千米、低压线路18.19千米，户均配电容量由1.42千伏安提升至2.82千伏安，最大供电半径由1.67千米下降到0.79千米，大幅提升了该村的供电能力和供电可靠性。
    该公司积极履行社会责任，在该村派驻驻村第一书记，持续深化帮扶工作，鼓励犹豫不决的农户走上产业脱贫的道路，引导缺乏技能知识的村民掌握一技之长，保障已经脱贫的户子继续感受政策的温暖，帮助贫困学子走进学校课堂，守护被病痛折磨的贫困户与病魔抗争到底……同时，还着力解决村民最关心、最直接、最现实的利益问题，从衣食住行到文化生活，精心策划帮扶措施，对贫困户进行产业扶贫、教育扶贫、消费扶贫，为解决村民发展难、就医难、吃水难，打好精准扶贫组合拳。
    该公司相关负责人表示，一直以来，国网宁乡市供电公司积极响应政府号召，积极履行“人民电业为人民”的企业宗旨，落实各项电力扶贫措施，争取项目资金，加大电网改造力度，为助力当地脱贫致富、实现美好生活提供了坚实有力的电力保障。

道林镇善山岭村召开交通问题顽瘴痼疾整治宣教会

发表于 2020-09-09 分类于 乡镇动态 本文字数： 561 阅读时长 ≈ 1 分钟

为进一步加大道路交通问题顽瘴痼疾整治宣传力度，2020年9月8日，善山岭村开展了一场别开生面的交通问题顽瘴痼疾宣教会，会议由善山岭村村主任王青山主持，镇整治办、中小学校长、幼儿园园长、党员、志愿者共同参与。
    善山岭村党总支书记邓志坚在会上指出善山岭村地理位置特殊，过往车辆多、车速较快，极容易发生交通事故，并对前期在学校建立交通问题宣传台账，在学校播放安全教育警示片、悬挂横幅和展板、向每一位家长发放宣传资料，鼓励学生们把学到的交通安全知识讲给父母及身边人并监督好家长接送孩子路上佩戴头盔（或系安全带），利用微信、村村响、屋场夜话户主会等方式向村民普及交通安全知识等工作开展情况作了简要总结，在会议中，邀请了一位7月份发生车祸的事故当事人以自身的经历现身说法，给大家宣传摩托车戴帽的重要性，同时倡议邻居朋友们做到遵规守法，一起做“文明交通”的践行者。
    道林镇整治办结合善山岭村实际情况，在下阶段如何推进交通问题顽瘴痼疾整治工作时强调：一、重点排查道路安全隐患，对已经排查出的隐患路段及时进行整改，消除隐患，降低行车风险；二、着力整章建制，强化“两客一危”车辆安全监管，严格落实驻点治超和流动治超工作措施；三、加强交通安全知识宣传。
