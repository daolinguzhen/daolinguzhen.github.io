道林镇：筹资又投劳，美丽家园一起造

发表于 2021-08-24 分类于 乡镇动态 本文字数： 3.3k 阅读时长 ≈ 3 分钟

乡村振兴，生态宜居是关键，良好的生态环境是农村的最大优势和宝贵财富。
    市第二次党代会指出，要着力打造宜居安宁幸福城，让人民群众共享美好生活、共享和谐安宁、共享文明成果。道林镇严格落实党代会精神，紧紧围绕实施乡村振兴战略总要求，着力推进美丽宜居村庄建设，道林社区窑岭组、金华村歧界山组等地党员群众齐心协力，用自己勤劳的双手建设美丽家园，一幅幅“村庄美、产业强、村民富”的乡村振兴画卷正泼墨挥就……

【窑岭美丽宜居村庄】

    提升家园颜值，融入古镇景区
    8月18日，记者来到道林社区窑岭美丽宜居村庄建设现场，看到10多名工人正手握砌刀，在烈日下挥汗如雨，一条近500米长的文化长廊在他们的辛苦劳动下初见雏形。他们的身后，是碧草如茵的绿化带、干净整洁的农家庭院、宽阔的马路和涟漪浅漾的一池碧水……
    “原来连摩托车都开不进来的小路，现在可以开进小车了。只用了一个月时间，这里的变化真大！”窑岭组村民崔先亮告诉记者，原来这里只有一条不足一米宽的小路，路面坑洼不平，出行很不方便，而且，农户的家禽都是散养，弄得周边环境脏乱差。自从开始建设美丽宜居村庄，路变宽了，鸡鸭进笼了，房前屋后清理干净了，环境变好了，昔日冷冷清清的路上早晚散步的人不断。“路过的邻里乡亲都喜欢来我家坐一坐，喝口茶，我每天都要准备好几瓶开水，家里很热闹。”崔先亮笑着说。
    “社区刚开始提出建设美丽宜居村庄时，不少人还不明白是怎么回事，我们就组织群众外出观摩学习，同时邀请专业设计公司进行科学规划设计。并先后组织召开多次屋主会，动员群众统一思想。”道林社区党总支书记杨红军介绍，为了推进道林古镇景区周边区域旅游配套服务水平提升，扩大旅游经济辐射效应，该社区选择了窑岭组作为美丽宜居村庄建设示范点。7月12日，杨红军组织窑岭组村民居民又开了一次动员会，弄明白了是怎么一回事的居民纷纷当场表态，将全力支持美丽宜居村庄建设。7月14日，窑岭美丽宜居村庄建设工作正式启动。
    在窑岭美丽宜居村庄建设指挥部，记者看到了一幅工期倒排作战图，大大小小40多个子项目铺排一目了然，包括拆违治乱、小微水体及污水治理、公共设施建设等10个大项。
    “从开工之日起，项目建设就没有停，最辛苦的是社区党员干部、项目建设理事会成员，他们没日没夜地在工地上操劳，保证工程进度和施工质量。附近居民一有空都来帮忙，只想早点把这个美丽屋场建好。”崔先亮告诉记者。
    “到目前，群众、乡贤等捐资已有55万多元，村民义务投工达到200多个，整个项目预计在9月底完工。”杨红军告诉记者，窑岭是革命烈士曾凤琪、蒋国良的家乡，他们都曾经为毛泽东提供过帮助和保护，道林社区充分挖掘红色故事，在美丽宜居村庄建设中融入文化元素，通过文化长廊来展示红色底蕴、村规民约、古镇文化等内容，并依托道林古镇核心景区，大力发展民宿、特色餐饮、休闲旅游等产业，实现村民家园美、生活富的目标。
    记者离开后，杨红军、崔先亮和周边的居民又马不停蹄地拿起扫把、提着水桶，开始清洗起道路来……

【歧界山美丽宜居村庄】

党建引领村庄建设，唤醒群众主人翁意识
    5月底，金华村支村“两委”班子成员来到歧界山召开美丽宜居村庄建设宣传发动会，但参会群众对此不冷不热，持观望态度；6月2日，该村组织户主代表到菁华铺乡陈家桥村参观，见到的是一片山清水秀、路畅人欢的美丽景象，大家了解到陈家桥村通过筹资投劳建设美丽屋场，不仅家门口变漂亮了，而且邻里之间和睦相处，处处充满着欢声笑语。
    村民们开始心动了。6月3日，歧界山组村民又聚在一起，对于美丽宜居村庄建设，大家从观望态度变成了积极支持。本着“谁受益，谁筹资投劳”的原则，村民通过了歧界山美丽宜居村庄建设方案，并选出3名项目建设理事会成员，党员崔龙庆为会长。“一定会尽自己最大的能力搞好美丽宜居村庄建设，不辜负群众的信任。”崔龙庆当场立下了“军令状”。
    “组上8名党员第一时间站出来，白天带头进行除杂清淤，晚上上门动员群众拆除围墙、让出土地，收拾整理自家房前屋后环境卫生。”崔龙庆告诉记者。
    村民许冬华养鸭30多年，家里有100多羽鸭子，驻村干部帮他全部销售出去，他也配合理事会工作拆除了鸭棚；因为他家的菜地靠村庄广场中心位置，他又无偿让出部分菜地用来扩宽道路；村民欧文武、周春元无偿拆除了自家围墙，欧文武还让出土地用来拓宽道路、建设绿化带；村民许夕均、许界胜、张海清拔掉自家菜地的蔬菜，支持美丽宜居村庄建设；70岁的老党员付楚华自告奋勇监督项目施工、动员群众，并通知在外地工作的儿女共捐款5000元……
    为了做到钱款、用工透明公开，打消村民疑虑，理事会还成立了财务监督委员会，从46户村民中选出监督组成员，每个子项目建设都由理事会、监督组签字，并交村委会审核，再进行公示，群众的筹资用途、投工等情况一目了然。
    “除了没有智能手机的老人和小孩，屋场其余132人全部加入‘歧界山美丽宜居村庄’微信群。”崔龙庆介绍，理事会每天白天在群里及时通报工作动态和工程进度，晚上将第二天的工作安排发布在群里。慢慢地，群众参与积极性越来越高，大家都把这个群当成沟通联系的重要纽带。
    “大家辛苦了，我马上送西瓜给大家解渴。”
    “许习君娭毑是我们的榜样，她老人家鼓动在外工作的子女捐款1万元，还为我们送茶水、送西瓜……”
    “明日工作安排，一组张少林（另加三个请的师傅）……”每天微信群里热闹非凡，建设工地现场更是一幅争先恐后集体出工、汗流浃背的劳动场景。
    “到目前为止，村民已经筹资10多万元；投劳172个大工，小工245个工日，折合工资63800元。”崔龙庆告诉记者，歧界山美丽宜居村庄建设已经顺利完成拆违治乱、农户房前屋后的卫生清理、道路两旁和主要地段的绿化、广场和其他挡土墙的护砌等工作任务，道路黑化、广场硬化、亮化等工作马上就要进行，一个全新的歧界山美丽宜居村庄即将亮相。

在歧界山美丽宜居村庄旁边，记者看到了一个近20亩的葡萄园，葡萄架上挂满了串串珍珠般的果实。“我希望这个美丽宜居村庄早日建成，这对葡萄销售有很大帮助。下一步，我想扩建葡萄园，扩大水稻种植面积，让更多的人来我们歧界山游玩、采摘葡萄，感受幸福的农家生活。”在歧界山种植了5年葡萄的崔湘乾还种植了几十亩水稻，美丽宜居村庄的建设让他看到了更加美好的明天，他虽然身体残疾，但依然踊跃捐款，并为美丽宜居村庄建设工作提供一切力所能及的帮助。
    “我们在美丽宜居村庄建设工作中，坚持‘三确’（定位准确、分工明确、执行正确）和‘三主’（政府主导、村〔社区〕为主体、提高群众主人翁意识）原则，实施‘三包’（镇领导包村、村支书包点、村干部包项目）机制，以打造“四园”（美丽菜园、果园、家园、田园）为目标，充分发挥党建引领作用，党员干部带头奉献，群众共同参与，形成共建共治共享的乡村治理新格局。目前，全镇5个美丽宜居村庄正在加快建设当中。”道林镇相关负责人介绍。

全员出动！道林镇党员带头“护航”疫苗接种

发表于 2021-08-17 分类于 乡镇动态 本文字数： 1k 阅读时长 ≈ 1 分钟

近日来，道林镇迅速推进疫苗接种，全力以赴构筑全民免疫屏障。该镇党员切实发挥先锋模范作用，带头接种疫苗，并做好疫苗接种志愿服务工作，用心服务接种疫苗群众。
    八旬老党员成为疫苗接种“推广大使”
    “你们尽管放心地去接种呢，我打了几天了，除了打完那天胳膊有点胀痛外，没其他哪里不舒服。打之前卫生院会先给你们做个健康检查，如果不适合打，他们也不会拿你的身体开玩笑的。”最近几天，道林镇金华村83岁的老党员龙文斌成了疫苗接种的宣传志愿者，看见街坊邻里就问打了疫苗没有。前不久，他终于如愿在道林镇卫生院接种了新冠肺炎疫苗，“我家人起初担心我年纪大了，怕接种疫苗有啥不良反应，可是经常看到新闻里好多年纪比我还大的老人打了都没事，在我强烈要求下他们终于带我去打了这针。”自己身体力行后，再去做别人的思想工作就容易多了，在他的影响下，附近又有几位老人也陆续去接种。
    新党员为群众送清凉
    “这个棚子搭得好，这一大片都不要晒太阳了。”在道林镇卫生院前坪接种群众休息等候区，最近新增了一个超大遮阳棚和鼓风机，这是道林社区七月才转正的党员黄丹义务贡献的，黄丹在参与疫情防控巡逻的过程中发现卫生院接种的人比较多，天气一热，大家都挤到摩托车棚改造的休息区吹电扇，容易造成人员扎堆聚集，维持秩序比较费力。
    于是他主动将自家做红白喜事用的大型遮阳棚和鼓风机搬到现场，搭建后阴凉面积扩大到了原先三倍，可以实现座位间隔一米摆放，等候群众从此不会扎堆挤在一起。
    女党员义务接送群众打疫苗
    “明天有想去卫生院打疫苗的可以跟我联系，包接送。”华鑫市村女党员佘志群看到本村有村民想去接种疫苗，却没有车过去，顶着炎炎烈日走路又会中暑，于是她得空的日子就会提前跟附近村民打好招呼，约好时间用自己私家车接送村民们去打疫苗。
    据悉，该镇在疫苗接种工作上广泛宣传、大力发动，镇村网格干部下沉扎实开展摸排，以党建为引领，大力推进全民接种工作进度，截至目前，该镇疫苗第一剂次接种26585人，累计接种48146人次。

道林镇全力开展新冠肺炎疫情防控工作

发表于 2021-08-11 分类于 乡镇动态 本文字数： 619 阅读时长 ≈ 1 分钟

道林镇结合实际情况，及时调整新冠肺炎疫情防控指挥部成员及分工，统筹调度全镇疫情防控工作，镇村两级紧密配合，精准施策，全力做好新冠肺炎疫情防控工作。
    一是做好政策宣传发动。通过“村村响”广播、LED显示屏标语全天滚动宣传政策。镇村干部全部下沉到网格加强宣传及人员摸排。道林社区党员志愿者还成立了一支义务巡逻队，协助摸排及巡查。
    二是做好人员摸排管控。通过村（社区）摸排和大数据反馈，对中高风险地区返回管控对象分别采取集中隔离、居家隔离、居家健康监测，全部落实“六包一”管理措施到位。
    三是推动疫苗接种进度。号召应接种对象应接尽接。15-17岁对象除因禁忌症或其他特殊情况不能接种外，已全部完成接种，15岁以上人员累计接种率91.7%。对12-15岁对象正有序组织集中定点进行疫苗接种。
    四是加强经营性场所管理。联合职能部门组织对经营性场所落实疫情防控措施每日进行三次巡查，督促店主佩戴口罩消毒，引导群众戴口罩，从严防控疫情。上门到139户经营户张贴疫情调度令、防控指导；关停1家游泳馆、1家网吧、4家采耳（足浴）店、6家培训机构、16家大小KTV、120家麻将室。关停整改不符合防疫要求的餐饮店10家。镇域内两家养老机构严格按照疫情防控要求进行封闭式管理。

道林镇多管齐下加强疫情防控

发表于 2021-08-02 分类于 乡镇动态 本文字数： 487 阅读时长 ≈ 1 分钟

7月29日，自宁乡市通报发现新冠病毒核酸阳性检测者开始，道林镇迅速落实新冠肺炎疫情应急响应，多措并举，多管齐下加强疫情防控工作。
    科学调度，加强管控。镇疫情防控工作指挥部迅速落实宁乡市防指调度令精神，将任务分解到办到人，确保工作实效。切实加强人员管控，对中高风险地区返乡人员、密切接触者，立即报告，并督促其进行核酸检测，按照“六包一”工作要求进行人员管控。
    广泛发动，集中宣传。各村（社区）利用“村村响”广播、微信群等途径，大力宣传疫情防控知识，号召群众减少非必要外出和人员聚集行为。
    大力劝导，及时整治。7月29日晚，镇综合行政执法队组织连夜关停镇域内所有夜宵店和娱乐场所，并与餐饮门店签订疫情防控承诺书，截至当晚凌晨1:18，全镇所有聚集性经营场所全部暂停营业。7月30日，副镇长吴明兴组织政府相关线办和市监所对集镇经营场所和人员密集区域进行督查，确保各项疫情防控措施落实到位；各村（社区）组织对麻将馆等人员聚集场所下达停业通知书。

道林镇与长沙医学院联合开展暑期“三下乡”青年社会实践活动

发表于 2021-07-26 分类于 乡镇动态 本文字数： 307 阅读时长 ≈ 1 分钟

为深入学习贯彻习近平总书记“七一”重要讲话精神，道林镇团委联合长沙医学院“青春映像”三下乡团队开展“党建带团建、青春心向党”社会实践活动。实践活动包括“老人免费体检、辅助疫苗接种、科普支教服务”等服务项目，一周来，志愿者们累计发放科普资料1000余份，服务老人、学生人数达2000余人，赠送感冒药、清凉油等药品800余份，得到了广大群众的赞誉。

7月19日，长沙医学院联合道林镇团委召开相关人员交流座谈会，就“三下乡”志愿服务、团建共建合作等工作进行交流，长沙医学院马克思主义学院院长周建兰、副院长邓明丽、共青团宁乡市委副书记陈俊磊、道林镇组织委员邹培参加交流会。会后，周建兰院长为与会青年上了一堂精彩、生动的党课。

道林镇：推行“五长”负责制，筑牢防溺水“安全网”

发表于 2021-07-21 分类于 乡镇动态 本文字数： 588 阅读时长 ≈ 1 分钟

连日来，道林镇结合“我为群众办实事”实践活动，多措并举，织密织牢未成年人防溺水“安全网”，严防溺水事件发生，切实保障学生生命安全。
    高度重视，形成合力。组织召开全镇未成年人防溺水工作推进会，及时传达上级有关文件精神，对全镇防溺水工作进行周密安排部署，将责任压实到人。建立“党委书记、镇长、校长、村长、家长”的“五长”负责制，筑牢“政府、社会、学校、家庭”四道“责任防线”，形成上下联动、齐抓共管的工作格局。
    加强教育，广泛宣传。持续抓好防溺水“六个一”警示教育活动，坚持每天放学时段1分钟、长假前5分钟安全教育。学校从4月份开始，每天上、放学高峰时段校门口常态播放防溺水等安全教育音频内容。每周通过家校通、微信群、QQ群等方式，定期下发各种形式的防溺水提醒资料。各村（社区）利用“村村响”“屋场夜话”及雷锋志愿者串家入户等方式开展针对性宣讲，在公共文化场所、易发溺亡点路口显著位置等处悬挂防溺水宣传标语。
    全面排查，杜绝隐患。组织驻村干部、村干部对辖区内山塘、河流、积水坑等易溺水区域进行排查，做到不留死角，对发现存在的隐患，立即进行整改。在山塘水库、河段等水域醒目位置设立警示牌1800余块。加强重点时段、重点水域、重点人群的巡查力度。
