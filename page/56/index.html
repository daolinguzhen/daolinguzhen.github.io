道林镇“五抓”做好70周年大庆特护期间安全维稳工作

一是抓方案制定。成立以党委书记为组长的特护期信访维稳工作领导小组，详细制定《道林镇70周年大庆特护期信访维稳工作方案》，确保任务到人、责任到人。

一份与众不同的市长热线交办单

发表于 2019-09-02 分类于 乡镇动态 本文字数： 806 阅读时长 ≈ 1 分钟

8月26日，道林镇收到了一份“12345市长热线”问题交办单，与往常的交办单不同的是，这次既不是投诉举报，也不是问题求助，而是点名道姓感谢道林社区驻村第一书记李国。
    这件事还得从李国的一次走访说起。今年32岁的蒋罗是道林社区的建档立卡贫困户，因为患有二级智力残疾，不但没有劳动能力，还时刻需要有人在身旁监护照顾。据周边邻居反映，只要家里没人时，蒋罗就会拿砖头去砸邻居们的门，所以一直以来蒋罗都是与父母共同居住在一起，由母亲在家照看。
    这个月初，李国在去蒋罗家的一次例行入户走访中得知，蒋罗的父亲蒋序贵尽管60来岁了，可是为了家庭生计，时不时在外打些零工挣钱。可是随着年纪增大，蒋罗母亲身体状况大不如以前，照顾蒋罗有些力不从心，于是他父亲只能辞工回家帮忙照看蒋罗，家里就没了经济来源。
    了解到这一情况后，李国一直寻思着如何想个两全的办法，既能让蒋罗的父亲安心外出打工，又能让蒋罗得到更好的照顾。经过多方打听，李国得知在大成桥镇有一家阳光残疾人托管中心，全托的费用也不贵。经过上门咨询和实地考察后，李国将这个地方推荐给了蒋罗父母。在征得其父母同意后，8月23日，李国便带蒋罗前往宁乡市民之家办好有关资料和申请手续。8月26日，李国又驱车将蒋罗送至残疾人托管中心正式入住。
    看到儿子有这么专业的地方进行照顾，蒋序贵可以放心外出继续打工了，为了表达自己对李国书记的感谢之情，于是拨打了12345市长热线表扬李国。
    据了解，李国自2018年3月被派驻至道林社区任驻村第一书记以来，坚持走访常态化，对该社区86户贫困户的家庭情况了如指掌，在贫困户遇到困难时，他总是尽自己最大的能力帮忙化解难题。

千企帮千户，宁乡花明粮油精准扶贫暖人心

发表于 2019-08-31 分类于 乡镇动态 本文字数： 597 阅读时长 ≈ 1 分钟

8月30日，宁乡市花明粮油食品有限公司在道林镇靳水村开展爱心助学活动。
当天，该公司的工作人员与市农业农村局、发改局、道林镇相关负责人一道共同为靳水村的47名贫困学子发放爱心助学款合计4万元，勉励受助学生振奋精神，坚定信念，好好学习，以优异的成绩和良好的品德回报社会。

“谢谢花明粮油对我的无私帮助，我一定把这份爱心化作自己成长的动力，更加努力学习！”来自靳水村的曾钰莹以优异的成绩考入了长沙理工大学，其父亲不幸患有尿毒症，家庭经济捉襟见肘。此次获得花明粮油2000元助学金，有效缓解了她开学学费的燃眉之急。受助学生纷纷表示，将会铭记捐助者的善举，把社会的关爱化为学习的动力，立志成才，把自己锻造成为有爱心、有责任心、有感恩之心的社会合格公民。

“靳水村作为我们‘千企帮千户’的联点村，我们致力于通过产业扶贫、就业帮扶、教育扶贫等一系列措施，助力该村贫困群众脱贫致富。”宁乡市花明粮油食品有限公司总经理李万兵介绍，公司通过发展靳水村的水稻产业，鼓励当地群众大规模种植水稻，同时对农户提供农资供应和技术指导，待水稻成熟后再以高于市场价进行收购。初步预计，今年公司将为该村增收50万元以上。

近年来，宁乡市花明粮油食品有限公司通过诚信经营，规模不断壮大的同时，也格外重视社会效益。他们一方面加大环保治理和技术创新投入，让宁乡粮油品牌享誉全国；另一方面积极承担社会责任，热心支持和参与社会扶贫及慈善公益事业。

道林镇：多措并举坚决打好非洲猪瘟防控阻击战

发表于 2019-08-27 分类于 乡镇动态 本文字数： 1k 阅读时长 ≈ 1 分钟

目前，非洲猪瘟疫情已持续数月，防控形势十分严峻，道林镇党委政府高度重视，保持防控高压态势，多措并举坚决打好非洲猪瘟防控阻击战。
    一是强化组织领导，层层落实责任。制定非洲猪瘟疫情应急预案，成立了以党委书记李纯为政委，镇长姜晓为指挥长的应急指挥部，各村（社区）均成立非洲猪瘟防控领导小组，多次召开会议研究部署非洲猪瘟防控工作。同时，健全完善镇党政领导干部包村、驻村干部包组、村干部和村防疫员包户的网格化责任体系，层层压实防控责任。
    二是加强宣传培训，提高防控能力。通过“村村响”、宣传车、QQ、微信、短信、电话、发放宣传资料、入户走访等形式，进行非洲猪瘟防控知识的正面宣传，避免谣言惑众，积极营造群防群治的良好局面。同时，组织养殖户、贩运户、屠宰户、冻库经营户、村防疫员等涉猪产业链条式人员培训10余次，宣讲动物疫病防控知识和防治措施，提高群众防范意识和能力，做到科学防范，规范调运。
    三是实施严堵严查，加强疫情外堵。设立道林高速、桐子坪、河东新村、龙泉湖村等4个非洲猪瘟防控哨卡，安排专人24小时值班值守，对过往车辆、人员进行严格登记、检查和消毒，对运输无检疫证明的生猪及其产品或病死猪体、不合格猪肉产品进行强制无害化处理，阻截外来运输生猪及其产品入镇。截至目前，共检查可疑车辆400余车次。
    四是严格排查消毒，加强疫情内控。加大疫情排查监管力度，镇村包片干部每日对生猪养猪场（户）开展地毯式排查，建立台账，确保不漏一场一户，重点检查养殖户是否饲喂泔水、是否按时消毒、养殖猪是否有异常，并要求严格执行“日巡查”、“日报告”制度。同时，加强镇域内养殖大户和无害化处理消毒，分发各类消毒药6.1吨、生石灰176吨、消毒麻袋1200个、消毒地毯220平方米，集镇及养殖集中的村主干道消毒100余次，严格疫情内控。
    五是严抓督促检查，严肃工作纪律。成立督查小组，开展对检查点值守人员、各村、相关部门巡查情况进行不定期督查，全方位督查督办，抽查养殖农户家中防控情况，并指导村级做好疫情监测工作，对发现问题的地方做到立行立改。严格落实24小时疫情值班、疫情报告和举报核查制度，对失职失责行为进行严肃问责。

道林镇一个月拆除“空心房”123处

发表于 2019-08-19 分类于 乡镇动态 本文字数： 809 阅读时长 ≈ 1 分钟

7月26日，拆除金华村“空心房”10处905平方米；8月1日，拆除河东新村“空心房”9处116平方米……在道林镇工作微信群里，动态不断更新、点赞加油鼓励不断，“空心房”整治工作在“指尖”不断传递着最新的讯息。7月中旬以来，该镇全面开展农村“空心房”综合整治，用农村“空心房”整治这“一根针”，穿起了“三农”工作“多条线”，在创新体制机制、破解发展难题、推动乡村振兴上探索了一条新路子。
    高位规划，设定“时间表”。将“空心房”整治与美丽乡村建设、全域旅游发展相结合，紧盯7月到9月拆除“空心房”200处的目标，优化整治路线图及时间表，逐村制定了“空心房”拆除工作方案，并积极发动群众自拆，秉持示范带动、先易后难的原则稳步推进“空心房”拆除。
    详细摸底，打好“基础牌”。联合国土、规划、城管中队组建“空心房”整治工作小组，按照“全面、细致、准确”的原则，详细摸清“空心房”的栋数、面积、权属、界址以及户主信息，逐户制定并落实整治措施。
    深入宣传，树立“风向标”。通过“村村响”宣传、悬挂横幅标语、发放宣传单、深入村组与群众面对面宣传“空心房”整治的意义等方式，抓住务工人员中元节返乡的契机，做好全方位、立体式动员宣传工作。
    重拳整治，彰显“执行力”。走进石金村，一股陈年土砖夹杂焦炭的气味便随风飘来，不远处，被拆下的横梁、土砖堆成一大堆。正值酷暑，在道林镇各个村“空心房”拆除现场，挖掘机、工程车来回穿梭，一片忙碌的景象。“空心房”拆除工作服务队对需要拆除的“空心房”及时调配挖掘机等机械设备进行拆除，拆除后进行土地平整，让农户可以充分利用土地，保障农户的土地使用权利。在一个月的时间里，该镇拆除“空心房”123处，面积达1.1万平方米。

道林镇：开好村组户主会，架起干群连心桥

发表于 2019-08-09 分类于 乡镇动态 本文字数： 560 阅读时长 ≈ 1 分钟

“各位乡亲，今天再次组织大家召开户主会，就是想再着重讲解一下涉黑涉恶线索举报的11类重点和举报电话，你们如果有相关的线索，请积极举报……”连日来，道林镇全体机关干部职工积极发扬“五加二”“白加黑”精神，纷纷走村入户，到所联系的村组，结合“平安创建”、扫黑除恶、垃圾分类、改厕、空心房整治等多项工作召开户主会，掀起了户主会的新一轮热潮。

据悉，道林镇的户主会从今年4月开始全面推行，即镇村干部不定期组织各村民小组召开户主会，宣讲党的政策和有关精神，传达部署乡镇中心工作。截至目前，全镇499个村民小组已召开了3轮户主会，户主会的召开既满足了群众知晓政策的需求，又调动了群众参与中心工作的积极性，还密切了党群干群关系，拉近了与群众之间的距离。
    “以前乡镇部署工作，只在镇村两级开会，政策精神很难及时传达到各村民小组，很多工作得不到群众的有效支持。现在开户主会，不仅人齐了，心也齐了。”道林社区党总支书记杨红军说。
    “开好户主会，便能架起干群连心桥。所有镇村干部都要下到村民小组召开户主会，原原本本讲政策，真心实意听民意，求真务实办实事。一定要让群众与党委、政府齐心，形成“振兴千年古镇 共建大美道林”的发展合力。”道林镇党委书记李纯说。
