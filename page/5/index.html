启程时光行，探秘农耕记——参观宁乡市道林农耕文化博物馆

发表于 2024-09-13 分类于 乡镇动态 本文字数： 898 阅读时长 ≈ 1 分钟

2024年7月21日，湘潭大学计算机学院非遗创新探索一号实践调研团继续在宁乡市道林镇开展暑期三下乡社会实践活动，参观了道林农耕文化博物馆。此次活动旨在通过实地调研，增进实践团成员对道林镇农业历史和农耕文化的了解，加深其对道林镇经济文化基础的认识，为保护中国传统文化做出贡献。
    道林农耕文化博物馆作为一处集教育、研究与展示于一体的文化场所，呈现了道林镇传统农业的发展脉络。博物馆内珍藏有传统农业时期道林人民的住宅、农具、生活用品以及还原道林人民真实劳动情景的铜塑，每一件展品都承载着厚重的历史与文化。

在调研实践过程中，实践团成员们逐一参观展馆。展馆通过生动的场景重现和详实的文字解说，让实践团成员们穿越到那个时代，让他们亲身感受道林人们耕耘的辛勤和智慧。
    “棉纺机可以织布，筛子可以筛五谷，风车可以车稻谷，石磨可以磨豆浆”“没有牛的时候，可以两个人一起耕地，一根扁担一根绳，两人挑扁担，后面的耕犁，前面的牵绳”实践团成员在认识农业生产工具，解说农业劳动过程中，仿佛看到了辛勤织布的主妇，热气腾腾的豆浆；仿佛听到了车稻谷时咚咚歌唱的风车，筛稻谷时沙沙欢呼的筛子，这使得他们对于道林镇传统农业有了更加深刻的认识。

此次参观学习是一次宝贵的教育经历。它不仅让大家对道林镇传统的农耕文化有了更加全面和深入的了解，也激发了青年学子对传统文化的兴趣和传承责任感。实践团成员纷纷表示，“我们将把此次实践的体验融入到我未来的中国文化传承研究和生活实践中，来进一步传承和弘扬中华优秀的文化遗产。”
    博物馆的负责人表示，“道林农耕文化博物馆始终致力于农业文化的传播和教育工作，希望能通过不断的努力，让更多的人了解并重视这片土地上的农耕文化，进而为保护和发扬中国农业遗产作出贡献。

湘潭大学计算机学院非遗创新探索一号实践调研团此次的实践活动，无疑促进了实践团成员对道林镇农耕文化及其经济文化基础的了解，加深了实践团成员对中国传统文化的认识，同时也为农耕文化的传播与推广做出了积极的贡献。

道林镇卫生院：开好“清廉处方”，建设“清廉医院”

发表于 2024-09-13 分类于 乡镇动态 本文字数： 2.1k 阅读时长 ≈ 2 分钟

道林镇卫生院坐落于美丽的靳江河畔，成立于1956年，历经近70年的建设和发展，现已成为我市乡镇卫生院中基础设施完善、科室设置合理、人员配备齐全、技术力量雄厚、综合实力较强的标准化基层卫生院，是长沙市乡镇卫生院建设的样板工程。
    近年来，道林镇卫生院全面落实省、市及宁乡市委关于清廉建设的部署要求，以加强医德医风建设为抓手，以群众看病就医遇到的“急难愁盼”问题为导向，紧紧围绕“党风清正、院风清朗、医风清新、作风清明”，打造“医疗服务优质、医务人员廉洁行医”的清廉医院，推进全面从严治党向纵深发展。

一、开好党建引廉“固本方”，构筑清廉防线

一是发挥党建领航定向，把稳方向之舵。坚决贯彻习近平新时代中国特色社会主义思想，深学笃行党的二十大精神，严格落实“一岗双责”和“谁主管、谁负责” 的要求，形成党支委统一领导、党政齐抓共管、纪检监督、科室配合、党员干部群众合力共建的工作格局。制定印发《清廉医院建设实施方案》，成立领导小组，召开专题会议；领导班子成员签订《“一岗双责”责任书》、全院干职工签署《医疗机构工作人员廉洁从业九项准则》承诺书、《党风廉政建设责任书》；充分运用“三会一课”、党员“三亮三比”、党支部书记讲“廉洁行医”党课，实现党建工作与清廉医院建设互融互通、互促互进，党风廉政建设与业务发展同部署、同检查、同落实、同促进。二是压实责任链条，实行层级负责制。坚持高位推动，把打造“清廉医院”品牌建设作为“一把手”工程来规划落实，巩固深化“四责协同”机制，划好“责任田”，建立“院长+班子领导+科室主任+普通职工”四位一体的责任落实链条，切实将责任压实到“神经末梢”，做到“清廉医院”建设事事有人管，件件有人抓。定期召开党支部会议、院长办公会研究党风廉政建设和反腐工作中遇到的各项问题，切实发挥党的引领和带动作用。建立健全民主决策、绩效考核、财务管理、物资和工程建设项目管理四项内控制度，制定《医德医风考评制度和奖惩办法》，落实党务院务公开，让医护人员有章可循、有据可依，有效杜绝不规范行为的发生。

二、开好文化育廉“益气方”，弘扬清风正气

一是全面覆盖筑牢基础。组织全院干职工观看反腐警示教育片，召开“清廉医院建设暨行业作风建设动员大会”。大力开展社会公德、职业道德教育，深入展开全院性的医务人员行为规范教育和讨论，为清廉医院建设营造良好的文化氛围。二是建立廉洁文化阵地。在住院大厅、医生办公室等公共区域张贴廉洁心语、廉政准则，摆放清廉宣传栏，制作主题文化墙，门诊电子LED显示屏滚动播放清廉文化标语，统一以清廉格言为主题，以莲花等廉洁象征物为点缀的电脑清廉屏保及鼠标垫，努力营造抬头可见、驻足可观，崇廉尚廉守廉的浓厚氛围。三是丰富廉洁文化形式。利用5.12护士节、8.19医师节开展医护人员清廉宣誓、清廉模范评选活动，树立清廉榜样，在医院干职工中推进清廉教育，打造了清廉医生、清廉护士、清廉医技和清廉行政人员四支清廉团队，在全院营造了崇廉、尚廉、守廉的良好氛围，让干职工在潜移默化中受到启发，在耳濡目染中接受廉洁守纪教育，使廉洁文化在心中构筑一道牢不可破的“廉政防火墙”。

三、开好监督促廉“驱邪方”，提高肌体免疫

一是开展民主管理，强化重大决策监督。召开职工代表大会，干职工代表表决通过《宁乡市道林镇卫生院2023年度绩效考核方案》等，对重大决策、重要项目等“三重一大”事项都经院委会、职代会集体讨论决定。二是实施效能监督，强化行业作风建设。制定《关于医德医风问题和医药领域腐败问题集中整治工作实施方案》，召开院委会、中层骨干会和职工大会研究部署有关工作，深入学习领会省、市医德医风问题和医药领域腐败问题集中整治工作的相关文件及会议精神，将廉政账户告知所有干职工，聚焦“关键少数”和关键岗位，对照八类四十三项整治重点，印发《落实廉洁从业九项准则学习自查表》和《个人自查自纠表》，逐项逐条自查自纠。设立举报箱、举报电话、举报邮箱，主动接受社会监督，全面促进院风清朗。三是强化重点监督，筑牢廉政风险防控。对药品、器械、检验试剂、耗材供应供货商代表进行集体廉政谈话，并签署《清廉医院共建协议及供货商购销合同》；医院班子成员组织对科室负责人每季度集体谈话、新任中层骨干任前谈话，在端午节、中秋节等节假日、重要节点进行全院干职工廉洁谈话，释放警示信号，切实筑牢廉政风险防线。四是加强日常监督，提高全员廉洁意识。总值班人员每日不定时到各科室巡查，及时发现问题解决问题；纪检科、医务科、医保科等多部门联合开展清廉医保自查自纠专项整治活动，严格规定禁止收受“红包礼金”，坚决纠正不正之风，传递廉洁风尚，树牢思想自觉和行动自觉。
    “清廉医院 ”建设唯有持之以恒，久久为功，才能善作善成。医院将持续深入开展“清廉医院 ”建设，把清廉思想、清廉制度、清廉规则、清廉纪律、清廉文化融入医院建设发展的各方面全过程，让清朗院风吹拂医院每一个角落，让清新医风深入每一位医务工作者内心，向党风清正、院风清朗、医风清。

宁乡：开出“廉处方”，建设“清廉医院”

发表于 2024-09-13 分类于 乡镇动态 本文字数： 1.2k 阅读时长 ≈ 1 分钟

“清廉理政、廉洁行医，是我们医务工作者最基本又最根本的要求。坚决不进有回扣的药品，不开不必用的药品，保证药品质量，不增加病人药品费用，保证病人治疗效果。”对于一家公立医院而言，加强党风廉政建设是促进发展、稳定的重要保证，这不仅事关医院的发展前景，同时也关系着每一位患者的切身利益，道林镇卫生院院长张宁湘一字一句掷地有声。
    近年来，为加强党风廉政建设，强化制度的刚性约束，道林镇卫生院始终以党建为引领，以抓廉洁从业和行业作风建设为基础，建立‘微腐败’问题线索，台账实行销号管理，并于今年3月份出台《道林镇卫生院关于推进清廉医院建设实施方案》，全院85名干职工均签订了廉洁承诺书。
    “这家医院卫生搞得好，服务态度也非常好，尤其是这个清单设计得好，每天用了什么药打了什么针都清清楚楚。”68岁的黄大爷今年已是第四次住进卫生院了，身患多种慢性疾病的他从七年前开始就是卫生院的“常客”，他表示，“这里的医生熟悉我的病情，不会多开药，也不会开贵的药，我信得过。”
    黄大爷口中的清单，是道林镇卫生院自2018年12月搬入新址后就开始采用的，“来我们这里住院的主要是附近村民，尤其是老人居多，这个清单每天都打出来给到他们手上，让他们安安心心治疗，不怕花了冤枉钱。”
    让患者花明白钱，治安心病，道林镇卫生院在这一块下了很多功夫。每一个护士站，都可以看到一间光线通透、恒温恒湿的房间，里面整整齐齐摆放着许多输液盒。这个输液盒对应到每一个床位，更特别的是它有多个分格，便于分类摆放不同药物，而且每一个盒子都佩戴着一张粉色的药物清单。“用了这个盒子，杜绝了药物混淆，有效提高了我们护士的工作效率，也便于患者对我们工作的监督。”
    据悉，为了更好地的储存暂置护士站的药物，医院特意从有限的资金中拨出款项，为这个房间增加了独立的空调和电源，以防断电造成药物因保存不当而失效。
    细节打动人心，廉洁赢得民心。作为一家仅有85名干职工、172张床位的乡镇卫生院，道林镇卫生院用优质的服务、良好的医风医德以及清廉行医赢得了患者的信任，去年一年住院患者达5300人，满床率常年维持在80%左右。
    2022年1月起，宁乡已在全市范围内所有医院、卫生院开展清廉医院建设工作。下一步，宁乡将继续按照“清廉医院”建设“四个四”标准，从健全四项内控制度、搭建四个工作平台、打造四支清廉团队、搞好四项专项整治几个方面入手，以点带面，持续推动清廉制度、清廉文化、清廉服务融入医院发展的全过程，解决老百姓就医的痛点、难点，切实提升市民群众的获得感、幸福感。

长沙市生态环境局关于《宁乡市道林镇善山岭村消纳场项目环境影响报告表》审批前的公示

发表于 2024-09-13 分类于 乡镇动态 本文字数： 1.7k 阅读时长 ≈ 2 分钟

项目名称：宁乡市道林镇善山岭村消纳场项目
    建设地点：湖南省宁乡市道林镇善山岭村
    建设单位：湖南省宁乡市道林镇善山岭村村民委员会
    环境影响评价机构：湖南和昱环保科技有限公司
    公示日期：2024.2.2-2024.2.8
    项目概况：湖南省宁乡市道林镇善山岭村村民委员会拟投资2000万元在善山岭村建设宁乡市道林镇善山岭村消纳场项目。消纳场项目总占地146955m2，总消纳量约221.21万m3，服务年限为4年。消纳场占地为林地，为本项目建设单位村集体所有，所占林地不属于天然林；不占用基本农田、不占用耕地。本项目不涉及房屋、电力通讯电杆等拆迁。
    主要环境影响及预防或者减轻不良环境影响的对策和措施：
    （1）施工期环境影响措施
    本项目施工期将产生噪声、扬尘、建筑垃圾和生活垃圾。由于施工期时间有限，影响范围以局部污染为主，因此施工期重点是加强管理，只要精心安排，施工进度严格管理，对扬尘、噪声采取有效措施进行控制、治理，建筑和生活垃圾按规定处理，采取一定的措施防治水土流失，这样可将污染减少到较低程度。
    （2）营运期环境影响措施
    1、水环境影响措施
项目生活污水经化粪池处理后用于场地周边林地施肥，不外排；洗车废水经隔油沉淀池处理后回用于洗车，不外排。采取上述措施后，项目营运期废水对周边地表水环境影响较小。
    2、大气环境影响措施
    项目汽车、设备尾气产生量较小，经自然扩散后排放消纳场区；卸料、堆填作业、运输路线扬尘通过出入车辆清洗、洒水抑尘处理后排放，运输车辆在出场前进行清洗，避免车轮带土上路，减速慢行，运输渣土车辆采用篷布进行覆盖，对进场道路定期进行清扫，保持路面清洁，通过采取上述措施，废气满足《大气污染物综合排放标准》（GB16297-1996）表2无组织监控浓度限值。
    3、声环境影响措施
    本项目营运期噪声主要来自在渣土运输、堆存作业等过程中使用机械设备产生的噪声，本项目运营后采取选用低噪声设备、设备底部隔声减震、控制车辆运输时间、限速、禁止鸣笛等措施、距离衰减后，噪声能满足《工业企业厂界环境噪声排放标准》（GB12348-2008）中2类标准。
    4、固体废物影响措施
    本项目营运期固体废物主要为生活垃圾和沉淀池污泥、油污。生活垃圾由环卫部门统一清运；沉淀池污泥定期清理至消纳场内；洗车废水沉淀池会收集到少量油泥及油污，收集后于危废间暂存，定期交危废资质单位处置。采取上述措施后，本项目固体废物可得到妥善地处理，对周围环境造成的影响很小。
