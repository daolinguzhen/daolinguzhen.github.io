发表于 2020-07-23 分类于 乡镇动态 本文字数： 411 阅读时长 ≈ 1 分钟

近日，道林镇龙泉湖村为进一步推进交通问题顽瘴痼疾整治，组织开展顽瘴痼疾安全宣讲活动。
    “骑电动车也要戴头盔才能上路，安全第一，不要觉得麻烦，关键时刻它能救命。”“您好！这是交通问题顽瘴痼疾集中整治宣传资料，空闲时可以多看看。”当天，龙泉湖村党员干部深入群众家发放宣传单，叮嘱村民戴“盔”上路，注意交通安全，将整治“触角”延伸至农村道路“末梢”。另外，通过悬挂横幅展板、发放宣传单、定点播放音频、微信群转发等方式，宣传安全出行知识。50多名党员志愿者、村民还观看了安全教育警示片，并为自己的摩托车登记上牌。

道林镇三举措切实提升政务服务水平

发表于 2020-07-16 分类于 乡镇动态 本文字数： 372 阅读时长 ≈ 1 分钟

一是统筹服务力量。全镇9个村（社区）划分为59个三级网格片，共开展“屋场夜会”105场次，为群众解决实际问题176个，其中发动社会力量帮助解决40户贫困户饮水问题。
    二是优化服务流程。推行“服务前台+功能后台”运行模式，党群服务中心作为“服务前台”，直接面对公民和法人，承担行政执法和行政审批、公共服务职责；党政内设机构、驻镇职能部门和村（社区）作为“功能后台”，为“服务前台”工作提供支持和保障。2020年上半年共收到民声受理工单154件，已全部按时办结，工单满意度达100%。
    三是跟进服务评价。紧盯群众不满意评价，坚持问题导向进行分析和进一步处理。对经办干部为民服务态度和业务办理能力等存在问题的，及时指出和纠正，不断提高干部服务群众的能力水平。

道林镇开展道路安全联合执法行动

为贯彻落实市整治办下达的关于交通问题顽瘴痼疾整治工作部署会议精神，有效防范道路安全交通事故的发生，2020年7月14日，道林镇整治办联合交警二中队、综合执法大队，在道林镇镇域范围内多个车流量大的路口分批次进行了道路交通联合执法，此次行动共暂扣7辆未戴头盔、安装遮阳伞且无牌无证的摩托车，1辆无牌无证四轮电动车，1辆无证改型面包车，查处违法行为共计32起，发放“一盔一带”宣传单100余份，对部分驾驶人进行了交通安全宣传劝导，同时对过往客车、私家车、农用车进行了安全带佩戴和超载限载等方面的安全检查。

一封来自道林镇河东新村贫困户的感谢信

发表于 2020-07-08 分类于 乡镇动态 本文字数： 2.3k 阅读时长 ≈ 2 分钟

尊敬的河东新村扶贫干部们：
你们好！
    首先，让我代表我们全家向你们表示最衷心的感谢！是你们的帮助让我家的经济压力得到了缓解，是你们的关心让我们体会到了人间的温暖，是你们的鼓励让我们不忘初心，砥砺前行。
我家是河东新村建档立卡户的一份子，我是“建档立卡户”的受益者。父亲在我很小的时候就因病去世，家中欠下巨款、一贫如洗，只留下母亲照顾我和弟弟。母亲很坚强，勇敢地担任起了家庭重任，还掉欠下的债务，并抚养我和弟弟读书。但随着日子的迁移，读书的费用越来越多，母亲的身体也每况愈下，虽然我和弟弟能靠拿奖学金和做兼职赚点生活费，但经济压力显然还是家里的最大困难。
    直到国家精准扶贫政策的到来、镇政府和村里扶贫干部组的到来，让一切都在慢慢地好起来！
    情感关怀暖民心
    夏书记平近易人，为了群众，跑前跑后，每次来到我家都会亲切地询问我家里的相关事情，每次都会问除了已享受的政策，还有什么其他的困难没有。他担心我和弟弟读书费用是否已有，他会帮我家一起销谷酒，他会关心我妈妈的身体是否健康，他会顾及我家大大小小的事情。就比如这次的展销会，是他积极向政府推荐我家的谷酒，并在展销会的那天，一大早从宁乡赶过来，和卢主任一起帮我们运酒去宁乡方特集市展上卖酒。这天烈日炎炎，人群熙熙攘攘，他们如火如荼地帮我们卖酒。他们的热情灼烧到我们的心里去了，那份感动我实在难以用词汇来形容，唯有一句：谢谢！
    卢主任细致入微，对待工作一丝不苟，卢主任每次上门工作，极其认真，不管她有多忙，她都会耐心地记录和解决每一个问题，她还会像朋友一样经常跟我聊天，告诉我珍惜现在，珍惜亲人，她曾以自己的经历告诉我子欲养而亲不待的无奈。我很喜欢跟她聊天，之前学习任务没那么繁忙的时候，每次放假回家都会去找她聊聊天。印象中，夏书记和卢主任每个月都会来我家几趟，不是工作需求就是嘘寒问暖。最记忆犹新的是，我在学校读书的时候，每次卢主任去了我家都会拍一张照片给我，那就是我家的狗，小布。每次看到手机里的照片，我总是会高兴的笑出来，是的，这张照片或许微不足道，但又是因为这张照片让我知道家里一切都安好，知道卢主任他们又去看望我母亲了，让远在学校的我和弟弟放心安心地读书。
    姚叔叔，之前于我而言是一个活着“云”端的帮助人。过去的一年里，我在学校总是接到母亲的电话，姚总今天又过来了，给我们家带了几瓶油、姚总今天又来了，帮我家销了几十斤谷酒、姚总今天又来了，马上要过中秋，给我们家送了月饼……因为各种不凑巧，我一直没有见到这个默默关心我家的好人，所有的一切都是从母亲的电话中得知。是的，母亲会把家中大大小小的事情都跟我说，她希望以后的我不仅仅能长大撑起这个家，更是让我记住这一切，并希望以后能够像他们一样去汇报社会、感恩社会。直到因为这次疫情没有开学长时间在家才见上姚叔叔，姚叔叔很亲切，会询问我和弟弟在学业上是否有困难，会尽心尽力的帮我家销酒，鼓励我们，支持我们！
    精准扶贫促发展
    为了贯彻落实中央、省、市、县扶贫工作系列会议精神，实施精准扶贫，加快贫困群众脱贫致富步伐，河东新村的扶贫干部们高度重视，没有丝毫懈怠。为了寻找河东新村贫困户发展良策，夏书记和卢主任带着扶贫干部多次来到贫困户家里，为每户人家因地制宜地制定系列经济发展方案。
    据我所知，每个建档立卡户都有一套适合自己家的发展经济方案。有的人家发展种植业，有的人家发展养殖业，有的人家多种方式共同发展。如养殖业有养鸡、养蜜蜂等。当时政策出来的时候，夏书记他们会下乡到每个贫困户家，咨询相关事宜并询问他们想法，给他们推荐经济发展方式，得到大家一致的同意后，统一由他们发放生产苗子到每个贫困户家，并在成熟后除了自己出售后所留下的一并进行回购。这个经济发展方案，对于贫困户来说着实是一个福音，让贫困户不在为了经济发展而担心成本问题和回收问题，同时让每一个贫困户充满信心和勇气去干活。
    春暖花开，爱洒人间
    有一种温暖，叫精准扶贫；有一种责任，叫大家行动。这几年有各个扶贫干部不遗余力地为河东新村脱贫工作努力奔走、倾心付出，我们会越来越好，河东新村会越来越好。你们村干部在实地中摸准问题，在与民同甘苦中建立感情，在贴心交流中寻求脱贫良策，你们实事求是、任劳任怨的形象和风貌会永远留在我们的心中。
    滴水之恩当涌泉相报。此时此刻，我唯一能说的是感谢。面对你们帮助，我唯有铭记父母的嘱托和你们的期望，抛弃后顾之忧，刻苦学习，努力拼搏，勤俭节约，珍惜现在，绝不辜负党和国家及老师的期望。我会树立远大的理想，明确学习目标，以自信的态度，执着的信念，刻苦学习，以德博学，完成学业，把自己磨炼成品学兼优的合格有用之才，为将来更好的回报国家，回报社会打下坚实的基础。
    道林镇河东新村肖叶（现就读于湖南城市学院）
    2020年7月6日

第一书记给贫困户义务打零工

发表于 2020-07-01 分类于 乡镇动态 本文字数： 514 阅读时长 ≈ 1 分钟

6月29日早上六点多，道林镇河东新村第一书记夏强准时来到贫困户杨国清家菜园报到，同其夫妇二人一起摘秋葵。七点半，摘了满满两大桶秋葵后，夏强打开自己的朋友圈开始吆喝叫卖“贫困户家大量出售秋葵6元一斤，土鸡26元一斤……”
    半个月前，贫困户杨国清跟夏强诉苦，自己种的2.7亩秋葵正值收获季节，按照市场价一天能卖几百块，自己还有其他养殖种植业要花时间打理，没那么多精力采摘，可是请人工来采摘一天的工价就得花掉一百每人，除去成本，这样算下来根本赚不到什么钱。夏强听了后当机立断表态，只要不下雨，自己每天早上6点到8点和他们夫妇俩一起摘秋葵，省下这笔人工费。随后，夏强还联系了一家秋葵基地每天到杨国清家收购一批，剩余的少部分，自己通过朋友圈为其销售。半个月来，杨国清仅人工采摘劳务费就省下了一千多块，仅这片秋葵地预计年产收入可达1.5万元。
    去年七月，夏强到该村接任第一书记以来，为杨国清家销售鸡、鸭、鹅共计1.2万余元。今年，在夏强的指导下，杨国清家养殖了一百多只鸡、鸭、鹅，种植了西瓜、辣椒、秋葵，还租了5亩地种植水稻，预计年收入可达四万元。

长沙市生态环境局关于《宁乡市公路养护中心G354宁乡段（宗师庙—道林—联湖塘）公路工程建设项目环境影响报告表》拟审批意见公示

发表于 2020-06-19 分类于 年度报告 本文字数： 3.7k 阅读时长 ≈ 3 分钟

项目名称：G354宁乡段（宗师庙—道林—联湖塘）公路工程
    建设地点：湖南省宁乡市道林镇和大屯营镇
    建设单位：宁乡市公路养护中心
    公示日期：2020年6月19日
    项目概况：宁乡市公路养护中心拟投资23023.5万元，在湖南省宁乡市道林镇和大屯营镇境内建设G354宁乡段（宗师庙—道林—联湖塘）公路工程项目。本道路工程建设内容包括道路工程、桥涵工程、交叉工程、排水及防护工程、绿化工程、交通工程及其他设施、临时工程、环保工程等。本工程全长22.203km，采用双向两车道的二级公路设计标准，设计车速60km/h（部分路段40km/h），起于宁乡市道林镇宗师庙X087与X081交叉口处，路线走向为由东北向西南，全线基本沿原X087、X089、X090、X057旧路布线，在道林镇沿东侧新开辟路线绕避道林集镇，然后经大屯营镇集镇、大屯营镇三仙坳，终点位于大屯营镇联湖塘与韶山地区交界处。
    主要环境影响及预防或者减轻不良环境影响的对策和措施：
    （一）施工期环境影响措施
    （1）废水
    本项目施工期废水主要为施工人员生活污水和施工生产废水。施工期生活污水主要来源于营地施工人员就餐和洗涤产生的生活污水及粪便污水，其主要污染因子有SS、COD、BOD5、NH3-N等，施工人员生活污水经隔油沉淀池处理后回用公路沿线农田灌溉，不外排；施工废水主要为混凝土拌合站机械设备和车辆冲洗废水、路基施工时产生的泥浆水、机械设备运行的冷却水和清洗水、施工场地的冲洗水等，主要污染物为COD、SS和石油类，施工废水经隔油沉淀池处理后回用于施工场地洒水抑尘，不外排；项目桥梁4座，其中拆除重建2座，完全利用1座，加宽改造1座，无涉水桥墩施工，通过优化涉水桥梁的施工工艺，减少涉水施工量，禁止废水进入水体，加强施工期水环境风险防范措施，防治机械设备和车辆漏油进入工程区域周边水体，可以有效降低桥梁施工对饮用水水源保护区、周边水体水质的影响。
    （2）噪声
    本项目施工期的噪声主要为施工机械噪声和运输车辆噪声，本项目噪声源强在73—92dB（A）之间。由于噪声具有阶段性、临时性和不固定性，随着工作阶段的不同，其影响也不同，通过采取相应噪声防控措施后，项目施工将不会对周边环境造成明显影响，同时施工结束时，噪声也自行消失，对周围环境影响小。
    （3）废气
    本项目施工期大气污染物主要为施工扬尘，施工机械废气以及沥青烟气。施工扬尘不可避免，施工车辆行驶、施工现场作业等过程均会产生扬尘，露临时堆场也会因风产生扬尘，施工单位可以通过加强施工期管理，采取相应的抑尘措施，使本项目在施工期对周围空气环境的影响降到最低；项目施工机械废气主要为燃油机械运行过程中会产生一氧化碳、二氧化氮、总烃等污染物，施工单位可以通过加强施工机械和车辆的管理，减轻机械尾气对周围空气环境的影响；项目不在施工现场设置沥青混凝土搅拌站，沥青混凝土直接从沥青加工厂拖运至施工场地直接铺设，沥青混凝土的铺设过程中仅产生少量沥青烟，对空气环境的影响较小；食堂油烟经净化设施处理后能达标排放，对周边环境空气及敏感目标的影响较小。
    （4）固废
    本项目施工期固体废弃物主要为拆迁建筑垃圾、土石方弃渣以及施工人员生活垃圾。项目产生的拆迁建筑垃圾先进行可回收资源筛选，对无利用价值的废弃物运至项目弃渣场；项目路基弃方运至弃渣场处理；项目施工人员生活垃圾及含油抹布集中收集后交由环卫部门清运处理。本项目施工过程的固体废物不外排，对环境影响较小。
    （5）生态环境
    本项目施工建设过程中，将会对项目周边的生态环境造成一定的直接与间接损害，可能产生不同程度的水土流失，桥梁施工不当将会对施工河段的水生生物、底栖动物的生存环境产生影响。但项目施工期间对生态环境的破坏可采取一定的措施避免或减轻其影响，且这些影响是短期的，随着施工期结束，项目建设不会对周围生态环境产生明显影响。
    项目施工期采取在湖南宁乡靳江省级湿地公园内原有路线的基础上尽量往湿地公园外拓宽的方式进行改扩建，完全利用湿地公园内中桥，不涉及涉水施工，同时不得在湿地公园内设置取土场、弃土场、施工生产生活区、施工道路，并加强生态环境保护宣传教育及管理的措施后，项目建设对湖南宁乡靳江省级湿地公园的影响较小。
    （二）营运期环境影响措施
    （1）废水
    本项目营运期对附近水体的污染源主要为降雨初期的路面雨水径流。项目区域排水体制为雨污分流排水体制。根据国内的环境影响评价和监测经验，路面初期雨水或桥面径流进入河流后，将在径流落水点附近的局部小范围内造成污染物浓度的瞬时升高，但在向下游流动的过程中随着水体的搅浑将很快在整个断面上混合均匀，对河流的污染较小。
    本项目运营期对饮用水水源保护区的影响主要为发生交通事故时，泄露的危险化学品未及时收集而泄露至地表水体从而影响下游的饮用水水源保护区。项目主要采取提高桥梁防撞等级，并设置警示牌提醒过往司乘人员注意并小心驾驶，同时在桥梁上设置了导流槽，并在桥梁两端设置了事故应急池，建立了事故应急响应机制，采取以上措施后，项目运营期对饮用水水源保护区的影响可降低到最低水平。
    （2）废气
    本项目运营期大气污染物主要为汽车尾气及道路扬尘。汽车尾气中污染物排放量的大小与交通量成比例增加，与车辆的类型、汽车运行的状况以及当地的气象条件有关，本项目通过加强道路两侧绿化工作，净化空气，提高空气质量，减轻汽车尾气对周边环境空气影响；道路扬尘的产生量与交通道路洁净度及行车速度有关，道路运营期通过洒水、路面清扫，保持路面干净整洁，减少扬尘产生，同时道路两侧设有绿化带，对道路扬尘能起到吸附降尘的作用，本道路运营期扬尘对周边居民影响不大。
    （3）噪声
    本工程运营期间对环境的影响主要是交通噪声的影响，根据声环境专题可知，本项目在改善区域交通条件的同时，对沿线环境敏感点产生交通噪声污染，项目涉及的20个声环境敏感点中，除道林中学、靳兴村村委会、大屯营镇三仙坳村、三仙坳村委会、三仙坳集镇、大屯营镇联湖村等6处敏感点预测期各时段均达标外，其他敏感点不同时段均有不同程度超标。其中2类区夜间最大超标量为0.50dB（A）—2.12dB（A）；4a类区夜间最大超标量为0.22dB（A）—3.22dB（A），本项目首选降噪措施为低噪声路面和绿化降噪，低噪声路面降噪量约4.5dB（A），绿化降噪量约0.5dB（A），同时在大屯营卫生院、同荣小学、天里幼儿园等处设置减速带并减速行驶，在采取以上措施后可降噪5dB（A）以上，能够使各敏感点噪声满足声环境质量相应标准要求。
    （4）固废
    营运期固体废物主要为沿线车辆散落的物品、乘客和沿线居民丢弃的生活垃圾、绿化树木的落叶等。营运期固体废物由环卫部门统一收集转运至城市垃圾处理场处置。
（5）生态环境
    本项目对区域植被的损失占总量的比重很小，区域植被覆盖率不会因道路的建设而有明显变化，道路建设配以适当的绿化工程，可以减轻其影响；在公路边缘采用加密绿化带，防止灯光和噪声对动物的不利影响，在桥下进行植被的自然景观的恢复，有利于动物适应新的生境。同时对绿化植被的保护和管理，通过绿化，一方面减少水土流失量，另一方面保持本地植被生物量。
    项目运营期采取加强湖南宁乡靳江省级湿地公园内绿化工程和防护工程建设、提高桥梁防撞等级、加强危险化学品运输管理、设置事故导流槽和应急池等措施的后，可以将因交通事故造成的影响降低到最低程度。
