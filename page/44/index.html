发表于 2021-05-18 分类于 乡镇动态 本文字数： 253 阅读时长 ≈ 1 分钟

5月17日，道林镇食安办、市监所在集市和古镇开展野生蘑菇防控集中宣传活动，通过悬挂横幅、发放资料等方式，向群众宣传野生蘑菇防控和科普知识。

道林镇：答题送头盔、平安伴你行

发表于 2021-05-07 分类于 乡镇动态 本文字数： 484 阅读时长 ≈ 1 分钟

近日，道林镇道安办联合交警一中队、派出所、交管站、城管中队在道林社区开展道路交通安全宣传有奖答题送头盔活动。
    活动开始，道林镇党委委员、副镇长张文宏向群众介绍了“有奖答题送头盔”活动的目的和意义，通过讲述近期发生的交通事故案例，来警示群众应遵守交通规则。同时通过赠送头盔活动，帮助群众培养规范佩戴安全头盔的良好骑行习惯，提高群众交通安全意识和事故防范意识。

在活动现场，群众观看交通安全宣传视频后，参加交通安全知识问答考试，达到80分以上即可免费领取头盔。发放头盔时，由交警和镇政府工作人员一边现场示范如何正确佩戴头盔，一边向到场群众耐心讲解骑行摩托车、电动车的交通安全知识，并向群众倡议发挥示范带头作用，督促提醒身边的亲朋好友养成骑行必戴安全头盔的良好习惯。
    道林镇道安办将继续联合各村（社区）、部门单位，持续开展交通安全知识宣传活动，进一步提高群众的交通安全意识和事故防范意识，引导群众树立安全出行意识，自觉成为文明交通的“践行者”，共同构建安全、文明、和谐的道路交通环境。

道林镇强力推进交通安全集中整治行动

发表于 2021-04-23 分类于 乡镇动态 本文字数： 892 阅读时长 ≈ 1 分钟

我们要从对全镇群众生命财产安全负责的高度，扎实开展道路交通安全集中整治行动，全面排查道路交通安全隐患，为全镇人民群众平安出行营造良好交通环境。”近日，宁乡市道林镇召开道路交通安全整治紧急会议，会上道林镇党委书记彭舜对道路交通安全集中整治行动进行全面安排部署。
    据悉，该镇着重从“五个全面”上下功夫，扎实推动镇域交通顽瘴痼疾整治行动不断向纵深开展。即日起组织镇村干部、派出所民警及志愿者140余人到镇域内29个重点路口进行为期一个月的交通安全宣传及文明劝导。一天时间发放宣传资料1800余份，现场拆除摩托车非法安装的遮阳伞58件，劝导未戴头盔者249人，强制要求69人学习道路交通相关法规。
    该镇全面排查道路安全隐患。组织镇村干部按照“四联五到位”民情大走访工作安排，到所联片区摸排道路交通安全隐患。责任线办将对隐患点科学分类、精准施策，进一步提升道路交通安全系数。
    通过联合交警、交运等部门，针对老年代步车无牌无证驾驶、货车超限超载超速等9类违法行为，开展集中整治行动。4月18日以来，该镇共查处车辆200余台，其中涉及违法违规车辆53台，查扣三轮车4台、摩托车12台、超限超载车辆10台，并对未上牌、未佩戴头盔和两名无驾驶证人员进行治安拘留。该镇将加大路面管控力度，严惩交通违法行为，为辖区群众出行创造良好的交通环境。
    同时，建立监管责任网络，实行联村干部包片包点，执法人员进驻源头企业巡查、监管的责任制度，加强对道路专业运输企业、专业运输户的监督管理工作。道林镇道安委对源头企业每月开展不少于8次的超限超载巡查，结合不定期夜间巡查，从源头上杜绝违法超限超载运输行为。
    该镇绷紧交通安全之弦，严格落实交通安全管理责任，围绕“预防为主、防治结合”的总体方针，完善应急预案、强化应急措施，做到早预警、早预防、早应对，扎实做好道路交通安全防范和应急处置工作。

道林镇召开2021年经济工作大会暨党史学习教育推进大会

发表于 2021-04-14 分类于 乡镇动态 本文字数： 1.3k 阅读时长 ≈ 1 分钟

4月12日，道林镇召开2021年经济工作大会暨党史学习教育推进大会。盘点总结去年工作，表彰先进集体和先进个人，动员和号召全镇上下切实做到学党史、悟思想、办实事、开新局，更好地推进全镇各项工作。宁乡市政协主席刘永红、政协秘书长苏剑波出席会议。
    大会对2020年度目标管理考核先进单位和先进个人、脱贫攻坚先进单位、烟叶生产先进单位和先进个人进行了表彰。
    会上，道林镇党委书记彭舜作了以《旗帜鲜明讲政治 雷厉风行抓落实 为建设“千年古镇、幸福道林”而努力奋斗》为题的工作报告。他号召全镇上下要旗帜鲜明讲政治，雷厉风行抓落实，聚焦“强党建、促发展、惠民生、保稳定”四个重点，提高政治站位，突出学习重点，抓好统筹结合，高标准高质量开展好党史学习教育，扎实做好各项工作，奋勇走在前列。

据悉，2020年以来，全镇上下在认真做好疫情防控的同时，围绕市委、市政府“决胜全面小康、决战脱贫攻坚”和“工业强市、幸福宁乡”的战略部署，认真贯彻落实市委各重要会议要求及市委主要领导关于道林工作一系列指示精神，抓发展重点、补扶贫短板、强作风弱项，致力将道林的区位优势、生态优势、人文优势和班子优势转化为发展优势；紧盯基层党建、脱贫攻坚、经济发展、人居环境、民生保障五大重点，以改作风为主线，形成了党政班子心齐气顺、镇村干部凝心聚力、全镇上下政通人和的工作局面。

2021年，道林镇将重点办好五方面的事，即：办好核心的事、办好发展的事、办好创新的事、办好民生的事、办好提高能力的事。全面开展“零纠纷、零上访、零违建”的“三零村”创建工作；以古镇开园为契机，引进社会资本，高标准指导建设一批现代农庄、家庭农场、诗意民宿，让道林成为好玩、好吃、好住的乡村旅游目的地，大力营造“人人维护道林形象、人人都是道林形象”的良好氛围；持续推进安全生产专项整治三年行动，深入开展道路交通安全整治行动，防范和打击各类违法犯罪活动，继续保持长沙市平安建设优秀乡镇称号。
    刘永红充分肯定了道林镇2020年以来各项工作所取得的成效，他指出，过去一年，道林镇在党建引领、安全生产、项目建设、环境保护、文明创建、民本民生等方面做了大量卓有成效的工作，取得了令人瞩目的成绩。
    他强调，全镇上下要“学党史 悟思想 办实事 开新局”，要以古镇开园为契机，深入贯彻落实“三高四新”、“四联五到位”民情大走访等精神，坚持以人民为中心的发展理念，坚持问题导向，突出项目建设，围绕农旅结合做文章，进一步优化营商环境，汇聚更多乡贤乡友力量，助力乡村振兴，以优异成绩，为建党100周年献礼。
    当天，大会还邀请宁乡市党史学习教育巡回指导组第九指导组组长谢政讲授党课。深入学习党的百年历史，从党的光辉历程中，感悟真理的力量，汲取奋进的力量，更加坚定有力的加快道林镇现代化建设。

道林镇：违建“零容忍”，集中整治建房乱象

发表于 2021-04-08 分类于 乡镇动态 本文字数： 664 阅读时长 ≈ 1 分钟

为落实最严格的耕地保护制度，坚决遏制耕地“非农化”，市农业农村局、市自然资源局牵头指导，在全市范围开展乱占耕地集中整治专项行动，道林镇积极响应，经过一周的整治，现已取得阶段性成果。
    近日，记者来到道林镇，由该镇多部门单位组成的联合执法队伍，正在进行例行巡查，发现河东新村大林冲组有一村民试图违规搭建杂屋。执法人员一行当即叫停正在施工的工人，并与屋主取得直接联系。经核实，该处确系违规建筑，执法人员当场下达责令整改违法行为通知书，要求屋主三天内自行拆除未经批准搭建的建筑，并认真向屋主宣传解读了当前的村民建房管理政 策。随后，执法人员一行来到河东新村下湖塘组一户准备拆旧建新的村民家中，核实空地为何种土地性质，并告诫该村民建房范围绝对不能超出红线范围之外。
    道林镇自然资源和生态环境办主任刘正林告诉记者，接到卫片图斑整改的通知后，镇里主要领导高度重视，组织镇村干部召开大会，“2020年第四季度，整个道林的卫片图斑有92例，我们逐个卫片图斑进行核查，扎实开展整改，目前已全部完成整改，并申请销号。”
    据了解，该镇结合实际，印发了《道林镇2021年村民建房管理实施方案》，从审批流程、审批面积、审批对象、宅基地选址方面做出了详细规定，并加强了日常监管巡查，将该项工作纳入绩效考核。刘正林表示，下一步，道林将继续按照市农业农村局、市自然资源局的指导要求，严格落实管理办法，确保村民建房工作走上正轨。

道林镇荸荠丰收，助农增收

发表于 2021-04-04 分类于 乡镇动态 本文字数： 832 阅读时长 ≈ 1 分钟

这个清明小长假，近郊游成为了不少市民的选择，宁乡市道林镇风景优美，距离长沙市区不到一个小时车程，成为了不少长沙市民青睐的近郊游打卡地。
    “早就听朋友推荐过，道林镇山清水秀，种植的蔬菜、水果特别好吃！”饶女士带着家人来到道林镇游玩，并根据朋友的介绍，到该镇的善山岭村田间地头玩了一圈，这儿种植的1200亩荸荠喜获丰收，既是踏青郊游休闲散心，又能体验农家采摘乐趣，可谓一次游玩、多重体验。
    记者获悉，为盘活现有的土地资源，善山岭村在去年4月成立了专业的荸荠种植合作社，从广西引进种苗和技术，并免费为村民提供种植技术培训，安排统一种植，将普通村民变成了“种植工人”。善山岭村党总支副书记王青山给记者算了一笔账，按平均亩产2500公斤、售价5.6元/公斤计算，荸荠种植可为当地村民增收逾150万元，去掉翻耕、种子、肥料以及人工费，每亩纯收益可达2000元。王青山开心地介绍说：“听说咱们村里有荸荠挖，这阵子有不少游客来体验下地的乐趣。”
    荸荠丰收，村民们也在想方设法找销路。王青山告诉记者，荸荠在常温条件下可保存8到10个月，现在采挖的荸荠还有5000多公斤没有卖出去，“暂时都没有进行深加工，欢迎市民游客朋友来采挖体验！”
    得知善山岭村荸荠丰收待采挖的消息，宁乡市文化旅游投资有限公司的员工纷纷给村里下起了订单。记者了解到，道林古镇、炭河古城等都是该公司运营管理的项目，随着道林古镇景区的盘活运营以及节会的开展，大量游客的到来带动了周边旅游业的发展和当地农副产品的销售，给乡村振兴注入了强劲动力。
    最近有近郊游计划的市民朋友以及愿意帮助村民的爱心人士，如果你有采购荸荠的需求或想体验采挖，可以随时来一场道林游，收获不一样的体验哦。

道林镇开展防汛应急演练，确保安全度汛

发表于 2021-04-01 分类于 乡镇动态 本文字数： 634 阅读时长 ≈ 1 分钟

为有效应对发生洪水灾害时确保人民群众人身安全和生命财产安全，提升应急抢险能力，3月31日，道林镇开展2021年防汛应急演练。本次演练共分巡查组、转移安置组、抢险救援组、治安秩序组、医疗救护组、技术指导组和后勤保障组7个工作小组，共有机关干部、民兵、医务人员和志愿者共计160余人参加。
    演练共分两个部分，抢险狮子山地质灾害点和清水塘水库救援。演练开始前，由镇防汛指挥部副指挥长陈伟军对各组人员职责进行说明，对防汛演练工作进行安排部署。演练现场，执法车、救护车、抢险工具等一应俱全，党旗、“道林镇社工站”、“志愿者服务队”等红旗迎风飘展。

收到巡查组险情报告后，各组人员立即投入抢险战斗，全面实施紧急救援。转移安置组的队员手持喊话器，一边跑一边向群众喊话，队员们挨家敲门通知、动员村民转移，并帮助村民将打包好重要财物，转移到安全地区；医疗救护组为安全转移人员进行了简单的医疗和卫生检查；抢险救援组利用提前备好的防汛布、铁锹等抢险工具开展抢险行动；治安秩序组对地质灾害点周边进行警戒，实施交通管制，防止无关人员进入，并在路口上设立“危险禁止通行”标牌。各组各司其职，有序开展疏散群众、转送伤员等救援工作，整个救援过程井然有序、措施得当，取得了良好效果。
    通过这次演练，进一步提高了机关干部、民兵等人的防汛应急处置能力，提高了广大人民群众的防灾避灾意识，为确保我镇安全度汛奠定了基础。
