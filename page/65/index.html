道林镇为“共产党员户”授牌

发表于 2018-06-07 分类于 乡镇动态 本文字数： 370 阅读时长 ≈ 1 分钟

6月6日，镇党委委员、组织委员荆红叶前往靳水村、华鑫市村，为两村“共产党员户”进行授牌，并召开村党小组长会议。为进一步加强基层党员队伍建设，发挥党员先锋模范作用。近日来，道林镇充分利用支部主题党日活动载体，在全镇党员中开展“共产党员户”授牌活动。通过集中授牌与走访授牌相结合的方式，为全镇先进党员、老党员授予“共产党员户”牌匾，并在家门前显著位置进行悬挂。公开亮明党员身份，主动接受群众监督。进一步激发了广大农村党员的责任意识和奉献精神。
    组织委员荆红叶在村党小组长会议上强调：一是要进一步提高党务工作者的思想认识，进一步压实党建工作责任；二是进一步强化党员意识，让党员主动作为、接受群众的监督；三是进一步巩固基层党建基础，发挥党员密切联系和服务群众的作用，同时发挥好党员教育和引导群众的作用。

道林镇部门联动打赢蓝天保卫战

发表于 2018-05-10 分类于 乡镇动态 本文字数： 476 阅读时长 ≈ 1 分钟

因地制宜，全面覆绿。采取种草籽、种树苗为主、贴草皮为辅的方式对全镇17块裸露土地实现全部覆绿。栽种植物品种繁多，包括爬山虎、樱桃树、夹竹桃树、樟树、茶梅、红桎木、金森女贞。针对陡坡山坡，采取先降坡后骨架护坡的方式防止沙土流失。覆绿实现全覆盖，通过前期调查摸底以及现场考察，将纳入覆绿范围的1151亩裸露山坡及土地进行了整改，其中15块土地种草籽541亩，105亩种草皮；另2块种植树苗，共610亩。加强后期维护，聘请专业人员进行后期盖膜、洒水、施肥，爱卫办每日进行巡查，保证出苗率。
    部门联动，上下一心。爱卫办加强裸露黄土覆绿的部署与督查，城建城管办集中整治集镇“脏、乱、差”，拆除镇域内违法搭建的建筑物、构筑物，共拆除违章建筑15处，约300平方米，对集镇、主公路定期清洗除尘，多次发现并制止出现污染路面行为的车辆。综治办、安监站、派出所联合行动，打击矿山非法开采，取缔非法场地8个。环保站建立污染企业“黑名单”制度、禁止垃圾焚烧现象。农办禁止秸秆焚烧。河长办开展河道整治、防止非法捕鱼、养殖户禁养喂养。

道林镇召开安全生产工作会议

发表于 2018-04-28 分类于 乡镇动态 本文字数： 686 阅读时长 ≈ 1 分钟

为进一步贯彻落实市安委会会议精神，强化安全思想意识，4月28日，道林镇召开安全生产工作会议，分析当前安全形势，就进一步做好安全隐患排查整治工作进行再部署、再落实。各村（社区）安全专干、机关办负责人、联村干部、各部门单位及重点企业单位负责人参加会议。
    会上，镇党委委员、副镇长张文宏对第一季度安全生产工作进行了总结，就下一步安全生产隐患大排查大整治工作进行了具体安排和部署，明确责任分工，要求对全镇各村（社区）及各办主管范围内的生产经营单位和人员密集场所的特种设备、危险化学品、建筑施工、道路交通、消防安全等开展安全大排查大检查。镇党委副书记、镇长陈浩结合实际，对下一步工作提出具体要求。
    会议确立实施镇领导安全生产检查每月一次；镇安监站各相关职能部门联合检查每月一次；村安全专干隐患排查每月一次；行业单位、部门单位一日一检查，一月一上报；各企业单位每日一自查，一月一上报的道林镇“五位一体”的安全生产监管机制。强调全镇要把安全生产工作摆在更加突出的位置，切实增强政治意识、忧患意识和底线思维，强化监管责任，真正将安全生产各类隐患排查整改的各项措施落到实处，切实防范各类安全事故发生。要吸取教训、压实责任，建立健全安全生产责任体系；要举一反三，警鉴全镇，推进安全生产隐患大排查大整治；严格督查、严肃问责，确保安全生产各项任务落到实处。各村（社区）各科室、各单位要居安思危，未雨绸缪，强化“红线”意识，细之又细，严之又严，扎扎实实地把安全生产工作做到位，确保全镇安全生产形势保持稳定。

道林镇多举措积极应对强降雨

发表于 2018-04-24 分类于 乡镇动态 本文字数： 669 阅读时长 ≈ 1 分钟

针对4月21日至22日强降雨，道林镇迅速行动，做实安排，抓牢部署，积极防范应对。
    紧急动员、铺排工作
    我镇组织召开全体镇村干部会议，要求全体机关职工取消周末休假，切实下村到户做好巡查和摸排工作，要求各村（社区）积极摸排地质灾害隐患点和土坯房情况。
    危情时刻，紧急转移
    4月22日晚暴雨来临之前，全体党政领导组织并督促各线办人员、村干部对全镇12处地质灾害隐患点、约150户土坯房屋所涉及的群众进行分片包户，做好监测、预警和转移等工作。
    截至到4月23日凌晨2点，共转移群众41户、63人，对房屋暂时安全，未转移的，逐户上门巡查并告知安全的转移地点等。
    居安思危，未雨绸缪
    为切实做好我镇辖区内发生的地质灾害应急防治工作，避免和最大程度地减轻灾害造成的损失，提高救灾工作水平和应急反应能力，确保人民群众生命财产安全和社会稳定，我镇在前期建立了统一领导、分级负责、职责明确、运转有序、反应迅速、处置有力的突发地质灾害应急体系,要求高效有序地做好突发地质灾害应急防治,并制定地质灾害应急预案。利用村村响进行防汛宣传、在大美道林微信公众号上积极推送天气变化情况和避险防范小知识，让广大群众切实提高警惕意识，提前做好防范工作。

道林镇向渔业违法行为亮剑

发表于 2018-04-10 分类于 乡镇动态 本文字数： 390 阅读时长 ≈ 1 分钟

为深入贯彻落实宁乡市依法打击非法电鱼“亮剑”行动的相关要求，道林镇通过前期广泛宣传，重拳击出非法电鱼行为。
    3月起，道林镇政府连续出动宣传车前往各村组，同时利用“村村响”广泛宣传严厉打击非法电鱼的相关政策。
    4月9日晚，宁乡市渔政监督管理站联合道林镇河长办、农业办、畜牧水产站、城建城管办、道林派出所等部门，出动执法车3台，执法人员20人，对我镇靳江沿线及镇域内水域开展集中整治。
    此次行动，共收缴非法捕鱼装备7套，抓获非法电鱼人员6名，并对非法电鱼者进行了严厉的批评教育并予以处罚。渔政执法人员将没收的设备带回统一销毁。
    下一步，道林镇将进一步加强渔业执法力度，维护镇域河流水域良好的生态环境。
