“栉风沐雨扶贫路、砥砺奋进奔小康”，宁乡市国资集团帮扶道林镇河东新村工作侧记

发表于 2020-06-15 分类于 乡镇动态 本文字数： 3.3k 阅读时长 ≈ 3 分钟

近一年来，帮贫困户销售农副产品近10万元，为有劳动能力的贫困群众解决就业15人，鼓励贫困户自主发展养殖产业32户，安排专项资金70多万元用于解决贫困群众的住房、饮水等问题……作为道林镇河东新村的精准扶贫帮扶后盾单位，宁乡市国资集团主动作为，充分发挥自身优势，整合自身资源，与当地镇村一道共谋脱贫攻坚，用心用情用力讲述着“一个也不能少”的动人故事。
    当好“强力支持者”，为解决住房等问题出资出力
    5月12日，宁乡市国资集团党委书记、董事长曾辉专程来到道林镇河东新村，就该村建档立卡贫困户的住房安全、饮水安全等问题进行实地调研，并与当地镇村干部、驻村扶贫工作队员一起分析问题，确定解决问题的对策。
    20天过去，6月1日，看着新房即将粉刷完毕，河东新村贫困户黄新年开心地说，以前想都不敢想的变化，离不开市国资集团的强力支持。
    黄新年患有精神二级残疾，女儿正在读书。由于房子破旧，刮风下雨总担心会出问题。由于家里的经济条件太差，就算按照相关政策获得相应补助，他还是觉得自己无力建好新房。
    2020年春节一过，市国资集团工作队员再次上门，动员黄新年通过建新房的方式彻底破解住房不安全的问题，并和他一起算了建新房的账。黄新年表示，原先他家曾被村上定为危改对象，除去财政补贴，建新房自己只要花费2万多元，但仅靠种植水稻、低保和残补为收入来源的家庭来说，解决这笔资金比登天还难。
    就这个贫困家庭的住房问题，市国资集团进行了具体研究，为其制定了一个新建房屋方案。除去黄新年自己要求地面架空，花了2000多元水泥预制板材料钱，其余6万元建房资金全部由市国资集团承担。
    就这样，黄新年仅筹措资金2000多元，便建起一栋60平方米、一厅两房一厨一卫的新房。到6月底，黄新年就能和女儿一同搬进新家了。黄新年开心地对记者说：“感谢党的好政策。原来这里是土砖房，到处漏水，破旧不堪。如今，即将住上新房子，感到非常高兴，很满意。”
    曾辉告诉记者：“坚持问题导向，切实为贫困群众解决各种突出的具体问题，是我们特别注重做好的帮扶工作。”用心帮扶过程中，该集团在河东新村摸排贫困户住房、饮水等要尽快解决的问题八类84个。对于这些问题，集团党委高度重视，组织召开专题会议进行研究，明确安排扶贫工作经费70多万元。在当地镇村的着力推动和村民的积极配合下，这些问题可望在6月底前清零。
    当好“热情销售员”，一年帮忙卖出近10万元农副产品
    “彭大姐，等会要带回宁乡的20斤谷酒，钱已通过微信转给您了，请查收。”
    “钱已收到，谢谢夏书记！”
    6月5日，宁乡市国资集团驻道林镇河东新村第一书记夏强将200元钱通过微信扫码付款给该村莲花组贫困户彭筛筛。拿到卖谷酒所得的收入，她感到十分满意。
    “驻村帮扶工作队给予了很大的帮助，今年家里面添置了全自动洗衣机、电烤桌，买了新床等，女儿、儿子又把家里简单粉刷了一下，日子越过越好了。”得益于产业扶贫政策的实施，贫困户彭筛筛家的种养业有了较大发展。目前，家里饲养了60多只鸡，喂了4头土花猪，菜园子里种满了空心菜、黄瓜、辣椒、茄子等应季蔬菜，生活有了新起色。
    只有及时把农副产品销售出去，贫困群众的收入才会有效增加。 帮扶工作深入开展，该集团想方设法拓展销路，帮贫困户销售农副产品。夏强在走访过程中发现有不少贫困户自主发展产业，但生产的农副产品却找不到销路，一定程度上削弱了他们的积极性。为了让贫困群众服下“定心丸”，提高发展产业的劲头，夏强当场承诺：“你们负责生产，我们来帮忙销售，只要有要卖的东西就给我打电话。”
    “河东新村贫困户大量出售土鸡、洋鸭、鹅、酸枣皮、谷酒……有需要的可以和我联系”，夏强不断发布贫困户农副产品信息。他的微信群、朋友圈，尽是各种叫卖帖。他还有一张贫困户农副产品销售统计表，上面密密麻麻记录了他驻村以来为贫困户销售农副产品的情况。因为经常要前往宁乡、长沙等地送货，不到1年的时间，他的新车已跑了3万多公里。
    就这样，市国资集团坚持利用自身信息资源，拓宽渠道帮助贫困户销售农副产品，积极引导贫困群众通过辛勤劳作改变自己的生活。通过线上销售、帮扶干部跑腿、联系爱心企业和爱心人士等形式，已累计帮助贫困户销售鸡、鸭、鹅、谷酒等农副产品，销售金额近10万元，切切实实让贫困群众脱贫致富奔小康。
    种种得力举措，促使消费扶贫带动产业扶贫，助推脱贫攻坚。河东新村党总支书记蒋丙洪表示：“农副产品能够迅速变现，对贫困群众起到了很大的鼓励作用，大家发展产业的积极性高涨。”据统计，河东新村51户建档立卡贫困户中，目前发展养鸡、养蜂、养猪和优质稻、秋葵等种养产业的有32户。积极发展种养产业，成为当地建档立卡贫困户脱贫增收的重要途径。
    当好“高效牵线人”，引进百亩秋葵产业再添致富路
    授人以鱼不如授人以渔。为了让贫困群众有能力走出困境，增强“造血”功能，河东新村积极在产业扶贫上谋创新、求突破，做到因人因户实施产业脱贫项目。
    在河东新村秋葵种植基地里，秋葵长势喜人，伴随着滚滚热浪，几个村民忙着除草打理基地，现场欢声笑语不断。
    发展什么产业既能把村上闲置土地利用起来，又能提高村民和贫困群众的收入？市国资集团积极牵线搭桥，通过“公司+农户”的发展模式，引入聚鑫长源农业科技有限公司，发展秋葵和季节性蔬菜种植产业。实行统一供种、统一种植、统一管理、统一收购，同时与种植户签订了回收合同，解决他们的后顾之忧，进一步激发村民种植秋葵的积极性。
    今年全村流转120亩闲置农田种植秋葵，带动了3户贫困户脱贫致富。杨国清就是其中的一户。他家有7口人，由于家里开支较大，经济基础薄弱，日子过得相当困难。今年，他家种植了近3亩秋葵，长势良好，每亩预计收益达6000余元。“多亏了党的好政策，不仅让我们不愁吃、不愁穿、不愁住，还帮助我脱贫致富，对今后的生活更加有信心了。”正在田间劳作的杨国清说。
    “妻子智力四级残疾，儿子精神三级残疾，我不能外出打工，抽空就到这来做事，一个月能拿到2500元左右的工资。”谈及在家门口秋葵基地就业的感受，河东新村贫困户王祥生非常感激。
    自今年2月基地建立，百亩秋葵基地就吸收了本村3名贫困劳动力就业。下一步驻村扶贫工作队还将鼓励更多贫困群众加入种植队伍，通过自己的双手创造财富，早日脱贫致富。
    全村建档立卡贫困户51户116人，至去年底已有42户97人稳定脱贫，剩余9户19人今年头5个月人均纯收入超4000元，将在今年全部脱贫。在河东新村，市国资集团充分发挥后盾单位作用，持续增强帮扶力度，与当地干群一起，正沿着脱贫攻坚大道朝着既定目标坚定迈进。

道林镇将整治“触角”延伸至农村道路交通“末梢”

发表于 2020-06-09 分类于 乡镇动态 本文字数： 470 阅读时长 ≈ 1 分钟

为有效铲除严重影响交通安全的突出隐患，确保道路交通安全平稳有序，近期，道林镇多措并举，在全镇范围内开展交通安全顽瘴痼疾集中整治行动。

6月4日，道林镇为进一步推进交通问题顽瘴痼疾整治，在靳水村组织开展了顽瘴痼疾安全宣讲活动，将平安送到居民手中。

通过开展交通安全宣传劝导活动，道林镇进一步加强村民的交通法规观念和交通安全教育，预防和减少交通事故的发生确保道路安全与畅通。道林镇党委委员、副镇长周新湘介绍，该镇制定了针对村社区和相关成员单位的考核评分细则，并按照考核评分细则进行了深入动员，同时将组织各村和社区开展半个月一次的宣教活动，下一步，该镇整治办将联合交警二中队和交管站、城管等执法部门，进行每月不少于6次的集中整治行动，每天分村、分社区、分片区进行交通违法的劝导活动，并针对重点人群以及学校等重点隐患地区进行全面的宣教和排查。

还通过在各村、社区定点播放音频、微信群转发安全出行知识等方式将整治“触角”延伸至农村道路交通“末梢”。如今，教育引导广大群众自觉遵守交通法律法规，提高群众安全意识，在道林镇已形成了全民参与、共建共享的浓厚氛围。

王伟胜赴宁乡道林督战调研农村危房改造和乡镇污水处理厂建设运行工作

发表于 2020-06-04 分类于 乡镇动态 本文字数： 542 阅读时长 ≈ 1 分钟

6月3日，长沙市住建局党组书记、局长王伟胜一行来到宁乡市道林镇督战农村危房改造工作进度，调研乡镇污水处理厂运行情况。

来到道林镇靳水村，王伟胜入户走访了危房改造对象，实地督战农村危房改造工程进度、质量安全及鉴定挂牌等工作情况。他指出，要全面贯彻落实脱贫攻坚“住房安全有保障”工作要求，紧盯目标任务，跟踪管理服务，抓实做细，一鼓作气，确保全市符合农村危房改造条件的对象“应改尽改”。
    来到道林镇污水处理厂，王伟胜一行现场查看了设施设备运行情况，详细了解了管道建设以及污水收集情况，针对目前乡镇污水处理存在的困难和问题进行了深入调研。王伟胜指出，要因地制宜，分类管理，切实改善农村水资源环境；要加强管网建设，提高处理工艺，做到污水处理全覆盖；要广泛宣传，提升意识，普及知识，达成共识，坚决打赢污染防治攻坚战。

王伟胜强调，一是要加快危改工作进度。要进一步压实责任，倒排工期，确保4类重点对象危房改造6月底前全面完工；二是要强化安全质量管理。要进一步加大技术指导力度，加强质量安全巡查，确保建成“满意房”“放心房”；三是要提升环境治理水平。要进一步推进乡镇污水管网等基础设施建设，提高乡镇污水处理设施的运行效率和管理能力，推动乡村环境整体品质提升。

道林镇四个全覆盖抓实抓细住房安全保障工作

发表于 2020-06-02 分类于 乡镇动态 本文字数： 481 阅读时长 ≈ 1 分钟

一是理顺体制机制，做到责任全覆盖。建立危房清零整改由党委书记牵头、班子成员包村负责，各村（社区）党总支书记、驻村第一书记和危改专干对危房改造负直接责任的工作体系。
    二是深入精准摸排，做到鉴定全覆盖。镇域内共摸排核查490户土坯房（“四类对象”160户、边缘户119户、子女住安全住房父母住危房211户），其中属于C\D级危房的有86户，鉴定为B级但因漏风漏雨需要修缮的有112户。
    三是分类因户施策，做到整改全覆盖。对于“四类对象”，通过市委、市政府相关政策和镇村兜底解决；对于边缘户，通过镇村奖补、自筹、爱心人士捐赠解决；对于子女住安全住房父母住危房的，利用舆论、法律、政策等手段上门做工作解决。
    四是争取资金支持，做到保障全覆盖。通过向上级争取、社会捐赠、机关节省，多举措、多渠道筹措资金。目前，宁乡市国资集团、电力公司、城郊街道等后盾单位已资助100多万元、社会捐赠30多万元、机关节约100多万元，用于危房改造工作。

道林镇召开当前工作部署会暨党委书记讲党课

5月25日上午，道林镇召开当前工作部署会暨党委书记讲党课大会，全体机关干部职工、驻村第一书记各村（社区）党总支书记、村（社区）扶贫专干和环卫专干、驻村辅警参加。

道林镇表彰奖励7名油菜种植能手

发表于 2020-05-20 分类于 乡镇动态 本文字数： 453 阅读时长 ≈ 1 分钟

近日，道林镇在靳水村举行靳水源油菜示范基地种植能手表彰会，荣获“种植标兵”“种植能手”称号的7名村民身佩红色绶带，手持荣誉证书和奖金，脸上露出了自豪的笑容。
    据了解，为保障粮食安全底线，道林镇稳定种植冬季油菜1万亩，以“合作社+农户”的农业经营模式，以市场为引导，因地制宜发展特色农业；以奖代补、以奖促建的方式，激励当地油菜种植户通过勤劳双手脱贫致富。近年来，道林镇积极引导和技术支持，一大批农民实现了从种粮大户到油菜大户的“华丽转身”，有效带动了农民和村级集体经济增收，保护好了乡村的绿水青山，提升了镇域颜值，为发展乡村旅游创造了有利条件。
    油菜种植大户许宁益，流转土地种植油菜200余亩，以亩产300多斤油菜籽的高产量荣获唯一一个“种植标兵”称号。他掩饰不住内心的喜悦说道：“搭帮政府、凤凰山水稻专业种植合作社的重视，今年油菜产量比较高，估计有20万元收入。有收入，有荣誉，真是‘双丰收’，我明年还会继续种。”

5月3日，道林镇交通问题顽瘴痼疾整治办公室联合交警二中队、道林交管站组织道林社区支村两委志愿者在道林集镇及神仙岭等景区，持续开展交通问题顽瘴痼疾集中整治行动。
    这是该镇交通问题顽瘴痼疾集中整治行动的第六场，在市整治办的统筹安排下，该镇整治办先后组织志愿者走进学校、街道、商铺、景区等不同场所，组织开展交通问题顽瘴痼疾隐患排查和专项督查。对横穿马路未走斑马线，低头族不遵守信号灯，骑摩托车不戴头盔，摩托车违法装置遮阳棚，开车不礼让斑马线，未系好安全带，驾驶机动车接听电话等一系列交通问题顽瘴痼疾进行劝导和宣教。

道林镇召开第一届人民代表大会第四次会议

发表于 2020-04-27 分类于 乡镇动态 本文字数： 780 阅读时长 ≈ 1 分钟

4月24日上午，宁乡市道林镇第一届人民代表大会第四次会议胜利召开。会议听取和审议了《政府工作报告》《人大工作报告》《财政预决算报告》。
    大会期间，与会代表对三个报告的内容进行了分组讨论，大家踊跃发言，积极建言献策。代表们紧紧围绕中心工作和人民群众生产生活的热点问题展开讨论，对《政府工作报告》等提出建议意见，切实体现了人大代表为人民服务的情怀。
    据悉，今年，道林镇将聚焦决胜脱贫攻坚，提升发展的向心力。挂图作战，确保38户61人如期脱贫，投入122万元推动养鸡、优质稻种植、养蜂、养鹅“四跟四走”项目投产见效，预计带动307户810人增收。聚焦人居环境整治，提升发展的竞争力。重点打造9个美丽屋场，争创4个长沙市卫生村和1个湖南省卫生村。聚焦产业项目发展，提升发展的支撑力。进一步落实招商责任，大力推进包装策划和招商工作，争取成功引进2家规模项目落户道林。聚焦民本民生改善，提升发展的凝聚力。优化教育资源、师资队伍、评价体系，投入442万元新建道林中小教学楼、田径场。聚焦政府自身建设，提升发展的保障力。做到始终坚持党的全面领导，着力提升法治水平，全面提高办事效率，牢牢守住廉洁底线。
    会议通过了张亚光同志辞去副镇长职务请求的决定，并以无记名投票的方式选举周新湘同志为道林镇副镇长，发布了大会公告。新当选人员进行了宪法宣誓。会议还票决了道林镇2020年民生实事项目。
    道林镇党委书记彭舜代表镇党委对新当选的副镇长表示热烈的祝贺，并对新的领导班子提出了更高的工作要求，号召全镇上下干在一线，干到一起，干出一流，坚定信心，团结奋进，锐意进取，为建设“千年古镇 幸福道林”而不懈奋斗。
