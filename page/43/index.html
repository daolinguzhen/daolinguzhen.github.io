道林镇安全生产集中攻坚“快准狠”

发表于 2021-07-12 分类于 乡镇动态 本文字数： 1k 阅读时长 ≈ 1 分钟

为扎实抓好安全生产大排查大整治集中攻坚行动，道林镇迅速行动，全面部署，强化监管，狠抓“七日攻坚”，确保了全镇平安稳定。
    提高站位、加强部署、压实安全生产责任
    道林镇党委、政府高度重视安全生产“七日集中攻坚行动”，6月15日，组织召开了全体党政领导、机关干部职工、部门负责人、村支部书记、企业负责人参加的安全生产工作大会，安排部署“七日集中攻坚行动”。
    6月21日镇党政领导班子召开会议，学习传达中央省市各级领导讲话精神及安全生产会议精神，学习习近平总书记关于安全生产工作的重要论述；认真研究全镇安全生产隐患摸排整改情况。
    全员出动、全域覆盖、浓厚宣教培训氛围
    充分利用屋场会、四联五到位工作、安全生产月宣传咨询活动、网格化微信群及各类工作微信群、“村村响”等方式大力宣传液化气使用、道路交通、生产经营、特种设备、防溺水等安全知识并发放张贴各类安全宣传资料、图片35000份。全镇各村组召开屋场会116场次，参会人员2800人。出动22台次宣传车，沿各村组巡回宣传安全知识。
    迅速行动、多方协同、加大排查整治力度
    贯彻落实安全生产“党政同责、一岗双责”和“管行业必须管安全，管业务必须管安全，管生产经营必须管安全”的要求，组织各相关部门、单位开展安全生产大排查大整治。
    截至目前，全体党政领导带队检查安全生产工作20次，排查安全隐患75条。共出动执法人员140余次，对各行业开展联合整治15次。各部门单位对59家生产经营单位开展检查，排查隐患和问题167条，已整改152条，其余15条处于督促整改状态。
    针对以上发现的问题隐患，道林镇以责令整改、交办督办、行政处罚、移交查办等方式确保问题隐患实现闭环。
    道林镇通过深入开展安全生产“七日集中攻坚行动”，进一步夯实了安全生产工作基础，有效防控了安全事故的发生，确保全镇安全生产工作形式保持持续稳定好转。

道林镇召开非洲猪瘟防控7月集中整治工作部署会

发表于 2021-07-08 分类于 乡镇动态 本文字数： 246 阅读时长 ≈ 1 分钟

根据当前非洲猪瘟防控形势，按照国务院、农业农村部、省市相关做好非洲猪瘟防控工作的精神，为全面抓好非洲猪瘟防控工作，严防非洲猪瘟扩散蔓延，道林镇高度重视，于7月7日组织召开由动物防疫站、市监所、综合执法队、分管农业的村干部、屠商及生猪经纪人参加的非洲猪瘟防控7月集中整治工作部署会议。

会上，动物防疫站站长颜建龙对非洲猪瘟防控7月集中整治工作做了具体安排，所有参会屠商和生猪经纪人现场签订《宁乡市防控非洲猪瘟疫情承诺书》，表示将团结一致、守法经营，共同为防控非洲猪瘟等重大动物疫病作出应有的贡献。

道林镇召开“两优一先”表彰大会

发表于 2021-06-30 分类于 乡镇动态 本文字数： 436 阅读时长 ≈ 1 分钟

6月29日上午，道林镇召开优秀共产党员、优秀党务工作者和先进基层党组织表彰大会。镇全体党政负责人、各村（社区）党（总）支部书记、受表彰对象、镇党代表、镇属部门单位负责同志、乡村振兴派驻工作队成员等180余人参加大会。

大会在庄严的国歌声中拉开帷幕。现场，镇党委书记彭舜带领大家重温了入党誓词。同时，与会领导为全镇经济社会发展中涌现出的优秀共产党员、优秀党务工作者、先进基层党组织进行表彰，并为“光荣在党50年”老党员颁发纪念章。

镇党委书记彭舜号召全镇各党组织和广大党员坚定理想信念，做绝对忠诚的模范；站稳人民立场，作为民造福的模范；提振精神状态，做敬业奉献的模范；坚持率先垂范，做奋勇争先的模范，为全面建设“千年古镇、幸福道林”贡献力量。
    表彰大会后，该镇围绕“学史力行”主题开展党史学习教育第四次专题学习。市委党校教研室主任、副教授陶俊以《“作事不辞牛荷重”——学习何叔衡的“牛”精神》为题为全体与会人员讲授了一堂生动的党课。

道林镇开展道路交通安全联合执法行动

发表于 2021-06-24 分类于 乡镇动态 本文字数： 254 阅读时长 ≈ 1 分钟

6月23日晚，道林镇开展道路交通安全联合执法行动，镇党委副书记陈伟军组织镇道安委，联合市治超中队、交警一中队、交管站进行联合执法。
    整治行动主要排查过往机动车辆违反道路交通规则的行为，此次行动共检验挂车、货车33辆；查处轻微超限农用车1辆，教育放行；摩托车不戴头盔10人次，进行了警示教育。

通过此次道路交通安全联合执法行动，向驾驶人员宣传了交通法律法规知识，让他们真正意识到违规驾驶机动车辆对自身和社会带来的危害性，增强了驾驶人员的安全意识，确保了道路交通安全。

道林镇深入开展道路安全隐患点大排查工作

发表于 2021-06-16 分类于 乡镇动态 本文字数： 247 阅读时长 ≈ 1 分钟

道林镇为切实加强道路交通安全隐患治理工作，建立道路交通安全隐患排查长效机制，防止和减少交通事故的发生。在前期摸排治理的基础上，2021年6月15日，镇党委书记彭舜同志组织镇道安委、综合执法大队、交管站再一次对G354镇域范围内29公里路段进行地毯式隐患排查，共排查出23处安全隐患点。

该路段为辖区主干线，车流量大且路况复杂，交通事故频发，急需立即整改。经现场勘查测量，需增加波形护栏205米、振荡减速标线1813根、标志标牌（含凸面镜、爆闪灯）28块，1处路口（G354与道善公路交界）需优化。

道林镇开展“我为群众办实事、营养专家进校园”主题党日活动

发表于 2021-06-11 分类于 乡镇动态 本文字数： 464 阅读时长 ≈ 1 分钟

近日，道林镇卫生院党支部联合长沙市中心医院机关业务党支部和道林镇教育总支委，组织党员到道林中学开展“我为群众办实事，营养专家进校园”主题党日活动。

2021年全民营养周的宣传口号是“健康中国，营养先行”，主题是“合理膳食，营养惠万家”。
    湖南省营养学会、长沙市中心医院营养科主任孙艳就营养健康科普知识进行专题宣讲。她对中小学生多见营养性疾病作了简要介绍，重点对青春期营养需要、青春期配膳原则、如何选择运动饮料、合理膳食、少盐少油、控糖限酒、中国居民平衡膳食宝塔的运用等实用知识进行生动讲述。
    课堂上，学生、家长和老师们都全神贯注的听课，同时积极踊跃发言，主动参与问答互动，学习氛围十分浓厚。
    据悉，为净化校园周边食品安全环境，道林镇多部门携手，定期开展校园周边食品安全专项整治行动，让孩子们认清“5毛食品”的真相和危害，家、校、社携手，织密食品安全网，切实保障学生“舌尖”上的安全与营养健康。

道林镇：96名儿童点亮“微心愿”

发表于 2021-06-02 分类于 乡镇动态 本文字数： 998 阅读时长 ≈ 1 分钟

6月1日，宁乡市道林镇南塘小学礼堂内欢歌笑语，喜气洋洋，来自政府、企业、社会的各界人士和学校师生一起共同庆祝第71个“六一”国际儿童节。同时，道林镇庆祝“六一”儿童节暨留守儿童和困境儿童关爱保护活动也在此举行。

湖南省民政厅基政处处长廖剑波，湖南省民政厅社会组织管理局副局长鄂立辉，宁乡市政府二级调研员刘志勇，宁乡市民政局党组书记、局长潘志国，宁乡市道林镇党委书记彭舜，湖南省乐善社会工作服务中心理事长罗乐以及宁乡市人民检察院、市民政局有关同志参加活动。

“小朋友，如果恶魔来到我们身边，我们应该找谁求助？”活动开始前，宁乡市人民检察院检察官陈娟用儿童通俗易懂的语言，生动地进行了一场《中华人民共和国未成年人保护法》的专题宣讲。
    伴随着节奏轻快的开场舞《踏浪》，道林镇庆祝“六一”儿童节暨留守儿童和困境儿童关爱保护活动正式拉开序幕。

活动现场，还为96名儿童点亮“微心愿”，将自行车、电子手表等礼物当场送到孩子们手中，同时还联系爱心企业为6个困境家庭捐赠了床、衣柜、书桌等家具。
    据介绍，道林镇逐步建立健全了农村留守儿童关爱保护机制，形成了社工+志愿者+老师+村干部+监护照料人的“五位一体”关爱保护工作格局。该镇从2019年起，每年“六一”为儿童征集微心愿并发动社会力量助其圆梦，今年是第三个年头，每次微心愿都吸引了社会各界爱心人士踊跃参与，教育帮扶和学校关爱力度得到了普遍加强，群团组织和社会力量关爱服务参与度得到了极大提高。

活动对20名优秀志愿者进行了表彰，并为其颁发了荣誉证书。在道林社区儿童之家的孩子们表演的歌舞《听我说谢谢你》的音乐声中，活动落下帷幕。与会领导们一行参观了南塘小学儿童之家，并仔细查看了相关资料及帮扶记录。
    据悉，近年来，在宁乡市委、市政府的高度重视和正确领导下，在省、市民政部门的关心支持下，宁乡市民政局充分发挥职能部门作用，聚焦全市孤残儿童、事实无人抚养儿童、困境儿童和农村留守儿童，在全市278家村（社区）均建立了儿童之家，设立了1个市级未成年人保护救助中心，建立健全了乡镇儿童督导员和村（社区）儿童主任等基层儿童工作队伍，积极引导社会组织开展儿童关爱帮扶活动，不断完善农村留守儿童和困境儿童关爱服务机制，儿童权益得到了有效保障。

道林镇开展“党员服务示范门店”评选活动！

发表于 2021-05-27 分类于 乡镇动态 本文字数： 607 阅读时长 ≈ 1 分钟

5月22日道林镇召开党员大会对获评“道林镇党员服务示范门店”的党员进行授牌。
    
本次评选为宁乡市首次在党员经营户中评选服务模范、示范门店，此举创新了党员示范带动的举措，既增强了党员经营户的党员意识、责任意识、服务意识，又营造了经营户创先争优的良好氛围，充分发挥了共产党员的先锋模范带头作用。

据了解，本次评选经过党员申请、支部推荐、考察调研、会议评选等环节，过程公平公正、透明公开，充分综合考虑了党支部及党员群众的意见建议，得到了社会的一致认可和高度肯定。
“党员服务示范门店是一块金字招牌。我作为一名党员，将诚信经营，热情服务，提供免费停车、茶水与洗手间等，为古镇开园作出积极贡献。”道林社区党员、菜根香饭馆负责人、获评“党员示范服务门店”代表蒋江林获奖后开心地说。

道林镇相关负责人表示，开展党员服务示范门店评选不仅强化了党员经营户诚信经营、争先创优的意识，能带动普通群众向党员学习、以党员为榜样，共同提升道林镇良好的营商环境氛围，而且深化了门店经营户服务群众、服务游客的理念，完善了古镇商圈经营体系，将为道林古镇开园的游客服务工作奠定良好的基础。
    据悉，本次评选是道林镇党员服务示范典型的开始，接下来，道林镇还将开展党员服务示范岗、党员服务示范家庭等系列评选工作，为建党100周年献礼，为建设“千年古镇、幸福道林”贡献力量。
