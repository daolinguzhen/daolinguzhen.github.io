以练为战！靳水村开展全员核酸检测应急演练

发表于 2021-11-02 分类于 乡镇动态 本文字数： 440 阅读时长 ≈ 1 分钟

为充分发挥核酸检测在疫情应对与处置中的技术支撑作用，进一步提升新冠肺炎核酸检测能力和效率，11月2日上午，我镇在靳水村公共服务中心组织开展靳水村全员核酸检测应急演练，全体党政领导和9个村（社区）书记现场观摩。整个演练过程真采真检，设置2个采样点同时采样，持续了4小时共检测900余人。

演练现场设置了测温区、临时隔离区、登记区、采样区等区域。核酸检测采样组、现场工作组、秩序维护组、后勤保障组各组工作人员和志愿者各司其职，引导群众分批、有序进行测温、身份信息登记等，严格落实“1米线”间隔、规范佩戴口罩等防护措施，分10人一组依次开展样本采集。整个演练过程，各组人员分工明确、行动迅速、配合密切，所有志愿者和工作人员严格按照新冠肺炎疫情防控应急处置预案规范操作，达到了预期效果。
    通过此次演练，全面提高了防控人员的防范意识和应急处置能力，全面检验了应急指挥和组织协调能力。为坚决打赢疫情防控阻击战打下坚实基础，增强疫情防控的信心和决心。

近日，道林镇持续加大道路交通安全宣传力度，由联村办公室牵头，网格长为直接责任人，在全镇56个网格召开道路交通安全屋场户主会，多措并举开展道路交通安全宣传。

道林镇坚持三个“到位”做好防返贫监测与帮扶工作

发表于 2021-10-03 分类于 乡镇动态 本文字数： 691 阅读时长 ≈ 1 分钟

道林镇始终将防返贫监测和帮扶工作作为大事要事来抓，通过系列举措，进一步压实了工作责任，健全了工作体系，筑牢了防止返贫的底线。
    一是坚持责任落实到位。成立由党委书记为组长的道林镇防止返贫监测与帮扶工作领导小组，落实党委书记第一责任人责任，构建党委副书记统筹抓、分管领导具体抓的工作格局。党委会每两周研究一次防返贫监测与帮扶工作，组建镇防止返贫监测与帮扶办公室，明确办公室主任及专职工作人员3名。各村（社区）明确以党总支书记负总责，驻村干部、联村干部和村专干具体抓落实的工作机制。
    二是坚持风险监测到位。结合党员干部“四联五到位”民情大走访，每季度对全镇所有1.49万户农村人口“一超过、两不愁、三保障”及饮水安全情况进行全面摸排和监测，全面宣传防返贫监测与帮扶政策和全国防返贫监测信息系统手机APP使用方式。针对系统反馈的风险点数据，由驻村工作队、联村干部、村专干上门核实情况，截至目前已核实401条风险点数据，对反馈过风险点的人员进行动态台账管理，综合评估其风险指数。
    三是坚持政策帮扶到位。对已纳入监测的38户监测户每户安排一名镇干部进行一对一结对帮扶。针对监测户的致贫返贫风险“一户一策”制定帮扶措施，确保对象及时消除风险，38户监测户全部享受低保或特困供养等社会兜底保障，4户给予临时救助，6户享受残疾补助，4户享受教育补助。安排30个公益性岗位帮助脱贫户就近就业。制定《道林镇农村饮水安全工作预案》，切实保障监测户和稳定脱贫户饮水安全。

道林镇开展中秋、国庆节前食品安全专项检查

发表于 2021-09-20 分类于 乡镇动态 本文字数： 496 阅读时长 ≈ 1 分钟

中秋、国庆佳节即将到来，恰逢以“千年铁魂·湖南人”为主题的道林古镇即将迎来开园盛景，为有效保障食品安全，营造安定、欢乐、祥和的假日氛围。近日，道林镇食安办、市监所结合长沙市创建国家食品安全示范城市工作，聚焦辖区内食品生产流通重点行业，开展中秋、国庆及古镇开园前食品安全专项检查行动。
    针对“双节”食品消费特点，结合食品安全“问计于民”座谈会与“四联五到位”第三季度民情大走访中群众反映热点、难点问题以及在日常监督检查中暴露出的薄弱环节，相关单位对生产企业、商场超市、生产加工手工月饼的小作坊等重点场所开展专项检查。此次专项检查以超市、便利店、小作坊销售的月饼、糕点等节令食品为重点，主要针对各类食品及原材料存储摆放是否规范、外包装是否完整、生产和保质日期是否合规、票证是否齐全等方面，进行全面排查食品安全风险隐患，严防食品安全问题。
    除此之外，中秋、国庆节期间道林镇严格执行24小时值班和领导带班制度，密切关注食品安全动态，确保投诉举报电话畅通，及时受理群众投诉举报，积极维护群众生命安全和社会稳定。
