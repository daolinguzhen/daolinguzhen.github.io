道林镇“三个聚焦”全力推进人居环境整治工作

发表于 2020-12-02 分类于 乡镇动态 本文字数： 546 阅读时长 ≈ 1 分钟

一是聚焦顶层设计完善。遵循乡村整治“从干干净净、到整整洁洁、再漂漂亮亮”循序渐进的规律，积极探索分片包干的领导机制、上下联动的责任机制、多元支撑的投入机制、重奖重罚的考核机制，形成“政府主导、群众主体、村级服务、社会参与”的工作体系，促使环境卫生从突击整治向规范管理、常态保持转变。
    二是聚焦突出问题整治。以治垃圾、治水、治房、治厕、治风“五治”为抓手，对镇域内“脏、乱、污、堵”等突出问题进行有效整治。新建石金村10T压缩式垃圾中转站，各村（社区）垃圾减量50%以上；强化小微水体管控，龙泉湖村正在打造市级“五治”示范屋场；拆除空心房328处、违法建筑16处；完成农村改厕705座，实现旱厕“清零”；投入约400万元对古镇景区、集镇7条主要道路60公里、59个网格进行环境整治；完成复绿面积25万平方米，卫片图斑整改18处。
    三是聚焦动态监督管理。对村（社区）人居环境、卫生费收取、路长制、河长制等工作按照“分项检查、综合计分、逐月考核、定等排名”的方式进行考核，每月通报讲评，将各项工作落实情况纳入镇对村目标绩效考核，年底严格奖优罚差，倒逼各村（社区）压实工作责任，切实巩固农村人居环境整治成效。

道林镇持续推动“五老四教”工作

道林镇深入开展交通问题顽瘴痼疾集中整治

“经过这段时间的交通整治，道路两边乱摆乱放的现象基本消失了，过节集镇上也基本不堵车了，交通秩序比以前好多了。”道林社区居民张爹爹介绍。这是道林镇深入开展交通问题顽瘴痼疾集中整治的一个缩影。
    “自交通问题顽瘴痼疾集中整治行动启动以来，我镇第一时间成立了领导小组，把人民群众的生命财产安全放在首位，走出‘交通事故不可控’的误区，把隐患当事故来看。对全镇摸排的176处事故多发点科学分类、精准施策，提升了道路交通安全系数。”道林镇镇长姜晓表示，为了从根源上整治交通问题顽瘴痼疾，道林镇建章立制、立行立改，举一反三、由表及里，突出重点部位、关键环节，强力推进专项整治行动，抓牢抓实交通运输、道路交通安全生产。
    据了解，道林镇结合该镇交通运输安全生产、道路交通安全生产的实际，拟写了专项整治方案；成立了安全生产专项整治三年行动领导小组，组建了工作专班；召开了交通运输安全生产、道路交通安全专项整治9次工作推进会，宣教会议开到了各村（社区）、源头企业、加油站等，活动涉及到靳水、龙泉湖、华鑫市、鑫星等村的村民小组长、党员群众、志愿者等500余人，交通问题顽瘴痼疾劝导人数累计达1800余人次，多次约谈在建项目、公路、污水处理厂项目建设单位、施工单位、运营单位代表；多次召开源头企业和货运公司治超工作会，专题部署安全工作。
    “骑电动车也要戴头盔才能上路，安全第一，不要觉得麻烦，关键时刻它能救命。”“您好！这是交通问题顽瘴痼疾集中整治宣传资料，空闲时可以多看看。”道林镇靳水村组织开展的顽瘴痼疾安全宣讲活动，将平安幸福送到村民手中。
    道林镇党委委员、副镇长周新湘介绍：“我们在第一时间制定了集中整治行动考核评分细则，按照考核评分细则进行深入动员，同时将组织各村（社区），按照半个月开展一次宣教活动的频率，针对交通问题顽瘴痼疾开展集中整治宣教活动。同时，镇整治办联合交警二中队、交管站、城管执法大队等相关部门，每月集中整治执法行动不少于6次。另外，针对重点人群、学校等重点隐患地点进行全面宣教和排查。”同时，通过在各村（社区）采用悬挂横幅展板、发放宣传单、定点播放音频、微信群转发安全出行知识等方式，在全镇范围内掀起整治行动宣传热潮。
    道林镇自交通问题顽瘴痼疾集中整治行动以来，镇整治办联合交警二中队、交管站、综合执法队对货运车辆超限超载、无证上路、货物扬撒污染公路、路产路权及路域环境整治等违法行为进行综合治理，组织联合执法行动，检查重点源头企业，拆除电动车、摩托车遮阳伞。同时，以开展超限超载治理“夏季攻势”行动为载体，以花明楼卡（站）点为依托，强力有效打击了交通运输安全生产、道路交通安全的各类违法行为。
    安全无小事。对于接下来的交通问题顽瘴痼疾集中整治，姜晓表示，将突出重点领域，以“两客一危一校”为重点，以治超和打非治违为主抓手，进一步深化道路运输行业安全“隐患清零”行动。进一步加强马路市场的监管，防止死灰复燃，确保来之不易的成果。继续做好隐患的排查整改工作，做到全方位、多维度、零死角。建立问题隐患和制度措施“两个清单”，制定时间表路线图，明确整改责任单位和整改要求，坚持边查边改、立查立改，确保整治治理取得实实在在的成效。同时，进一步加大宣传力度，提升全民安全意识。

心宁之乡，待“宁”打卡道林：靳江明珠、千年古镇

发表于 2020-11-06 分类于 乡镇动态 本文字数： 456 阅读时长 ≈ 1 分钟

道林镇是一座有着1100多年历史的古镇，位于宁乡市东南部，古为商贾重镇，人文底蕴深厚，素有“小南京”之称。湘江下游支流靳江河穿镇而过，使得道林自古以来就有着伴水而居、枕水而眠的独特风韵。

道林古镇景区面积约3000亩，围绕“乡愁、古韵、穿越、风情”4个主题打造，分为乡土中国、潇湘故事、时尚风情、道林人家、田园牧歌五大功能片区，充分挖掘当地历史文化和民俗民风，原乡故里的风土人情和生活场景在这里得到集中的体现和延续。每年都有不少摄影爱好者、外地游客慕名来此，只为吃一口酸枣粑粑，尝一口米豆腐，嗦一口手工面，买一块烟熏腊肉……
    千年古镇的底蕴、古色古香的韵味、特色美食的风味，道林古镇现已成为长株潭及周边地区的网红打卡地。
    古香古色古城墙。道林特产酸枣糕、道林红曲鱼、坛子菜等本地特产颇受游客欢迎。笑颜绽放，快乐在古镇定格。古桥流水，互相辉映，置身在风情街，恍惚有种穿越感。怀真抱道，秀士如林，漫步这座古镇，寻找历史的痕迹。

记道林镇华鑫市村驻村第一书记张进

发表于 2020-11-06 分类于 乡镇动态 本文字数： 1.9k 阅读时长 ≈ 2 分钟

“真是太感谢了，帮我筹集资金，又帮我找人把房子修缮好。要不是你们，我真不知道该怎么办了？”近日，道林镇华鑫市村建档立卡贫困户黄成华感动地对驻村第一书记张进说。
    2018年3月经组织选派，原道林镇卫生院工会主席张进到华鑫市村任驻村一书记，主要负责精准扶贫工作。离开熟悉的工作环境和工作岗位，张进怀着忐忑的心情来到华鑫市村，接触陌生的脱贫攻坚驻村扶贫工作，一切从头开始。
    变“要我脱贫”为“我要脱贫”
    华鑫市村是道林镇西北边上与花明楼镇相邻的一个村庄，占地9.7平方公里，总人口3607人，其中建档立卡贫困户57户118人，占全村人口比率的4%，村上无资源和村级集体经济收入。为了迅速掌握建档立卡贫困户的所有信息，张进一来到华鑫市村，上午查阅建档立卡贫困户的一户一档资料、十大扶贫台账、整改台账以及之前的扶贫相关资料，下午就深入到建档立卡贫困户家中，宣传最新的扶贫政策，与他们交谈了解家庭生活和收入情况，并作好详细记录。很快，他就将村里的57户建档立卡贫困户的情况都详细了解清楚了，并将他们的基本情况、反映的问题和诉求一一记录在册。
    “第一书记驻村，除了找门路、筹资金、帮技术之外，更重要的是帮思想。”从驻村的第一天起，张进就立志把扶“智”和扶“志”作为工作重心，变“要我脱贫”为“我要脱贫”，把华鑫市村的贫困根子拔掉。
    “感谢张书记，帮我找到现在的工作，每个月有了固定收入。”建档立卡贫困户黄朝阳，肢体三级残疾，通过张进的思想引导想谋一份工作摆脱家庭困境。张进自己驾车带着他到宁乡金盾安保公司求职，但因为其身体不适合未被聘用。张进又找支村“两委”成员协商，安排他为村上的垃圾清运员。今年4月，黄朝阳和村上正式签订垃圾清运员劳动协议，每月固定有了1800元收入。贫困户姜小莲的女儿成娜，株洲技术学院药学专业毕业，想找一份适合自己专业的工作。张进多方联系，并请朋友帮忙，在双凫铺九芝堂药店为她谋得了一份工作。贫困户蒋冬彪、张先华失业在家多年，都想就业，但多方应聘无果，是张进通过各种渠道，介绍他们到长沙洋湖湿地公园上班，每月收入2800元，稳定达到脱贫要求。
    想方设法帮群众解决困难
    “点子好、办法多”是当地村民对张进的印象。自驻村以来，在开展扶贫帮扶的过程中，张进时刻把群众利益放在第一位，扑下身子真扶贫、扶真贫，获得了群众的认可。
    建档立卡贫困户罗文兵的妻子伍平系精神三级残疾，建档立卡贫困户蒋建华的儿子鲁超光系智力四级残疾，张进和扶贫专干周银花携同其家属，驾车送他们到宁乡市精神病医院进行残疾升级，并得到应有的政策保障。
    由于资金缺乏，张进多方努力，积极争取后盾单位、镇政府和支村“两委”的支持。再和贫困户多次沟通、交流，争取到他们的支持和理解，并自筹一部分资金，为贫困户鲁明、成友信、蒋冬文、杨战武、黄寿桃、蒋建华，五保户黄成华、刘金科等22户进行房屋修缮、自来水安装、厕所改造等。在资金有限的情况下，保障了他们的住房和饮用水安全，赢得贫困群众一致的称赞。
    不顾危险抢救群众财产安全
    8月18日晚9时许，风雨交加，张进接到耕湖塘组建档立卡贫困户杨辉武的求助电话，称其家房屋被树枝击穿，请求帮助。
    接到电话后，张进第一时间通知周银花一同赶往杨辉武家中，因周银花为女同志，当时风大雨急，她便叫上丈夫黄新佂一同前往。到达现场后，他们发现，两棵直径达20公分的酸枣树枝遭雷击后断裂，将杨辉武家的前后屋面击穿，屋面出现两个水缸大的窟窿，造成堂屋严重漏水。三人立即爬上屋面，用斧头、锯子对倒塌的树枝进行清理。待将处理好后，张进突然感觉足部疼痛，周银花用手电筒照看，才发现张进足部鲜血淋淋，立即将其送至镇卫生院进行伤口处理。第二天一早，张进又和支村“两委”成员商议杨辉武户受灾后的处置情况。
    “老百姓的事无小事，只要沉下心来为老百姓做事，就一定能获得老百姓的认可。”张进表示，当前脱贫攻坚已到了决战决胜的关键时刻，他将继续努力，踏踏实实为民办实事，不负组织和村民的信任。
