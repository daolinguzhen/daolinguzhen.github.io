宁乡市道林镇：以“党建+”深入推进乡村振兴

发表于 2023-07-24 分类于 乡镇动态 本文字数： 1.1k 阅读时长 ≈ 1 分钟

改善农村人居环境是实施乡村振兴战略的一场硬仗，事关广大农民健康福祉，事关农村社会文明和谐。
    今年以来，宁乡市道林镇深入践行党员“三亮三比”行动，通过“党建+人居环境提升、党员带头、群众参与”的模式，充分调动党员群众的积极性，扎实推进人居环境整治工作。道林镇在全镇范围内采取分网格包干、集中整治的形式努力提升人居环境水平，以长沙市生态文明示范乡镇为基础，为创建国家级卫生乡镇打下基础，为乡村振兴擦亮底色，为“千年古镇 幸福道林”建设添砖加瓦。
    强化党建引领，筑牢环境整治工作堡垒。道林镇党委会6次专题研究农村人居环境工作，印发《道林镇2023年农村人居环境常态化管理考核方案》，由党委书记任顾问、镇长任组长，实行镇、村网格干部考核制，明确职责，压实责任。对村实行人居环境考核结果与资金支持直接挂钩的机制。开展“美丽庭院”评选，持续推进“厕所革命”。在全镇9个村（社区）形成你追我赶的良好氛围，共同推进农村人居环境整治工作。
    强化网格职责，以点带面整体推进。以网格为单位，全镇划分为98个网格，网格干部为第一负责人，即镇村网格干部积极参与本网格内的环境卫生日常督导和清整工作。98个网格的镇村干部、深入一线亮身份比奉献、亮立场比担当、亮作风比作为。积极发动本网格群众参加“逢十大扫除”村庄清洁行动，采取逐组逐户清洁方式对网格内小组的陈年垃圾、隐秘垃圾进行地毯式搜寻清扫，做到了全村清洁全覆盖、全村群众全知晓。全镇502个小组，组织开展环境卫生整治，以点带面，整体提升人居环境水平。
    强化示范引领，营造良好氛围形成合力。镇村网格干部与党员群众积极沟通，共同探讨提升人居环境的良策。一方面党员干部以身作则，自觉开展环境卫生整治，主动搞好自家房前屋后卫生，打造美丽庭院；一方面主动交流、积极宣传环境整治的重要性，帮助后进群众提高认识、凝聚力量，积极参与到环境卫生整治工作中来。不断拉近党群、干群距离，着力形成“党员带头，群众参与”的生动局面，切实发挥党员在人居环境整治中的纽带作用和示范引领作用。
    庭院评比树新风。道林镇上半年共发布人居环境宣传推介网络链接20次，阅读量7万余人次，开展人居环境屋场会90余场次，评选出36户“美丽庭院”及36户“较差庭院”，形成“比、学、赶、帮”氛围，激发全镇群众爱护环境、积极参与环境卫生整治的良好社会新风气。大力营造村民既是农村人居环境整治的受益者，更应成为重要的参与者和建设者的良好氛围。

道林镇：靳江河上碧波荡漾，呐喊声中众桨翻飞

发表于 2023-06-25 分类于 乡镇动态 本文字数： 747 阅读时长 ≈ 1 分钟

靳江河上碧波荡漾，呐喊声中众桨翻飞。6月21日，靳江河畔聚集了越来越多的“观演”群众，多艘龙舟“蓄势待发”，当天，由宁乡市道林镇新时代文明实践所、道林镇龙舟协会、道林镇社工站举办的道林镇“我们的节日·端午”龙舟展示文化活动拉开序幕。
    靳江河又名靳水，为湘江下游支流，因河道经楚国大夫靳尚墓前而得名。端午节靳江河龙舟文化展示活动是道林镇历来的传统节目，风雨无阻，参与人数众多，场面壮观，历史悠久，早已成为了道林的名片。
    本次龙舟展示活动，道林镇龙舟协会共组织道林社区2条、龙泉湖村1条、河东新村1条、石金村1条龙舟共5条龙舟参演。河面上，桨手们齐力挥动船桨向前，劈波斩浪，你追我赶，场面超燃。
    为确保活动安全顺利进行，道林镇党委、政府高度重视，活动前期，多次组织各村（社区）相关负责人召开部署会议，明确责任分工，细化活动流程。活动中，全镇党员积极践行党员“三亮三比”行动，数百名志愿者参与秩序维护、后勤供应等工作。
    “我们举办龙舟文化展示活动，主要是大力弘扬中华龙舟传统文化，营造欢乐祥和的节日氛围，增强人民群众文化生活获得感、幸福感，助推‘千年古镇·幸福道林’建设。”道林镇相关负责人表示，活动的成功举办，也可以让更多的人用心感受屈原的廉洁思想和浩然正气，屈原不仅是一名伟大的爱国诗人，还是一位志洁行廉的仕者。司马迁称其志洁行廉，可与日月争光，李白称其辞赋“悬日月”，苏轼称自己文章不能及其之万一。屈原的生命虽然终结，但他廉洁正直、心性高洁的品格与坚持操守、绝不妥协的斗争精神，如一盏明灯，烛照千秋，万世敬仰。

道林古镇上榜第六批长沙市劳动与实践教育创建基地（营地）

发表于 2023-06-12 分类于 乡镇动态 本文字数： 434 阅读时长 ≈ 1 分钟

日前，第六批长沙市劳动与实践教育创建基地（营地）评审结果公示，道林古镇研学实践教育基地上榜。
    道林古镇，有着1100多年历史，旧“宁乡铺市”之一。古为商贾重镇，靳江河穿镇而过，宁乡物产在此集散，直达湘江。唐代诗圣杜甫流寓长沙时，曾留下“玉泉之南麓山殊，道林林壑争盘纡”的千古名句。
    道林古镇研学实践教育基地通过丰富多彩的国学与非遗的实践活动，充分发挥中华优秀传统文化在学生成长、成才过程中的浸润、化育与熏陶作用，让学生知家乡、爱故土、懂非遗、传文化，在潜移默化中得到精神的滋养，增强学生的文化素养和实践能力，树立正确的价值观，增强文化自信。
    建立劳动与实践教育基地，是落实劳动课的有效途径，是将思想教育、知识教育、技能教育、劳动实践融为一体的劳动教育平台，有效弥补学校和家庭教育的缺陷和不足，扩展学校教育空间，延伸和补充学校教育。
