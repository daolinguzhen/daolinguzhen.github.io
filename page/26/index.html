宁乡市道林镇：积极推行家庭教育，为孩子身心健康和谐发展保驾护航

发表于 2023-12-18 分类于 乡镇动态 本文字数： 445 阅读时长 ≈ 1 分钟

为进一步加强家校共育，帮助和引导家长树立正确的家庭教育观念，建立更好的亲子沟通模式，促进亲子和谐共处，守护孩子的健康成长，12月14日，由宁乡市道林镇卫生院主办，项目合作方湖南释然心理健康服务有限公司协办，宁乡市首个青少年心理健康促进试点系列活动在道林中心小学开展，道林中心小学和善山岭小学近200名家长参加讲座。
    本次活动邀请了长沙市妇联家庭教育专家讲师团骨干讲师文一瑀主讲“如何与孩子有效沟通”专题讲座，共话亲子沟通的有效方法，助力家长提升育儿能力。
    案例引导，和在座家长互动练习，指导家长在教育孩子时，怎样做到有效听，与孩子高效链接，说出孩子的感受，给在座家长们传递了很多建设性的指导与实用的沟通方法。
    本次家庭教育指导讲座增强了家校间的双向合作交流，进一步树立了科学育儿理念，有助于提升教育合力与成效，更好地为道林学子身心健康和谐发展保驾护航。

道林镇：组织引领促振兴

发表于 2023-12-11 分类于 乡镇动态 本文字数： 1.2k 阅读时长 ≈ 1 分钟

建设宜居宜业和美乡村，关键在党，重任在基层党组织。自开展“走找想促”活动以来，宁乡市道林镇善山岭村党总支引导党员在深化学习、服务群众、基层治理中主动作为、担当有为，共同构建宜居宜业和美乡村，以强烈的责任感和紧迫感推动“走找想促”落到实处。
    理论学习送上门，思想充电强信仰
    强大的精神信仰来源于坚实的理论基础，为确保年迈体弱、行动不便的老党员“不掉队”，善山岭村党总支开展送学上门活动，把学习资料送到支部党员家中。在询问了老党员的身体状况后，党支部书记向他们送上了组织的关怀、宣传了党的政策、留下了学习的资料，不仅让老党员及时“充电”，加深了对党的认识，还激发了老党员“活到老，学到老”的热情，同时，真诚传递党组织对老党员的关怀和重视，增加了党员的归属感，激励其始终牢记党员身份，坚定理想信念，提高党性修养。
    干群协同齐发力，产业道路共拓宽
    神仙岭风电场坐落于道林镇善山岭村和鑫星村，是长沙地区首座风力发电厂、全国首家生态发电厂。近年来，户外露营旅游逐渐火热，神仙岭逐渐成为长株潭周边游的网红打卡地，但由于道路等基础设施不完备，从山脚仅有一条土路通往山顶，神仙岭文化和旅游发展一直受限。“要想富，先修路”，为发展旅游产业，支村“两委”通过“四议两公开”决定将产业路硬化通车并将原道路拓宽，通过召开当地片长、组长、邻长及群众会议，大家主动作为，片长王青山、彭水军等同志全身心投入道路建设，广泛争取各方支持，组长阎小平起早贪黑在现场监督施工，确保施工质量，邻长卢先锋、金梅圣等人积极协调群众关系，在大家的共同努力下，产业路顺利通车。
    产业帮扶助增收，经济焕发新活力
    先富带动后富，共富成就幸福。善山岭村优秀共产党员卢先锋利用虎形山花草众多的环境优势，找准养蜂采蜜的发展定位，成立了宁乡何乐不为养蜂专业合作社，依托合作社创立了“养蜂+贫困户+合作社”的新模式，带领全村33户脱贫户发展蜜蜂养殖，并上门进行送蜂培训全程指导。优质蜂蜜既要制作出来还得销售出去，合作社一方面承诺对脱贫户养蜂产的蜂蜜实行保底回收，另一方面通过短视频等网络销售平台销售蜂蜜并帮助其他农户代售当地特产。农户们拓宽了销售渠道，实现了稳步增收，过上了美好生活，脱贫攻坚成果同乡村振兴有效衔接得到了进一步巩固。
    下一步，善山岭村党总支部将继续引导党员干部服务群众，全面推进村级各项事业发展，努力提高群众的获得感幸福感安全感，奋力推进乡村振兴全面发展。

道林镇：全域大扫除机制，创建国家卫生镇

发表于 2023-11-27 分类于 乡镇动态 本文字数： 421 阅读时长 ≈ 1 分钟

今年以来，道林镇狠抓镇域环境卫生整治提升，构建了由镇村干部片长包片、组长包组、邻长包户的镇、村（社区）、组、户联动的全域常态化大扫除工作机制。
    每月由镇农村人居环境常态化管理工作领导小组办公室牵头组织镇村网格“三长”、环卫员、志愿者，对每个村（社区）8个以上村民小组开展人居环境大扫除，主要对小巷、河道、沟渠、树林、房前屋后的陈年垃圾、焚烧痕迹、堆码乱象等顽疾问题进行集中整治。截至目前，已对全镇500个村民小组进行了两轮的全域大扫除。
    全域大扫除制度充分发挥了镇村干部的带头示范作用，压实了干部包村、包片区、包网格、包小组工作责任，通过村村响、微信群、宣传车、屋场会等方式，走街入户，加大爱国卫生运动政策宣传，极大增强了农户主人翁意识，带动广大村民朋友们人人动手主动进行卫生大清整，切实提高了人居环境整治成效，为道林镇争创国家卫生镇打下了坚实基础。

今年以来，道林镇大力推进人居环境全域整治，以“三长制”（片长、组长、邻长）为立足点，坚持“党员带头、发动群众、全域覆盖、到户到人”，构建“全域有网、网中有格、格中定人、人负其责”人居环境大整治机制，形成了党委书记为总负责人、镇长为调度员、镇村196名干部为片长、小组组长为组长和基层党员为邻长的群众民意“第一来源人”的“全镇一张网”组织体系，做到了“三长”人居环境工作责任人人上肩，群众爱卫政策人人知晓。
    道林镇以“三长制”工作机制为重要抓手，坚持每月对80余个村民小组进行轮番大扫除，镇农村人居环境常态化管理工作领导小组办公室每月对全镇网格全覆盖检查，评选“最美庭院”、“最差院落”并进行公示，不断提高镇村群众爱卫意识，不断提升镇域人居环境工作实效。
