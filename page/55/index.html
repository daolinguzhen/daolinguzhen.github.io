道林镇用洒水车为贫困户送水

发表于 2019-11-15 分类于 乡镇动态 本文字数： 512 阅读时长 ≈ 1 分钟

11月15日，一辆洒水车缓缓地驶到道林镇河东新村贫困户童新文家门前，原来，是村书记蒋丙洪租了一辆洒水车来为其送水。
    童新文有二级肢体残疾，大腿以下全部截肢，独自一人居住，九月份的时候家门口的井不出水，就打电话给驻村第一书记过来查看。第一书记夏强过来，发现井里实际有水，原来是抽水机的管子出了问题，于是找来一捆胶带，把水管重新缠了一圈，就又恢复了正常供水。
    这个月，童新文又打电话说井里没水了，村委派人上门查证，确实因为天气持续干旱，井水已枯竭。虽然附近还有其他水源，可是考虑到童新文身体情况，不方便出门打水，生活用水就成了摆在他面前的一道难题。
    村书记蒋丙洪得知这一情况后，自掏腰包租借了一辆洒水车来为童新文专程送水，解决其生活用水难题。
    无独有偶，早在九月份，道林社区贫困户张泽良老人独身一人居住，因井里缺水，家里又没有其他男劳力帮忙打水，石壁垅砖厂老板鲁光明得知这一情况后，也是租借洒水车为其义务送去了生活用水，得到了贫困户的阵阵好评。

百余村民齐参与，看宁乡这个屋场的“蝶变”之路

发表于 2019-11-04 分类于 乡镇动态 本文字数： 1.9k 阅读时长 ≈ 2 分钟

来到位于道林镇河东新村的狮子山屋场，没有看到大改大建的场面，这里的景致大多依据地势或现有的景观，进行改建或者改造而成。在这个依水而建的美丽屋场，沿路而行，可以找到诸多独属于道林的文化元素，比如龙舟，比如酸枣糕。
    狮子山屋场有30户人口111人，紧靠道林古镇，今年7月份开始着手美丽屋场建设。“我们依托于农村‘五治’工作，依据现有的地形地貌和资源，小投入开始对狮子山屋场进行改造和建设，目前还在建设当中。”道林镇镇长姜晓说，在屋场建设中，会更加注重于对环境卫生的整治以及文化元素的挖掘，力争打造出一个文明、和谐、美观的狮子山屋场。
    现在的狮子山屋场，有4个100%指标：改厕率100%、垃圾分类到户100%、禁养区退养100%、“空心房”拆除100%。
    对于狮子山屋场来说，要做到这4个100%，镇村两级下了很多苦功夫，在他们的努力下，村民由不配合到配合到主动加入，这一路，村民的变化看得见，而狮子山屋场的变化也看得见。
    “以前家家户户屋前屋后，马路旁边到处都堆着杂物，现在清理得干干净净的，卫生状况有了很大改观。”村民杨谷丰在外务工，不常在家，但是他在家的时候能做的总会做点。
    杨谷丰隔壁邻居常年不在家，家门口乱糟糟的，堆满了垃圾和杂物。进行美丽屋场建设后，杨谷丰在对自家环境进行大清扫的同时，主动承担起邻居家环境卫生的清扫工作。如今，两家屋门口皆是干干净净、清爽舒适。
    但因杨谷丰经常不在家，负责卫生清扫的担子就落在了年迈的老母亲杨爱兰身上，老人即使走路已颤颤巍巍，但仍坚持天天清扫好院子。她说：“做不了其他事，卫生还是要搞好的。”
    据悉，为了推进人居环境整治工作，今年河东新村投入环境治理经费约30万元，建立村级资源分拣中心1个、配备环卫三轮车10台、购置3200个垃圾桶发放到户、拆除原有敞口垃圾池72座。随后还通过“村村响”、宣传车、微信、宣传资料、标语横幅等密集宣传垃圾分类工作，专门组织联片镇村干部逐个村民小组召开户主会专题指导，家家户户发动、层层推进工作，努力营造人人讲环保、人人参与垃圾分类的良好氛围。
    如今，不仅是狮子山屋场片区，整个河东新村的环境卫生都上了新台阶。
    为了全力推进狮子山屋场“五治”工作，今年，该片区户主已经开了5次户主会，在大力发动和宣传下，家家户户都参与进来，一起为更美狮子山屋场出力。
    到达狮子山屋场范围内，首先看到的是一片地质灾害区，滚滚黄泥下，埋着一座房子。如何小投入地将这片有隐患的地方变成一个有吸引力的地方？姜晓说，当时就想着将这里变成地质灾害知识的宣传口和知识传播口。
    于是，这里出现了两块文化栏，一块写着“河东新村狮子山组泥石流救人干部典型”，一块着重介绍“地质灾害科学普及知识宣传”，让大家在看到道林干部风采的同时，还可以了解到相关知识，一举两得。
    “我们这里的酸枣糕非常好吃，也很有名。我们屋场有两棵好大的酸枣树，被打造成了‘酸枣之恋’，还蛮有味的。”村民杨科山说，屋场打造还是很有地方特色，除了“酸枣之恋”，龙舟文化也是道林镇的一大特色，于是也着重介绍了龙舟文化。
    在狮子山屋场的打造中，不难看出，除了巧立景外，更大一部分比重放在了文化宣传和弘扬中，弘扬着积极向上的风气。
    “这些东西看着不起眼，或者看上去作用不大，其实给屋场周边农户还是带来了触动。”村民杨金标说，如今这里的民风有了很大改变，以前说起要做什么事、要搞什么建设，大家的第一反应是排斥、不支持，现在随着这些知识的不断传播，随着典型的带动，许多人在点头支持的前提下，会站出来，投工投劳，一起参与到建设当中。
    “现在的狮子山屋场不是以前的狮子山屋场了，越来越漂亮了，相信以后会变得更好。”杨金标说，现在的狮子山屋场是天天有变化，几个月时间就已经大变样了，相信在外务工的村民回来后，都会大吃一惊的。

道林镇道林社区唱响基层协商民主“好声音”，建立健全机制确保食品安全

发表于 2019-10-31 分类于 乡镇动态 本文字数： 565 阅读时长 ≈ 1 分钟

“为什么要创建食品安全示范街？都有哪些好处？”“餐饮服务单位的创建标准具体是什么？”……近日，一场特殊的会议在道林镇道林社区举行，来自该社区的67名基层民主协商对话会成员借助对话会搭建的协商新平台，就食品安全工作展开热烈讨论，并对该社区“食品安全示范街创建”的相关议题进行投票表决。
    据了解，自我市开展食品流通领域创建“食品安全示范管理店”活动以来，道林社区进一步建立健全食品安全管理的长效机制，创造性开展食品安全信息员队伍建设，确保食品安全任务到人、责任到人。同时，建立了全社区食品生产经营单位普查台账，对道林大道39户集镇经营户进行逐户登记，分类汇总，列入电子文档。
    对话会首先宣读了《严厉打击食品生产经营环节违法违规行为，坚决取缔“黑工厂”“黑作坊”“黑窝点”“黑外卖”专项整治推进方案》《宁乡市2019年小餐饮“透明厨房”提质工程实施方案》两个文件。随后，大家围绕道林社区“食品安全示范街创建”相关议题展开讨论。针对提出的问题，相关特邀嘉宾一一作了耐心细致的解答。
    此外，大家还就办理手续需要的相关材料、集镇流动摊贩整治、规范集镇停车行为、餐饮服务单位的具体创建标准等议题展开了讨论。

道林镇道林社区召开“党建引领三优三创”暨“食品安全示范街创建”基层民主协商对话会

发表于 2019-10-18 分类于 乡镇动态 本文字数： 863 阅读时长 ≈ 1 分钟

2019年10月18日下午，结合“不忘初心、牢记使命”主题教育，道林镇党委、政府在道林社区大会议室召开了“党建引领三优三创”暨“食品安全示范街创建”基层民主协商对话会。道林镇党委书记李纯，党委副书记、镇长姜晓，党委委员、政协联络处主任、副镇长刘军，副镇长孟曙光出席会议，道林镇党政办、党建办、食安办、社会治安及应急办、自然资源和生态环境办、政务服务中心、综合执法大队主任，全体社区干部及农商行、农业银行、派出所、卫生院、市监所、城管中队的负责人，道林社区道林大道食品、餐馆经营商户43人，道林社区基层民主协商对话会成员30人参加了会议。

会议由党委委员、政协联络处主任、副镇长刘军主持，会上学习了关于《严厉打击食品生产经营环节违法违规行为，坚决取缔“黑工厂”“黑作坊”“黑窝点”“黑外卖”专项整治推进方案通知》、《宁乡市2019年小餐饮“透明厨房”提质工程实施方案》；道林镇市监所刘虎所长就“食品安全示范街”的创建、《餐饮服务单位创建标准》、《食品销售单位创建标准》进行了业务解读。
    5名道林社区食品、餐馆经营商户及基层民主协商对话会成员就“餐饮店提质改造的申报条件和补贴标准”、“创建食品安全示范街好处”、“食品安全示范门店创建要办理什么手续”、“道林社区党建引领三优三创具体内容”、“对流动摊贩采取什么措施和制度”等议题与到会领导进行交流答复，到会的73名人员全票通过创建道林社区“食品安全示范街”。

道林镇镇长姜晓指出，目前长沙市创建国家食品安全示范城市，各区县市也在积极进行创建工作。食品安全关系到消费者的安全和切身利益。创建“道林食品安全示范街”一是确保“舌尖上的安全”；二是提升道林镇营商的整体形象；三是为古镇开园运营创造良好的外部环境。
    道林镇党委书记李纯强调，全镇上下要进一步加强食品安全管理，强化食品安全工作责任，规范食品市场秩序，确保人民群众饮食安全。解决食品安全突出问题，把握食品安全关键环节，确保创建目标全面实现。

道林镇：让党旗高高飘扬在神仙岭上

发表于 2019-10-09 分类于 乡镇动态 本文字数： 1.1k 阅读时长 ≈ 1 分钟

自道林镇“不忘初心、牢记使命”主题教育工作开展以来，该镇党委政府深入学习贯彻党的新思想、新理念，立足于人民群众需求查问题、找差距、强落实，解决了一些人民群众关心的重点难点问题。
    道林镇神仙岭风电场因优美的自然风光，吸引众多市民自发前来观光和露营，造成了交通堵塞并对当地环境造成了困扰。对此，道林镇党委、政府积极开展巡查和劝导服务，确保了神仙岭风电场的交通安全和环境保护。
    据悉，神仙岭风电场为自然风光，未进行旅游开发，山间道路窄且陡，也无相关生活配套设施，存在一定安全隐患，但道林镇党委政府考虑到广大市民来此观光属正常需要，没有简单地封山，而是限制车辆进入和严禁火种进山，并增设安全警示牌，让观光的市民感受到道林自然风光的同时，消除安全隐患。
    为进一步维护交通秩序、保障出行安全、保护环境卫生，道林镇结合”不忘初心、牢记使命”主题教育，在国庆小长假期间组织镇村党员干部开展了一系列“文明出游、优化环境”等志愿服务活动，让党旗高高飘扬在神仙岭上。
    一是守牢安全底线，第一时间在通往神仙岭的便道上设置警示标牌，发布禁止机动车辆进入黄茅大岭的通告，提醒游客在景点便道入口有序停放车辆，并步行前往至神仙岭，做到安全出行、低碳出行；二是广泛宣传引导，镇村党员干部35人佩戴红袖章自发组成志愿队伍，一方面通过张贴温馨提示100余份告知广大游客要保护片区内的一草一木，爱护环境卫生，不乱扔乱放；另一方面对沿线道路及景点的卫生开展大扫除行动，大家既分工又合作，齐心协力细心清理每一个卫生死角，把片区内的脏、乱、差地带清理得干干净净；三是强化巡查管控，城管、质监所、派出所、镇村值班人员组成专职队伍，由镇主要领导带队，每天对镇域内的集镇和景区地段进行2-3次的巡查督查，尤其对神仙岭片区进行了重点管控，规范通行秩序，严查乱经营和乱收费行为。
    除此外，道林镇还扎实抓好了国庆维稳、秸秆禁烧和森林防火等工作。在国庆维稳方面，通过统筹驻村辅警、治调主任、民兵连长、平安志愿者等资源力量，最大限度地优化配置群防群治队伍，镇级治安巡防队伍90人、村级治安巡防队伍45人做到了每天一巡查。秸秆禁烧方面，通过村村响广播、出动宣传车、发放宣传单等方式深入村组宣传秸秆禁烧工作，并加强对秸秆燃烧的巡查管控，一经发现火点，立即制止并对村或个人作出相关处罚。森林防火方面，实行责任包干制，要求各村（社区）加大森林防火宣传力度，提高群众森林防火意识，并加强野外火源管理，消除森林火灾隐患。

道林镇举办“不忘初心、牢记使命”主题教育读书班进行集体学习

发表于 2019-09-30 分类于 乡镇动态 本文字数： 621 阅读时长 ≈ 1 分钟

9月29日，道林镇举办“不忘初心、牢记使命”主题教育读书班进行集体学习，道林镇党政领导干部、镇属党组织书记及全体机关干部职工参加会议，宁乡市委“不忘初心、牢记使命”主题教育第九指导组到会指导。
    会议播放《刘少奇的故事–忠诚担当》教育视频，并邀请市委党校黄学君高级讲师授党课《牢记初心使命 推进自我革命》。
    会议上指导组周旭文同志转达了市委主题教育工作的最新会议精神和要求，指出主题教育理论学习要全方位开展，一要理论学习上要深刻理解，在思想上实现提升；二要理论指导在实处，明晰道林发展的定位、存在的问题和发展的方向；三是通过理论学习统一思想认识，领导好班子队伍。

    道林镇党委开展“不忘初心、牢记使命”主题教育以来，开展多种形式的理论学习，目前各党支部已发动全体党员开展理论学习，接下来镇党委将分批组织基层普通党员进行集中学习，帮助党员补足“精神之钙”、把稳“思想之舵，激励广大党员密切联系群众和服务群众、勇担时代使命投身建设大美道林。
