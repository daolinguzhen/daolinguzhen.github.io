好别了，道林烟囱！

发表于 2018-11-19 分类于 乡镇动态 本文字数： 955 阅读时长 ≈ 1 分钟

拥有高耸入云的烟囱曾经是一个地方经济繁荣的标志工业走向发达的象征。但如今，烟囱正在成为历史记忆。对于道林镇的烟囱史来说2018年的11月11日将是一个值得记录的日子。因为在这一天里该镇的34根烟囱将全部倒地！这将宣告着道林这座千年古镇正式告别烟囱时代，翻开进步的一页！
    10月26日打响烟囱整治第一枪
10月26日上午十点整，随着“轰隆”一声闷响，道林镇鑫星村金家湾24米高的中南耐火材料厂烟囱应声倒下。不过短短5秒钟的时间，大烟囱灰飞烟灭。
    在对爆破现场检查确认安全后，爆破公司的工作人员马上赶往了下一个爆破点，进行爆破准备。道林镇政府、市公安局、市规建局的工作人员疏散了附近群众并拉起了警戒线。不到一个小时的时间，第二根烟囱也按照预定计划安全倒地。

当日上午，在现场各部门和乡镇的紧密配合下，鑫星村的解明耐火材料厂、金瑞新特耐火材料厂、中南耐火材料厂、金锋耐火材料厂四根烟囱全部按时、定向安全爆破倒地。
    与此同时，道林镇还安排了4组队伍，对不具备爆破条件的烟囱进行人工拆除。
    11月11日即将拆下最后一根烟囱
    10月27日，道林镇鑫星村村民刘利纯站在倒地的烟囱旁，看到离家不到10米远的那根矗立了30多年、近30米高的烟囱被“放倒”，很是兴奋。“以前只要是碰上刮风下雨的天气，这根‘老烟囱’就会掉砖屑下来，到处开裂了，真的担心会突然倒下来。如今烟囱炸掉了，我也就可以多睡几个安心觉了！”

在鑫星村烟囱爆破拆除现场，道林镇镇长陈浩告诉宁视君，该镇的耐火材料厂曾是宁乡乡镇工业的一面旗帜，仅鑫星村就聚集了近30家，可以说在当时的发展背景下，为宁乡的工业经济作出了特殊贡献。
    但随着时代的发展，这些企业由于原料开采对生态的破坏、产品烧制对环境的污染，越来越不适应生态建设要求，甚至成为当地群众人人喊打的“过街老鼠”，而产业转型升级的巨大投入，也使这些逐渐失去市场竞争力的企业难以承受，以至于目前鑫星村24家耐火材料厂基本处于停产和半停产状态。

道林城管中队2018年半年工作总结

发表于 2018-10-12 分类于 乡镇动态 本文字数： 1.2k 阅读时长 ≈ 1 分钟

道林中队成立以来，全队人员团结奋斗、齐心协力，全面贯彻落实上级的各项工作部署安排，坚持打基础、抓完善、促发展，严格履行自身职能，积极探索长效机制，大力加强制度建设和队伍建设，开展了城乡环境综合整治、打非治违等工作，较好地完成了各项工作任务。具体情况如下：

一、城乡管理展示了新作为

一是加强领导，健全机制。为进一步改进城乡管理模式，提升城乡管理水平，按照”精细化、全覆盖“的要求，进一步健全考核评价体系，制定了《道林镇2018年城乡管理考核工作方案》和《道林镇2018年“路长制”实施方案》，成立了以党委书记潘志国为顾问、镇长陈浩为组长、相关分管领导为副组长的领导小组，加强对我镇城乡管理工作的领导。二是强化督查，狠抓落实。将“路长制”工作纳入城乡管理工作（占比40%），实行一月一考核，定期通报讲评，严格奖优罚差。我队每月月底采用徒步的方式对各村（社区）主干道沿线的卫生、秩序和安全等方面的内容进行考核督查，积极开展问题交办和“回头看”工作，将发现的问题拍照留档并交办给相关村（社区），督促其限期整改。今年以来，共交办问题318个，整改问题302个，整改率达95%，同时，实行乱搭乱建、开设砂场、私建大棚“一票否决”制，截至目前，已有4个村（社区）月考核中因为存在乱搭乱建和非法开采现象被“一票否决”。三是多措并举，注重实效。以“路长制”为抓手，充分发挥“百姓城管”作用，持续加大保洁、清整、控违和治乱力度，夯实“蓝天保卫战”工作基础。今年以来，共发放“门前三包”宣传资料200余份、签订“门前三包”责任书180余份；线路巡查220余次、路面洒水降尘380余次；清理乱堆乱放650余处、劝导各类摆摊设点行为300余人次、拆除沿线非标、旧招牌320余块、清理牛皮癣2000余处；治理车辆乱停乱放1850余台次、协调处置交通事故3起、参与节假日保通保畅250余次、消除公路安全隐患5处，全镇主次干道沿线和市容秩序明显改善。

二、打非治违实现了新突破

为进一步规范集镇形象，消除“两违”建筑带来的不良影响，我队联合多部门加强对镇区景区范围、主要公路两厢、靳江湿地两岸的巡查管控，严格建房审批管理，严控新建违建行为，以“零容忍”的态度坚决遏制“两违”，坚决做到“露头就打”。截至目前，共拆除违章建筑24处，面积约1580平方米，取缔非法场地12处。另外，我队还积极配合镇重点办、农办开展“空心房”拆除和畜禽退养工作，20户养殖户全部签订退养协议，河东新村19户“空心房”全部拆除。
    下一步，我队将持续抓好城乡环境综合整治和打非治违工作，为了建设大美道林添砖加瓦。一是充分发挥“百姓城管”作用，加大舆论宣传力度，营造浓厚的城管氛围，增强群众参与集镇管理意识；二是整合执法资源，提高执法水平，通过规法执法、文明执法，进一步提高城管工作知名度和信任度；三是打好“组合拳”，稳步推进我镇城乡管理工作水平。

勤快的“卖鸡倌”

发表于 2018-09-21 分类于 乡镇动态 本文字数： 491 阅读时长 ≈ 1 分钟

今年82岁的王菊生和妻子两人是道林镇善山岭村贫困户，王菊生膝下没有儿女，只有妻子嫁给他以前带过来的继子，条件也一般，妻子身体不好，只能靠着年迈的王菊生做点农活勉强维持生计。
    今年6月，善山岭村的第一书记胡泉江牵线联系城郊街道免费送了700只鸡苗给贫困户，王菊生也去领了50只鸡苗。眼看着小鸡仔一天天长大，王菊生反倒发起了愁，自己年纪大了，成天挑着鸡去镇上卖来回折腾，卖掉了还好，要是没卖掉，还得自己赔上车钱。正巧，胡泉江到王菊生家里走访的时候，王菊生就跟他说到了这一苦恼，考虑到王菊生的身体情况，胡泉江爽快地答应了替他卖鸡。
    通过微信朋友圈和私下逢人联系，没几天，就有人找胡泉江预定鸡。这几天，胡泉江马不停蹄地奔走于王菊生家和买鸡客户家中，送货上门，被同事笑称为“卖鸡倌”，可是胡泉江却乐此不疲。几天下来，胡泉江共替王菊生卖掉了10只鸡，王菊生足不出户就赚到了一千来块钱。“真的感谢您啊，胡书记，您可真是我们的贴心人啊！”王菊生对胡泉江不住地道谢。胡泉江只是笑笑：“下次还要卖鸡再联系我。”
