道林镇375名党员集体“充电”

发表于 2021-03-25 分类于 乡镇动态 本文字数： 916 阅读时长 ≈ 1 分钟

近期，宁乡市道林镇党员轮训工作正在如火如荼进行，目前已顺利组织3期，全镇共计9个村（社区），375名农村党员参加集体“充电”，学习培训为期3天，号召广大党员认真学习贯彻习近平新时代中国特色社会主义思想和党的十九届五中全会精神，树牢“四个意识”，坚定“四个自信”，做到“两个维护”，以新气象、新担当、新作为积极投身乡村振兴的伟大实践。
    为抓好抓实党员教育培训，道林镇对此次培训的内容进行了系统的设计，将党史学习、党性增强、乡村振兴、集体经济发展、个人综合能力提高相结合，通过精选培训内容，为广大党员提供了有价值、有意义、有收获的学习课程。

在党史学习方面，道林镇结合课程学习、影片观摩等方式，激发学员的学习热情。通过邀请市委党校陶俊老师讲授《党的奋斗历程与光辉成就》，以及《建党伟业》、《刘少奇的四十四天》等影片的观看，让党员们在学习后立即参与讨论，体会收获。在党性增强方面，邀请市委组织部老师，通过《党员的权利和义务》、《党员如何在乡村振兴中发挥先锋模范作用、《如何做一名新时代的合格党员》等课程，利用主题探讨的方式让广大党员从先进典型、身边优秀党员和自身经历等方面出发，分享优秀事迹，引导党员不断提升自身党性和纯洁性。此外，道林镇还与宁乡职业中专教师合作，引入《农村电子商务》、《短视频制作》等课程，以期能为乡村振兴助力。在课程中，授课老师现场答疑解惑，教授学员们如何用手机制作出精良短视频。学员们广泛参与教学实践，河东新村的黄立平同志利用手机中已有的自家餐馆素材，现场制作完成了一份30秒的剪映短视频。黄立平表示课程操作性强、实用性高，希望未来能借助短视频宣传好自家餐馆，为乡村振兴贡献自家力量。
    据悉，此次道林镇开展的党员集中轮训将持续到本月底，未来几天将结合党史学习教育，开展“学史明理、学史增信、学史崇德、学史力行”主题交流，以多种丰富形式开展各村（社区）党员轮训，预计总学习人数达1800余人。接下来，道林镇还将有序开展党务骨干、机关事业单位党员等专题培训，贯彻落实好习近平总书记“学党史、悟思想、办实事、开新局”的重要讲话精神。

道林镇：从田头到案头，让百姓尝甜头

发表于 2021-03-17 分类于 乡镇动态 本文字数： 1.1k 阅读时长 ≈ 1 分钟

40多年没有清理过的渠坝，终于要疏通了，对于道林镇华鑫市村的村民们来说，这是一件天大的大好事。渠坝蓄水功能恢复后，他们就可以放心地发展种养殖业了。
    近日，道林镇华鑫市村为了赶在春耕前把渠坝淸整好，方便接下来的春耕生产，在家的劳动力只要看天气适合施工，就出来帮忙了。
    对该村种粮大户杨国强来说，整修河坝是一件事关他收成收入的大事，种了140多亩水田的他，灌溉用水都要从水坝里来。在他的印象中，水坝还是七十年代的时候看父辈修过，已经40多年没人动过了，没修整前，水坝淤塞严重，蓄水不足，抗旱时一台两寸管抽水机，抽一个小时就没水了。
    “以前种田每年靠天吃饭，而且差不多每年都干了，这个渠坝修好后，农田就水旱无忧了。”华鑫市村种粮大户杨国强高兴地说。
    据介绍，此次整修的渠坝有近2公里长，涉及华鑫市村小塘湾组、后背湾组、白石桥组、瓦屋湾组四个村民小组，近400亩水田的灌溉问题，此次主要是对渠坝进行清淤除杂、安装闸门。
    华鑫市村民风淳朴，勤奋好强，由于水利设施陈旧影响灌溉，多年来一直制约着当地村民发展种养殖业。如今，当地有群众想发展稻虾产业，但灌溉问题不解决，心里就没有发展的底气。在今年年初的“四联五到位”民情大走访中，宁乡市政协党组书记、主席刘永红得知了这一情况。
    华鑫市村离退休老支书黄定国说：“之前群众经常反映水利建设问题，在这次建议中，我提出农田基本建设主要是河道、渠道、骨干山塘，有些淤塞很多年了，我们希望多争取资金把农田基本建设搞得更好，引进大户把粮食生产，把水利建设搞得更好。 ”
    经过十多天的宣传发动，目前渠坝清淤除杂工作正在有序地开展，群众投工投劳，政府以奖代补投料，让目前群众反映最迫切的问题顺利得到解决。渠道通了，群众积极性提高了，心里的结解开了，春耕生产、农业产业发展更有保障了。
    截至目前，道林镇全镇56个网格，已实现走访100%覆盖，收集问题261个，已解决186个。
    据道林镇党委书记彭舜介绍，该镇坚持党建引领，按照发展类、示范类、纠纷类、信访类、建议类、公共服务类共6类对象进行走访，与9个后盾单位密切对接，做到“前提是联系，关键是问题解决到位”，紧紧围绕村民建房、禁止非法开采、交通安全、医疗保险、春耕生产等政策法规进行走访、宣传。

道林镇深入开展校园食品安全“护苗”专项整治行动

发表于 2021-03-12 分类于 乡镇动态 本文字数： 281 阅读时长 ≈ 1 分钟

为推动辖区内学校及周边食品经营者主体责任进一步落实，食品安全管理制度进一步完善，食品安全监督管理工作进一步加强，校园及周边食品经营秩序进一步好转，广大师生食品安全意识进一步提高，食品安全保障能力进一步提升，及时消除各类食品安全风险隐患，坚决抵制群体性食品安全事故和负面舆情事件的发生。近日，道林镇社会治安与应急管理办公室和城管中队、社会事务办、市监所、基地校等5个部门联合开展校园食品安全“护苗”专项整治行动。

该项行动主要针对校园食堂和学校及周边商店、流动摊贩食品安全情况进行专项整治。截至3月12日，道林镇辖区内10所小学，2所中学，5所幼儿园已全部完成整治。

道林镇：书写全面建设现代化道林新篇章

发表于 2021-03-04 分类于 乡镇动态 本文字数： 550 阅读时长 ≈ 1 分钟

2020年以来，宁乡市道林镇紧扣上级决策部署，聚焦“千年古镇、幸福道林”定位，全面提升新时代道林镇经济社会发展水平，成效初显。
    道林镇抢抓机遇，进一步放大区位、人文、生态等优势，致力于将道林打造成长沙的后花园和宁乡旅游新高地，让长沙的人流、物流、资金流、信息流在道林汇集，使千年古镇焕发新生机。
    道林镇持续加大对天大锅炉、鑫美矿业等规模企业扶持力度，加快耐火材料行业的转型发展。突出农旅产业融合发展，紧盯富民增收重点，在农业特色产业培育上，聚焦发展现代农业、壮大新型村级集体经济两个主战场，持续做好烟叶稳产提质扩规文章，深入推进生猪养殖、家畜农场建设和荸荠种植等项目，紧盯水电路、文教卫等重点，解决了一批人民群众急、难、愁、盼的身边事，全面落实了安全生产责任，确保了全镇政治安定、社会安稳、群众安心。同时，坚持以“五治”工作为抓手，努力争创长沙市级“五星”乡镇和省级卫生乡镇，全面展现道林山水自然和人文历史、乡土文明与现代文明高度融合的美丽画卷。
    “新年开新局，我镇将扎实做好‘四联五到位’民情大走访工作，书写全面建设现代化道林新篇章。”道林镇党委书记彭舜表示。

道林镇开展食品安全专项检查

发表于 2021-02-26 分类于 乡镇动态 本文字数： 325 阅读时长 ≈ 1 分钟

元宵节期间，为保障全镇人民群众“舌尖上的安全”，道林镇食安办联合市监所开展了食品安全专项检查行动。
    此次专项检查以超市、农贸市场、便利店，小作坊销售的元宵、汤圆、糕点等节令食品为重点，主要检查各类食品及原材料存储摆放是否规范、外包装是否完整、生产和保质日期是否合规、票证是否齐全、制作和销售人员是否办理健康证、是否按卫生要求佩戴口罩等，全面排查食品安全风险隐患，严防食品安全问题。
    截至2月25日，道林镇食安办、市监所检查辖区元宵、汤圆类经营业户11家，检查各类元宵、汤圆产品20余种，有效规范了元宵节前后市场经营行为，为辖区群众食用安全放心元宵提供了保障。

道林镇开展“家家户户搞卫生，干干净净过新年”人居环境集中整治活动

发表于 2021-02-08 分类于 乡镇动态 本文字数： 241 阅读时长 ≈ 1 分钟

为欢欢喜喜迎新春、干干净净过新年，结合当前疫情防控工作要求，2月7日，道林镇开展“家家户户搞卫生，干干净净过新年”人居环境集中整治活动，对机关单位办公区域及庭院周边、镇域范围内全体农户房前屋后及庭院、各村（社区）村部及人口密集区域的环境卫生及镇村主干道的道路两旁杂草和路障进行清理，全体机关干部职工、村干部、党员和群众600余人参加。
