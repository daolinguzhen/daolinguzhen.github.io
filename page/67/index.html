凝心聚力抓示范、匠心独运铸精品

发表于 2018-01-30 分类于 乡镇动态 本文字数： 1.3k 阅读时长 ≈ 1 分钟

2017年，道林镇按照“麟靳风情古镇、当好长潭融城先锋、加快建设宁乡最具活力优美乡镇”的总体思路，突出规划引领、基础完善和产业转型，以道林古镇建设为契机，出台奖补政策，广泛发动群众，由点到线及面推进美丽乡村建设，重点围绕华鑫市村整村推进和龙泉湖村以点推进打造精品，通过精品示范的建设打造精美、精细、精致的美丽乡村名片。
    科学规划乡村建设。华鑫市村位于道林镇西北部，与花明楼接界，距离镇中心1.5公里左右。地理位置优越，交通路网发达，县道X087线穿村而过。根据村级资源禀赋和传统格局，科学区分生产、生活、生态领域，依法适用土地，规范建设。龙泉湖村与湘潭接界，与道林古镇毗邻，该村着重把自然田园山水风光与农业文化相结合，加快完善以道路、绿化、亮化、硬化为重点的基础设施体系，村庄面貌视觉效果大为改观，村庄品味大步提升。
    着力完善基础设施。围绕改善农村人居环境，大力进行村庄基础设施建设。年内华鑫市村硬化罗家湾、石塘子组公路1公里，主干道架设节能路灯60盏，河坝、道路和庭院栽种苗木600多株。同时，不断完善公共卫生服务体系建设、便民服务中心，围绕村部打造公园式休闲中心；注重丰富村民群众文化娱乐方式，建设有篮球场、兵乓球台和农家书屋等，不断提升群众生活水平。龙泉湖村硬化公路4公里，路灯建设80盏，绿化公路栽种苗木100多株。
    大力培育拳头产业。将产业与发展相融合，在稳定粮食生产的基础上，大力培育种养大户，农业机械化水平大幅提高。同时，按照“一村一品”思路，大力推动休闲旅游、生态农业发展，华鑫市村卉苗木基地和水稻蔬菜水果基地初具雏形，龙泉湖村一批集餐饮、垂钓、休闲于一体的农家乐加快崛起。
    稳步提升乡村面貌。将生态环境建设纳入日常工作，制定“村干部包片、组长包组、农户包户”以及“环境卫生争先创优”制度，完善垃圾处理体系，建设大型垃圾焚烧炉并投入使用，打造村庄环境卫生样板村，华鑫市村成功荣获“长沙市环境卫生示范村”称号。
    积极拓宽建设渠道。在立足自身实际、坚持量力而行的基础上，更加注重扩大人民群众的参与面，更加注重发挥党员组长的带动力，更加注重调动乡贤乡友的积极性，凝聚美丽乡村建设的强大合力，真正形成镇奖补、村发动、组为主、各方参与、共建共享的生动格局，确保美丽乡村建设模式可持续、能复制、有强大生命力。
    根据年初规划以及项目铺排内容，华鑫市村、龙泉湖村按照循序渐进的原则，一步一步落实项目。目前，两村均已完成既定任务，受到了当地村民的一致认可，并且华鑫市村已经成功通过市级示范村验收。下一阶段，将继续探索产业融合发展模式，借力古镇建设开发，放大道林近城优势，积极推动适合本村发展的休闲旅游项目，有针对性地打造一批主题农庄、特色民宿。

道林镇：发展“甜蜜”产业助力扶贫事业

发表于 2018-01-23 分类于 乡镇动态 本文字数： 646 阅读时长 ≈ 1 分钟

近年来，道林镇立足区域优势，积极发展养蜂产业，走出了一条“养蜂+扶贫”的精准扶贫新路子，确保贫困户通过养蜂实现稳定增收。

一、因地制宜，精准施策

道林镇善山岭村虎形山附近森林茂密，环境优美，花的种类较多，适合发展养蜂产业，且该地位于长韶娄高速附近，邻近长株潭地区，产出来的蜂蜜也不愁销量。因此，道林镇扶贫办和善山岭村村委将养蜂定为本村的特色扶贫产业，大力争取扶贫产业资金，并积极向贫困户宣传滴灌式扶贫奖扶政策，吸引贫困户投身养蜂产业。2016年，为善山岭村32户贫困户每户免费发放三箱中华蜜蜂种群，充分调动了贫困户养蜂的积极性。

二、能人带头，技术先行

现年43岁的卢先锋是一位本地职业养蜂人，2011年起回到道林镇开始养蜂，2015年成立注册了宁乡何乐不为养蜂专业合作社，2016年响应宁乡县产业精准扶贫“四跟四走”的号召，成为了善山岭村产业扶贫的帮扶先锋。他的养蜂基地聘请了省蜂业协会多名专家担任顾问，每年定期举办多场技术培训和科普讲座。他还不定期地到养蜂贫困户家中现场指导，大部分贫困户的养蜂技巧有所提高，基本能够单独对蜂群进行管理。

三、创新模式，拓宽渠道

依托卢先锋的养蜂专业合作社，创立了“养蜂+贫困户+合作社”的新模式。合作社给养蜂贫困户提供免费技术指导，贫困户一箱蜂平均产蜜20斤，市场价一斤能卖到80元左右，贫困户通过养蜂实现了稳步增收。同时，合作社还承诺对贫困户养蜂产的蜂蜜实行保底回收，贫困户可以通过资金或技术入股的方式加入合作社参与分红，多种收益渠道给养蜂贫困户吃下了一颗定心丸。

扶贫专干当考官——道林镇脱贫攻坚考核出新招求实效

发表于 2018-01-18 分类于 乡镇动态 本文字数： 458 阅读时长 ≈ 1 分钟

近日，道林镇圆满结束对9个村（社区）的脱贫攻坚考核，在这次考核中，9个村（社区）的扶贫专干既是“考生”也是“考官”，全程参与到了考核各项环节。
    1月2日起，道林镇对村（社区）脱贫攻坚考核正式拉开序幕，为确保考核的公平公正，除每个入户调查组配备了录音笔外，该镇还进一步创新举措，抽调各村扶贫专干担任考核小组成员，除自己村被考核时回避外，其余8个村（社区）的考核不得请假缺席，而且每天轮换组次。这样一来，既保证了考核的客观公正性，也让扶贫专干走出本村，更好地学习借鉴兄弟村好的经验做法。
    “我去年下半年才接手这项工作，业务还不是很熟，得知要在这次考核里担任考官给其他村打分，不想到时一个问题都发现不了被笑话，我元旦假期恶补了各种精准扶贫知识，在检查的工作结合实际运用，知识点一下就记牢了。”龙泉湖村专干钟思政感慨参与这次考核让他受益匪浅。“通过查看其它村的台账资料，也更加清楚地看到了我们村存在的不足，以后整改方向也更明确了。”

道林镇脱贫攻坚“年终大考”开考了

发表于 2018-01-04 分类于 乡镇动态 本文字数： 477 阅读时长 ≈ 1 分钟

“请问是黄运其吗，我是道林镇脱贫攻坚工作考核小组成员，现在对您进行入户问卷调查……”2018年1月2日，道林镇河东新村的贫困户家里来了几位手持录音笔的“访客”。原来，是道林镇党委政府正在对该村进行脱贫攻坚“年终大考”。
    2017年12月27日，道林镇根据市精准扶贫工作有关部署，决定对全镇各村（社区）脱贫攻坚工作进行绩效考核，制定了《道林镇脱贫攻坚考核实施方案》，并于12月29日组织召开了脱贫攻坚考核部署会。镇党委政府成立了由书记、镇长担任组长的考核小组，考核小组成员由镇纪委、扶贫办、财政所等办抽调精英骨干组成。根据考核方案要求，将考核小组成员分成两个资料核查组和六个入户调查组，为确保考核成效，考核小组每天考核一个村（社区），镇党委书记潘志国要求所有村的贫困户要全部上户走访，不在家的由村上开具证明，并为入户队员配备录音笔，以备查证。各村扶贫专干也一同参与考核，以达到互相学习、促进工作的效果，镇纪委书记全程参与监督。镇政府将综合此次考核结果与市级抽查复合情况，根据排名定等对各村（社区）予以相关奖励。
